## ¿Qué es la Conversión XPS de Aspose Page?
La conversión XPS de Aspose.Page es un conjunto de APIs que leen un documento XPS, renderizan cada página y escriben el resultado en el formato de imagen o documento elegido. El proceso se gestiona completamente en memoria, por lo que no necesitas herramientas externas ni dependencias nativas.

## ¿Por qué usar la Conversión XPS de Aspose Page en Java?
- **Alta fidelidad** – conserva la información vectorial al convertir a PDF o PNG.  
- **Optimizada para rendimiento** – las conversiones por lotes se ejecutan rápidamente incluso con documentos grandes.  
- **API sencilla** – solo se requieren unas pocas líneas de código para obtener una salida de alta calidad.  
- **Multiplataforma** – funciona en Windows, Linux y macOS sin instalaciones adicionales.
