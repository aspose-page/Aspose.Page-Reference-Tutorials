## Problemas comunes y consejos

- **Missing fonts** – Asegúrate de que las fuentes referenciadas en el EPS estén instaladas en la máquina host o incrústalas usando la API.  
- **Large file size after resize** – Usa el método `saveOptions.setCompressionLevel()` para reducir el tamaño del archivo sin perder calidad.  
- **Unexpected clipping** – Verifica nuevamente las coordenadas del rectángulo de recorte; se expresan en puntos (1 pt = 1/72 pulgada).  

## Preguntas frecuentes

**Q: ¿Puedo procesar varios archivos EPS en lote?**  
A: Sí. Envuelve la lógica de recorte o redimensionado dentro de un bucle y reutiliza el mismo objeto `Page` para cada archivo.

**Q: ¿Aspose.Page admite EPS con imágenes raster incrustadas?**  
A: Absolutamente. Tanto los elementos vectoriales como los raster se conservan durante la manipulación.

**Q: ¿Es posible convertir EPS a otros formatos (PDF, PNG) después de editar?**  
A: Sí. Después de recortar o redimensionar, puedes llamar a `page.save("output.pdf", SaveFormat.Pdf)` o `SaveFormat.Png`.

**Q: ¿Qué ocurre si el archivo EPS está corrupto?**  
A: La API lanza una `FileFormatException`. Captura la excepción y valida el archivo de origen antes de procesarlo.

**Q: ¿Necesito cerrar manualmente algún recurso?**  
A: La clase `Page` implementa `Closeable`; usa un bloque try‑with‑resources o llama a `page.close()` para liberar memoria.
