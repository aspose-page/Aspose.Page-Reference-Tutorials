## ¿Qué es un degradado radial en PostScript?
Un degradado radial mezcla colores desde un punto focal hacia afuera de forma circular. En PostScript, esto se logra definiendo un diccionario de shading que describe los colores de inicio y fin, el radio y el punto central. Aspose.Page simplifica esto exponiendo una API fluida que genera el código PostScript necesario detrás de escena.

## ¿Por qué usar Aspose.Page para crear un degradado radial?
- **Velocidad:** Escribe unas pocas líneas de Java en lugar de crear manualmente código PostScript complejo.  
- **Precisión:** Controla los stops de color, el radio y la matriz de transformación con métodos tipados.  
- **Portabilidad:** El PostScript generado funciona en impresoras, visores y convertidores.  
- **Escalabilidad:** Combina múltiples degradados (diagonal, horizontal, vertical) en un solo documento.
