## Qué es “cómo agregar texto” en XPS?

Agregar texto a un documento XPS significa insertar programáticamente caracteres imprimibles en una ubicación específica, con una fuente, tamaño y estilo determinados. Con Aspose.Page, este proceso se reduce a unas pocas líneas de código Java, eliminando la necesidad de manipular manualmente el marcado XPS de bajo nivel.

## Guía paso a paso

### Paso 1: Configura tu proyecto
Agrega la dependencia de Aspose.Page a tu `pom.xml` (o al fragmento equivalente de Gradle). Esto te brinda acceso a las clases de manipulación XPS.

### Paso 2: Carga o crea un documento XPS
Crea una nueva instancia de `Document` o abre un archivo XPS existente usando `Document.load(...)`. Este objeto representa el lienzo donde colocarás el texto.

### Paso 3: Prepara el formato de texto
Instancia un objeto `Font`, elige el tamaño, estilo y color deseados. Este paso define cómo aparecerá el texto agregado.

### Paso 4: Agrega texto a una página
Utiliza el método `Graphics.drawString`, especificando la cadena, la fuente, el pincel (color) y las coordenadas en la página. Este es el núcleo de **cómo agregar texto**.

### Paso 5: Guarda el documento actualizado
Llama a `document.save("output.xps")` para escribir los cambios en el disco. El archivo resultante contiene el texto recién insertado listo para visualizarse o imprimirse.

## Problemas comunes y soluciones
- **El texto aparece fuera de la página:** Verifica el sistema de coordenadas (XPS usa puntos; 1 pulgada = 72 puntos).  
- **La fuente no se muestra correctamente:** Asegúrate de que la fuente esté incrustada o disponible en el sistema de destino.  
- **Ralentización del rendimiento en documentos grandes:** Usa `document.optimizeResources()` después de inserciones masivas de texto.

## Preguntas frecuentes

**Q:** *¿Puedo agregar texto multilingüe (p. ej., árabe o chino) con Aspose.Page?*  
**A:** Sí. Aspose.Page soporta completamente Unicode, por lo que puedes renderizar scripts de derecha a izquierda y caracteres CJK usando la fuente adecuada.

**Q:** *¿Necesito cerrar el documento manualmente?*  
**A:** La clase `Document` implementa `Closeable`; usar un bloque try‑with‑resources garantiza que se cierre automáticamente.

**Q:** *¿Existe una forma de agregar texto en lote a muchas páginas?*  
**A:** Recorre las páginas aplicando la misma lógica `Graphics.drawString`. Para lotes grandes, llama a `document.optimizeResources()` después del bucle.

**Q:** *¿Qué opciones de licencia están disponibles?*  
**A:** Aspose ofrece una licencia de evaluación gratuita y varias licencias comerciales (desarrollador, sitio, empresa). Elige la que se ajuste a tu modelo de despliegue.

**Q:** *¿Puedo convertir el XPS final a PDF?*  
**A:** Sí. Usa Aspose.PDF o las funciones de conversión integradas de Aspose.Page para transformar XPS a PDF, PNG u otros formatos.
