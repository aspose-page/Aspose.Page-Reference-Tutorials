## Consejos y mejores prácticas
- **Pro tip:** Siempre establece el nivel de PostScript (p. ej., Level 3) al inicio del documento para garantizar la compatibilidad con impresoras modernas.  
- **Avoid pitfalls:** Olvidar incrustar fuentes personalizadas puede provocar que el impresor utilice fuentes de sustitución. Usa la Font API para incrustar fuentes TrueType u OpenType.  
- **Performance tip:** Reutiliza el mismo objeto `Graphics` para dibujar varios elementos en una página y reducir la sobrecarga.
