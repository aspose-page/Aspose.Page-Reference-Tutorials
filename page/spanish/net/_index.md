## Introducción

¿Listo para **crear documentos XPS** de forma rápida y fiable? Este centro reúne todos los tutoriales de Aspose.Page para .NET que necesitarás, desde lo básico de cargar un documento hasta efectos visuales avanzados. ## Respuestas rápidas
- **¿Qué puedo crear?** Archivos XPS, PostScript y EPS con gráficos y texto enriquecidos.  
- **¿Qué versiones de .NET son compatibles?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.  
- **¿Necesito una licencia?** Se requiere una licencia medida o completa para uso en producción.  
- **¿Puedo convertir XPS a PDF?** Sí – consulta el tutorial “merge XPS to PDF”.  
- **¿Hay código de ejemplo?** Todos los tutoriales incluyen fragmentos listos para ejecutar (preservados a continuación).

## ¿Qué es “crear documento XPS”?
Crear un documento XPS significa generar programáticamente un archivo de diseño fijo que preserva la fidelidad de la página en todas las plataformas Windows. Aspose.Page para .NET ofrece una API fluida para dibujar formas, agregar texto, incrustar imágenes y aplicar efectos avanzados, todo sin necesidad de herramientas de Adobe.

## ¿Por qué usar Aspose.Page para .NET?
- **Control total** sobre gráficos vectoriales, degradados, texturas y tickets de impresión.  
- **Sin dependencias externas** – biblioteca .NET pura.  
- **Alto rendimiento** para el procesamiento masivo de grandes conjuntos de documentos.  
- **Compatibilidad multiplataforma** – convierte fácilmente a PDF, PNG o EPS.

## Preguntas frecuentes

**Q: ¿Cómo inicio un nuevo documento XPS desde cero?**  
A: Use la clase `XpsDocument`, cree una página y agregue gráficos o texto mediante la API `XpsPage`.

**Q: ¿Puedo aplicar un relleno degradado a formas en un documento XPS?**  
A: Sí – la clase `GradientBrush` le permite definir degradados lineales o radiales y aplicarlos a cualquier objeto de dibujo.

**Q: ¿Cuál es la mejor manera de fusionar varios archivos XPS en un solo PDF?**  
A: Cargue cada XPS con `XpsDocument`, luego use `PdfSaveOptions` para exportar y combinarlos, como se muestra en el tutorial “merge XPS to PDF”.

**Q: ¿Cómo incrusto un ticket de impresión personalizado en un archivo XPS?**  
A: Cree un objeto `PrintTicket`, configure los ajustes deseados y adjúntelo a la página usando `XpsPage.PrintTicket`.

**Q: ¿Es posible recortar una imagen EPS antes de insertarla?**  
A: Absolutamente – la clase `Image` admite recorte mediante la propiedad `CropRectangle`, demostrado en la sección “crop EPS image”.
