## Preguntas frecuentes

**Q: ¿Existe un límite en la cantidad de páginas que puedo agregar a un documento PostScript?**  
A: Aspose.Page no impone un límite estricto; las únicas restricciones son la memoria y el tamaño del archivo.

**Q: ¿Necesito convertir XPS a PDF antes de agregar páginas?**  
A: No, puede agregar páginas directamente a archivos XPS usando los mismos métodos de API que para PS.

**Q: ¿Puedo conservar las anotaciones existentes al eliminar una página?**  
A: Sí, la biblioteca mantiene automáticamente las anotaciones en las páginas restantes.

**Q: ¿Qué pasos de solución de problemas debo seguir si una página recién agregada aparece en blanco?**  
A: Verifique que el flujo de contenido esté correctamente poblado y que el tamaño de la página coincida con la configuración DPI del documento.

**Q: ¿Hay proyectos de ejemplo que pueda consultar?**  
A: La documentación de Aspose.Page incluye ejemplos listos para ejecutar tanto para la manipulación de páginas PS como XPS.
