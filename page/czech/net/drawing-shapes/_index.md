# Aspose.Page .NET – Kreslení tvarů

## Úvod

Aspose.Page .NET usnadňuje vývojářům kreslení tvarů v .NET aplikacích, ať už pracujete s PostScript (PS) nebo XPS soubory. V tomto průvodci se dozvíte, jak přidávat kruhy, elipsy, obdélníky a další, a zároveň se naučíte **jak přidávat tvary .NET** styl. Prozkoumejme možnosti a zjistěme, proč je kreslení tvarů s Aspose.Page .NET jak výkonné, tak intuitivní.

## Quick Answers
- **Co Aspose.Page .NET dělá?** Umožňuje programové vytváření a manipulaci s PS a XPS dokumenty, včetně kreslení geometrických tvarů.  
- **Jaké tvary mohu kreslit?** Kruhy, elipsy, obdélníky a vlastní cesty.  
- **Potřebuji licenci?** K dispozici je bezplatná zkušební verze; pro produkční použití je vyžadována komerční licence.  
- **Jaké verze .NET jsou podporovány?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.  
- **Existuje ukázkový kód?** Ano – každý odkazovaný tutoriál poskytuje připravené příklady ke spuštění.

## What is Aspose.Page .NET?
Aspose.Page .NET je .NET knihovna, která vám umožňuje generovat a upravovat PostScript a XPS dokumenty bez potřeby nástrojů Adobe. Nabízí bohaté API pro kreslení tvarů, aplikaci barev, gradientů a správu rozvržení stránky – vše z čistého, spravovaného kódu.

## Why draw shapes .NET with Aspose.Page?
- **Podpora více formátů:** Napište jednou, výstup do PS nebo XPS.  
- **Vysoká věrnost:** Vektorová grafika si zachovává kvalitu při jakémkoli měřítku.  
- **Žádné externí závislosti:** Čistý .NET, nevyžaduje nativní knihovny.  
- **API přátelské vývojářům:** Plynulé metody a jasné pojmenování usnadňují **kreslení tvarů .NET** aplikacím.

## Frequently Asked Questions

**Q: Mohu použít Aspose.Page .NET v komerční aplikaci?**  
A: Ano, platná licence Aspose umožňuje komerční použití; k vyhodnocení je k dispozici bezplatná zkušební verze.

**Q: Musím instalovat nějaké nativní komponenty?**  
A: Ne, Aspose.Page .NET je čistá spravovaná knihovna – stačí odkazovat na NuGet balíček.

**Q: Je možné kombinovat tvary s textem na stejné stránce?**  
A: Rozhodně. API vám umožní kreslit tvary, poté přidat textové objekty a řídit Z‑order podle potřeby.

**Q: Jak zacházet s velkými dokumenty s mnoha tvary?**  
A: Použijte přetížení `Document.Save` s bufferováním streamu a zvažte rozdělení stránek, aby byl nízký odběr paměti.

**Q: Podporuje Aspose.Page průhlednost a gradienty?**  
A: Ano, oba API pro PS i XPS zahrnují gradientové štětce a alfa kompozici pro bohaté vizuální efekty.
