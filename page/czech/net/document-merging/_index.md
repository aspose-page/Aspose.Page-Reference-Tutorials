# Sloučení dokumentů

## Úvod

Sloučte své dokumenty a dosáhněte plynulé správy dokumentů s Aspose.Page pro .NET. **Pokud potřebujete sloučit XPS do PDF**, tento průvodce vám přesně ukáže, jak to provést – rychle a spolehlivě. Objevte sílu sloučení dokumentů s našimi komplexními tutoriály.

## Rychlé odpovědi
- **Co znamená „sloučit XPS do PDF“?** Kombinuje jeden nebo více XPS souborů do jediného PDF dokumentu.  
- **Která knihovna provádí konverzi?** Aspose.Page pro .NET poskytuje nativní podporu formátů XPS a PDF.  
- **Potřebuji licenci?** Bezplatná zkušební verze funguje pro hodnocení; pro produkční nasazení je vyžadována komerční licence.  
- **Podporované verze .NET?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.  
- **Typická doba implementace?** Přibližně 10‑15 minut pro základní sloučení.

## Co je sloučení XPS do PDF?

Sloučení XPS do PDF znamená převzetí jednoho nebo více XPS (XML Paper Specification) souborů – fixního formátu dokumentu od Microsoftu – a jejich spojení do jediného, vysoce kvalitního PDF souboru. To je užitečné pro archivaci, tisk nebo sdílení dokumentů při zachování věrnosti rozvržení.

## Proč použít Aspose.Page pro .NET?

- **Nativní podpora XPS** – Není potřeba žádných třetích konvertorů.  
- **Výstup PDF vysoké kvality** – Zachovává vektorovou grafiku, písma a rozvržení.  
- **Dávkové zpracování** – Snadno sloučíte více XPS souborů jedním voláním.  
- **Cross‑platform** – Funguje na Windows, Linuxu a macOS s .NET Core/.NET 5+.

## Často kladené otázky

**Q: Mohu sloučit jak PostScript, tak XPS soubory do jednoho PDF?**  
A: Ano. Aspose.Page vám umožní přidat stránky z obou formátů do jednoho PDF dokumentu před uložením.

**Q: Potřebuji instalovat další software pro práci s XPS?**  
A: Ne. Aspose.Page pro .NET obsahuje nativní podporu XPS, takže není potřeba žádná další instalace.

**Q: Jak velké mohou být zdrojové XPS soubory?**  
A: Knihovna zvládá velké soubory, ale pro velmi velké dokumenty zvažte jejich zpracování po dávkách, aby se snížila spotřeba paměti.

**Q: Je výsledné PDF prohledávatelné?**  
A: Rozhodně. Textový obsah z původních XPS nebo PostScript souborů je zachován a je prohledávatelný v generovaném PDF.

**Q: Jaké licenční možnosti jsou k dispozici?**  
A: Aspose nabízí bezplatnou zkušební verzi pro hodnocení a různé komerční licenční modely pro produkční použití.
