## Jak generovat PostScript v Javě?
The process is straightforward:

1. **Create a Document** – vytvořte instanci třídy `Document` poskytované knihovnou Aspose.Page.  
2. **Define Page Settings** – nastavte velikost stránky, orientaci a okraje podle požadavků výstupu.  
3. **Add Content** – použijte kreslicí API k umístění textu, obrázků a vektorové grafiky.  
4. **Save as .ps** – zavolejte metodu `save` s volbou `SaveFormat.POSTSCRIPT`.
