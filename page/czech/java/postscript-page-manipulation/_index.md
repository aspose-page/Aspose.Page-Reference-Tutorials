## Jak převést PostScript na PDF a přidat stránky
Když potřebujete sloučit několik PostScript souborů nebo je převést na PDF pro distribuci, Aspose.Page nabízí jednosměrný přístup. Můžete načíst soubor `.ps`, zavolat metodu pro konverzi a vytvořit PDF, a poté použít stejné API k přidání dalších stránek. Tento pracovní postup eliminuje potřebu externích nástrojů a vše zůstane uvnitř vaší Java aplikace.

## Často kladené otázky

**Q: Mohu přidat stránky do existujícího PostScript souboru, aniž bych ztratil jeho původní obsah?**  
A: Ano. Aspose.Page vkládá nové stránky a zachovává veškerý existující obsah, písma a grafiku.

**Q: Je možné zkopírovat stránku z jednoho PostScript dokumentu do druhého?**  
A: Rozhodně. API vám umožní importovat stránky z libovolného zdrojového dokumentu a umístit je do cílového souboru.

**Q: Do jaké souborové formáty mohu po přidání stránek převést finální dokument?**  
A: Knihovna může výsledek uložit jako PostScript, PDF nebo XPS, což vám poskytuje flexibilitu pro následné zpracování.

**Q: Podporuje knihovna přidávání obrázků nebo vektorové grafiky na nové stránky?**  
A: Ano. Pomocí stejného API můžete kreslit tvary, vkládat obrázky a vykreslovat text na nově vytvořených stránkách.

**Q: Existují nějaká omezení velikosti dokumentů při přidávání stránek?**  
A: Knihovna efektivně pracuje s velkými soubory, ale pro velmi velké dokumenty se doporučuje používat 64‑bitovou JVM.

**Q: Jak sloučím více PostScript souborů před konverzí do PDF?**  
A: Použijte metodu `Document.append()` k kombinaci zdrojových dokumentů a poté zavolejte `save("output.pdf")` k provedení konverze.
