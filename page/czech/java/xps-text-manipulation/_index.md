## Jak přidat text pomocí přístupu aspose page add text?
`Page` představuje jednu stránku v XPS dokumentu. Třída `TextFragment` zapouzdřuje kus textu, který může být umístěn na stránku. Načtěte XPS dokument, vytvořte objekt `Page` a použijte třídu `TextFragment` k vložení požadovaného řetězce. Tento **aspose page add text** pracovní postup vyžaduje pouze tři volání API a funguje konzistentně napříč všemi podporovanými platformami, což vám umožní rychle přidat text při zachování existujícího rozvržení a formátování.
