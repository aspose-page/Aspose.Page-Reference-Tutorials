## 常见问题

**Q: 我可以在 Web 应用程序中使用 Aspose.Page 进行 XPS 到 PDF 转换吗？**  
A: 当然可以。该库是线程安全的，能够在 servlet 容器或 Spring Boot 服务中良好运行。

**Q: 我可以转换的 XPS 文件是否有大小限制？**  
A: API 本身没有硬性限制，但对于非常大的文档，需要考虑 JVM 堆大小。

**Q: 我需要在服务器上安装额外的字体吗？**  
A: Aspose.Page 默认使用系统字体。如果源 XPS 引用了自定义字体，请确保已安装该字体或将其嵌入 XPS 中。

**Q: 我如何处理受密码保护的 XPS 文件？**  
A: 使用 `LoadOptions` 类在加载 XPS 文档时提供密码。

**Q: 我能在不丢失矢量图形的情况下将 XPS 转换为 PDF 吗？**  
A: 可以。Aspose.Page 保留矢量图形，确保生成的 PDF 与原始 XPS 完全相同。
