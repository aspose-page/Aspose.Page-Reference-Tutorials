- **什么是径向渐变？** 从中心点向外辐射的平滑颜色过渡。

## 什么是 PostScript 中的径向渐变？
径向渐变以圆形方式从焦点向外混合颜色。在 PostScript 中，这通过定义描述起始颜色、结束颜色、半径和中心点的 shading 字典实现。Aspose.Page 通过提供流式 API，自动生成所需的 PostScript 代码，从而简化此过程。

## 为什么使用 Aspose.Page 创建径向渐变？
- **速度：** 只需几行 Java 代码，无需手动编写复杂的 PostScript。  
- **精度：** 使用类型安全的方法控制颜色停点、半径和变换矩阵。  
- **可移植性：** 生成的 PostScript 可在打印机、查看器和转换器之间通用。  
- **可扩展性：** 可在同一文档中组合多种渐变（对角线、水平、垂直）。

## 常见问题

**问：我可以在同一个 PostScript 文件中组合多种渐变类型吗？**  
答：可以，Aspose.Page 允许您按任意顺序堆叠对角线、水平、径向和垂直渐变。

**问：径向渐变的颜色停点数量有限制吗？**  
答：实际上没有限制；您可以根据需要定义任意数量的停点，但对于极其复杂的阴影，请注意性能。

**问：我需要手动计算渐变矩阵吗？**  
答：不需要，API 提供辅助方法，可自动设置中心点、半径和变换矩阵。

**问：如何在打印前预览渐变效果？**  
答：将生成的 PostScript 保存为 .ps 文件，使用任意 PostScript 查看器（如 Ghostscript 或 Adobe Distiller）打开即可预览。

**问：商业使用有哪些授权选项？**  
答：Aspose 提供永久授权和订阅授权；免费评估授权可用于开发和测试。
