## 常见问题

**问：我可以在不丢失原始内容的情况下向现有 PostScript 文件添加页面吗？**  
答：可以。Aspose.Page 在插入新页面的同时保留所有已有内容、字体和图形。

**问：是否可以将页面从一个 PostScript 文档复制到另一个文档？**  
答：完全可以。API 允许您从任意源文档导入页面并放置到文件中。

**问：添加页面后，我可以将最终文档转换为何种文件格式？**  
答：该库可将结果保存为 PostScript、PDF 或 XPS，满足后续处理的灵活需求。

**问：库是否支持在新页面中添加图像或矢量图形？**  
答：支持。您可以使用相同的 API 在新创建的页面上绘制形状、插入图像并渲染文本。

**问：文档在添加页面时是否有大小限制？**  
答：库能够高效处理大文件，但对于超大文档，建议使用 64 位 JVM。
