## 常见问答

**Q: 我可以批量处理多个 EPS 文件吗？**  
A: 可以。将裁剪或调整大小的逻辑放入循环中，并为每个文件复用同一个 `Page` 对象。

**Q: Aspose.Page 是否支持包含嵌入栅格图像的 EPS？**  
A: 当然。矢量和栅格被保留。

**Q: 编辑后是否可以将 EPS 转换为其他格式（PDF、PNG）？**  
A: 可以。裁剪或调整大小后，您可以调用 `page.save("output.pdf", SaveFormat.Pdf)` 或 `SaveFormat.Png`。

**Q: 如果 EPS 文件损坏怎么办？**  
A: API 会抛出 `FileFormatException`。捕获该异常并在处理前验证源文件。

**Q: 是否需要手动关闭资源？**  
A: `Page` 类实现了 `Closeable`；使用 try‑with‑resources 块或调用 `page.close()` 来释放内存。
