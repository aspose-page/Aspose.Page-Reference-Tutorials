## 如何使用 aspose page add text 方法添加文本？
`Page` 表示 XPS 文档中的单个页面。`TextFragment` 类封装了一段可以放置在页面上的文本。加载 XPS 文档，创建 `Page` 对象，并使用 `TextFragment` 类插入所需的字符串。此 **aspose page add text** 工作流仅需三次 API 调用，并在所有受支持的平台上保持一致，使您能够快速添加文本，同时保留现有的布局和格式。
