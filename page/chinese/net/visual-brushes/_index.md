## 快速答案
- **Visual Brush 是什么？** 可重复使用的绘图元素，可绘制到其他图形对象上。

## Visual Brush 是什么？

Visual Brush 捕获视觉元素——例如形状、文本或图像——并允许您将其作为填充重复用于其他绘图对象。此方法减少重复并确保文档中的样式保持一致。

## 为什么使用 Grid Visual Brush？

Grid Visual Brush 创建平铺的网格图案，可用作背景、指南或装饰元素。当您需要以下情况时，它尤其有用：
- **一致的对齐** 跨多个页面。  
- **轻量级样式** 无需嵌入大图像。  
- **动态缩放** 适应不同页面尺寸。

## 常见问题

**Q:** *我可以在单个文档中组合多个 visual brush 吗？*  
**A:** 是的，您可以叠加不同的 Visual Brushes 以创建复杂的图案和效果。

**Q:** *可以在运行时更改网格大小吗？*  
**A:** 当然。渲染前调整 `CellWidth` 和 `CellHeight` 属性即可动态修改网格。

**Q:** *Visual Brushes 会影响文件大小吗？*  
**A:** 由于它们基于矢量，相比光栅图像几乎不增加额外开销。

**Q:** *可以在 PDF 输出中使用 Visual Brush 吗？*  
**A:** 是的，Aspose.Page 支持 PDF、XPS 以及其他多种格式，同时保留 Visual Brush 定义。

**Q:** *如果刷子未显示，我应该采取哪些故障排除步骤？*  
**A:** 请确认刷子已分配给可绘制对象，并且文档的渲染设置已正确配置。
