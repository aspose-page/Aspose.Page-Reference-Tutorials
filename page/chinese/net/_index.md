- **我可以构建什么？** 具备丰富图形和文本的 XPS、PostScript 和 EPS 文件。

## 什么是 “create XPS document”？
创建 XPS 文档指的是通过编程方式生成一种固定布局的文件，能够在 Windows 平台上保持页面一致性。Aspose.Page for .NET 提供流畅的 API，用于绘制形状、添加文本、嵌入图像以及应用高级效果——全部无需 Adobe 工具。

## 为什么使用 Aspose.Page for .NET？
- **完全控制** 矢量图形、渐变、纹理和打印票据。  
- **无外部依赖** ——纯 .NET 库。  
- **高性能**，适用于大批量文档处理。  
- **跨格式支持** ——轻松转换为 PDF、PNG 或 EPS。

## 常见问题

**Q: 如何从头开始创建新的 XPS 文档？**  
A: 使用 `XpsDocument` 类，创建页面，然后通过 `XpsPage` API 添加图形或文本。

**Q: 能否对 XPS 文档中的形状应用渐变填充？**  
A: 可以——`GradientBrush` 类允许您定义线性或径向渐变，并将其应用于任意绘图对象。

**Q: 将多个 XPS 文件合并为单个 PDF 的最佳方法是什么？**  
A: 使用 `XpsDocument` 加载每个 XPS，然后通过 `PdfSaveOptions` 导出并合并，如 “merge XPS to PDF” 教程所示。

**Q: 如何在 XPS 文件中嵌入自定义打印票据？**  
A: 创建 `PrintTicket` 对象，配置所需设置，然后使用 `XpsPage.PrintTicket` 将其附加到页面。

**Q: 在插入之前能否裁剪 EPS 图像？**  
A: 完全可以——`Image` 类通过 `CropRectangle` 属性支持裁剪，教程中的 “crop EPS image” 部分已有演示。
