# Aspose.Page .NET – 绘制形状

## 介绍

Aspose.Page .NET 让开发者在 .NET 应用程序中轻松绘制形状，无论是处理 PostScript（PS）还是 XPS 文件。在本指南中，您将学习如何添加圆形、椭圆、矩形等，同时了解 **如何以 .NET 方式添加形状**。让我们一起探索其强大而直观的绘图能力。

## 快速回答
- **Aspose.Page .NET 的作用是什么？** 它支持以编程方式创建和操作 PS 与 XPS 文档，包括绘制几何形状。  
- **我可以绘制哪些形状？** 圆形、椭圆、矩形以及自定义路径。  
- **需要许可证吗？** 提供免费试用版；生产环境需购买商业许可证。  
- **支持哪些 .NET 版本？** .NET Framework 4.5+、.NET Core 3.1+、.NET 5/6/7。  
- **有示例代码吗？** 有——每个链接的教程都提供可直接运行的示例。

## 什么是 Aspose.Page .NET？
Aspose.Page .NET 是一个 .NET 库，可在无需 Adobe 工具的情况下生成和编辑 PostScript 与 XPS 文档。它提供丰富的 API，用于绘制形状、应用颜色、渐变以及管理页面布局，全部使用简洁的托管代码实现。

## 为什么使用 Aspose.Page 在 .NET 中绘制形状？
- **跨格式支持：** 编写一次代码，即可输出为 PS 或 XPS。  
- **高保真度：** 矢量图形在任何缩放比例下都保持质量。  
- **无外部依赖：** 纯 .NET 实现，无需本地库。  
- **开发者友好 API：** 流式方法和清晰命名让 **在 .NET 应用中绘制形状** 变得轻松。

## 常见问题

**Q: 我可以在商业应用中使用 Aspose.Page .NET 吗？**  
A: 可以，拥有有效的 Aspose 许可证即可用于商业用途；同时提供免费试用版供评估。

**Q: 需要安装任何本地组件吗？**  
A: 不需要，Aspose.Page .NET 是纯托管库——只需引用 NuGet 包即可。

**Q: 能否在同一页面上将形状与文本组合使用？**  
A: 完全可以。API 允许先绘制形状，再添加文本对象，并根据需要控制 Z‑order。

**Q: 如何处理包含大量形状的大型文档？**  
A: 使用 `Document.Save` 的流缓冲重载，并考虑将页面拆分，以降低内存占用。

**Q: Aspose.Page 是否支持透明度和渐变？**  
A: 支持，PS 与 XPS API 都提供渐变画刷和 alpha 合成，实现丰富的视觉效果。
