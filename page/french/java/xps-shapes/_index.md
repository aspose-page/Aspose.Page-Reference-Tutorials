## Comment dessiner des formes java dans des documents XPS

Dessiner des formes dans XPS avec Java suit un schéma clair : créez un objet `Page`, définissez la géométrie de la forme, appliquez le style (stroke, fill, gradients), et ajoutez‑la à la collection `Graphics` de la page. Cette approche vous donne un contrôle total sur la taille, la position et les effets visuels, vous permettant de créer des documents qui se démarquent.

## Questions fréquentes

**Q : Ai‑je besoin d’une licence payante pour utiliser Aspose.Page pour dessiner des formes ?**  
A: Une version d’essai gratuite est disponible pour l’évaluation, mais une licence commerciale est requise pour les déploiements en production.

**Q : Puis‑je combiner plusieurs formes sur une même page XPS ?**  
A: Oui, vous pouvez ajouter autant de formes que nécessaire ; il suffit d’ajouter chaque forme à la collection graphique de la page.

**Q : Est‑il possible d’appliquer des transformations (rotation, mise à l’échelle) aux formes ?**  
A: Absolument. L’API fournit des matrices de transformation qui vous permettent de faire pivoter, mettre à l’échelle ou incliner n’importe quelle forme.

**Q : Quelles versions de Java sont prises en charge ?**  
A: Aspose.Page for Java prend en charge Java 8 et les environnements d’exécution plus récents.

**Q : Comment exporter le document XPS final après avoir ajouté des formes ?**  
A: Appelez la méthode `save` sur l’objet `Document`, en spécifiant le chemin de fichier et le format souhaités.
