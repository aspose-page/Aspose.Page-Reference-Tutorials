## What is a Radial Gradient in PostScript?
Un dégradé radial mélange les couleurs depuis un point focal vers l'extérieur de manière circulaire. Dans PostScript, cela se réalise en définissant un dictionnaire de shading qui décrit les couleurs de départ et d'arrivée, le rayon et le point central. Aspose.Page simplifie cela en exposant une API fluide qui génère le code PostScript nécessaire en arrière‑plan.

- **Portabilité :** Le PostScript généré fonctionne sur les imprimantes, les visionneuses et les convertisseurs.  
- **Évolutivité :** Combinez plusieurs dégradés (diagonal, horizontal, vertical) dans un même document.
