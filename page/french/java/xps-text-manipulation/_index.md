## Qu'est-ce qu'Aspose.Page pour Java ?
Aspose.Page pour Java est une API puissante qui permet la création, la modification et la conversion de documents XPS de manière programmatique. Elle fournit plus de 30 classes pour manipuler les pages, le texte, les graphiques et les polices, prenant en charge plus de 50 formats d'entrée et de sortie. Avec cette bibliothèque, vous pouvez ajouter, supprimer ou styliser du texte dans les fichiers XPS sans édition manuelle.

## Comment ajouter du texte en utilisant l'approche aspose page add text ?
`Page` représente une page unique au sein d'un document XPS. La classe `TextFragment` encapsule un morceau de texte qui peut être placé sur une page. Chargez un document XPS, créez un objet `Page` et utilisez la classe `TextFragment` pour insérer la chaîne souhaitée. Ce flux de travail **aspose page add text** ne nécessite que trois appels d'API et fonctionne de manière cohérente sur toutes les plateformes prises en charge, vous permettant d'ajouter du texte rapidement tout en préservant la mise en page et le formatage existants.
