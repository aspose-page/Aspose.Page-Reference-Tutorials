## Frequently Asked Questions

**Q : Puis‑je utiliser Aspose.Page pour la conversion XPS en PDF dans une application web ?**  
R : Absolument. La bibliothèque est thread‑safe et fonctionne très bien dans les conteneurs de servlets ou les services Spring Boot.

**Q : Existe‑t‑il une limitation de taille pour les fichiers XPS que je peux convertir ?**  
R : L’API elle‑même n’a pas de limite stricte, mais vous devez tenir compte de la taille du tas JVM pour les documents très volumineux.

**Q : Dois‑je installer des polices supplémentaires sur le serveur ?**  
R : Aspose.Page utilise les polices système par défaut. Si un XPS source fait référence à une police personnalisée, assurez‑vous qu’elle soit installée ou intégrez‑la dans le XPS.

**Q : Comment gérer les fichiers XPS protégés par mot de passe ?**  
R : Utilisez la classe `LoadOptions` pour fournir le mot de passe lors du chargement du document XPS.

**Q : Puis‑je convertir XPS en PDF sans perdre les graphiques vectoriels ?**  
R : Oui. Aspose.Page préserve les graphiques vectoriels, garantissant que le PDF résultant soit identique à l’XPS d’origine.
