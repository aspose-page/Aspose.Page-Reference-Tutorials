## Frequently Asked Questions

**Q : Puis‑je ajouter des images à un fichier XPS existant ?**  
R : Oui. Ouvrez le XPS existant avec `XpsDocument` et utilisez les mêmes API `addImage` pour insérer de nouveaux graphiques.

**Q : Quelles limites de taille d’image s’appliquent ?**  
R : Aspose.Page prend en charge les images jusqu’à 10 Mo ; les fichiers plus volumineux peuvent impacter les performances mais restent traitables.

**Q : Le mosaïquage affecte‑t‑il la taille du fichier ?**  
R : Le mosaïquage répète les mêmes données d’image, donc la taille du fichier augmente très peu — seule la définition de la mosaïque est stockée.

**Q : Existe‑t‑il un moyen de conserver la transparence de l’image ?**  
R : Les images PNG conservent leurs canaux alpha lorsqu’elles sont ajoutées, de sorte que les zones transparentes s’affichent correctement dans le XPS.

**Q : Comment déboguer les problèmes de placement d’image ?**  
R : Utilisez la méthode `saveAsPdf` pour exporter un PDF de prévisualisation ; cela aide à visualiser les coordonnées et l’échelle avant la sortie finale XPS.
