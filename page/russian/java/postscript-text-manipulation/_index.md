## Распространённые проблемы и советы

- **Шрифт не найден:** Убедитесь, что файл шрифта доступен JVM или встроите шрифт с помощью `FontRepository`.  
- **Неправильная кодировка:** Всегда используйте `UnicodeString` при работе с не‑ASCII символами, чтобы избежать искажённого вывода.  
- **Проблемы с позиционированием:** Помните, что PostScript использует начало координат в левом нижнем углу; соответственно корректируйте координаты Y.

## Часто задаваемые вопросы

**Q: Можно ли встроить пользовательский TrueType‑шрифт в файл PostScript?**  
A: Да. Используйте `FontRepository.addFont("path/to/font.ttf")` перед созданием текстового объекта.

**Q: Возможно ли вращать текст?**  
A: Абсолютно. Установите угол вращения текста через метод `TextState.setRotation()`.

**Q: Нужно ли вручную закрывать поток документа?**  
A: Метод `Document.save()` обрабатывает закрытие потока, но всё равно следует освобождать любые пользовательские потоки, которые вы открываете.

**Q: Как Aspose.Page обрабатывает языки, пишущиеся справа налево?**  
A: Предоставив Unicode‑строку с соответствующим скриптом и установив направление текста в `TextState`.

**Q: Какие соображения по производительности важны для больших файлов PostScript?**  
A: Загружайте шрифты один раз, переиспользуйте объекты `TextState` и избегайте создания ненужных временных страниц.
