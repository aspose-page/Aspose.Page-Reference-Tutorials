## Часто задаваемые вопросы

**Q: Можно ли использовать конвертацию Aspose.Page в коммерческом проекте?**  
A: Да. Для использования в продакшене требуется действующая лицензия Aspose, но доступна бесплатная пробная версия для оценки.

**Q: Какие версии Java поддерживаются?**  
A: Aspose.Page for Java работает с Java 8 и более новыми версиями.

**Q: Как управлять разрешением экспортируемых изображений?**  
A: Установите значение DPI в объекте `ImageSaveOptions` перед вызовом метода `save`.

**Q: Можно ли конвертировать несколько файлов PostScript в одной партии?**  
A: Абсолютно — перебирайте свою коллекцию файлов и применяйте одинаковую логику конвертации к каждому документу.

**Q: Что делать, если мой файл PostScript защищён паролем?**  
A: Используйте конструктор `Document`, принимающий объект `LoadOptions`, где можно указать пароль.
