## Распространённые проблемы и решения

| Проблема | Решение |
|----------|---------|
| **Низкое разрешение изображений** | Увеличьте DPI в `ImageSaveOptions` перед вызовом `save()`. |
| **Потребление памяти при больших файлах** | Обрабатывайте страницы по одной и освобождайте каждый `Image` после сохранения. |
| **PostScript, защищённый паролем** | Используйте конструктор `LoadOptions`, принимающий строку пароля. |
| **Неожиданные цвета** | Проверьте настройки цветового пространства в `PdfSaveOptions` или `ImageSaveOptions`. |

## Часто задаваемые вопросы

**Q: Можно ли использовать конвертацию Aspose.Page в коммерческом проекте?**  
A: Да. Для продакшн‑использования требуется действующая лицензия Aspose, но доступна бесплатная пробная версия для оценки.

**Q: Какие версии Java поддерживаются?**  
A: Aspose.Page for Java работает с Java 8 и более поздними версиями.

**Q: Как контролировать разрешение экспортируемых изображений?**  
A: Установите значение DPI в объекте `ImageSaveOptions` перед вызовом метода `save`.

**Q: Можно ли конвертировать несколько файлов PostScript в одной партии?**  
A: Конечно — перебирайте вашу коллекцию файлов и применяйте одну и ту же логику конвертации к каждому документу.

**Q: Что делать, если мой файл PostScript защищён паролем?**  
A: Используйте конструктор `Document`, принимающий объект `LoadOptions`, где можно указать пароль.

**Q: Как уменьшить размер сгенерированного PDF?**  
A: Включите сжатие в `PdfSaveOptions` и рассмотрите возможность понижения разрешения изображений, если высокая детализация не требуется.
