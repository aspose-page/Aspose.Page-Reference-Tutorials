## Что такое радиальный градиент в PostScript?
Радиальный градиент смешивает цвета от фокусной точки наружу по круговой схеме. В PostScript это достигается определением словаря shading, описывающего начальный и конечный цвета, радиус и центральную точку. Aspose.Page упрощает процесс, предоставляя fluent API, которое генерирует необходимый код PostScript за кулисами.

## Зачем использовать Aspose.Page для создания радиального градиента?
- **Скорость:** Пишете несколько строк Java вместо ручного написания сложного PostScript.  
- **Точность:** Управляете остановками цвета, радиусом и матрицей преобразования с помощью типобезопасных методов.  
- **Переносимость:** Сгенерированный PostScript работает на принтерах, в просмотрщиках и конвертерах.  
- **Масштабируемость:** Объединяйте несколько градиентов (диагональный, горизонтальный, вертикальный) в одном документе.

## Как создать радиальный градиент в PostScript
Ниже приведён краткий пошаговый план, который вы будете выполнять в каждом отдельном руководстве:

1. **Создать документ** – Создайте новый объект `Page` с помощью Aspose.Page.  
2. **Определить параметры градиента** – Установите центральную точку, радиус и остановки цвета.  
3. **Применить shading** – Вызовите метод `addShading`, чтобы встроить радиальный градиент в страницу.  
4. **Сохранить результат** – Экспортируйте документ в файл `.ps`, готовый к печати или дальнейшей обработке.

> **Pro tip:** Повторное использование одного и того же объекта `Shading` для нескольких фигур уменьшает расход памяти и делает код более чистым.

## Общие сценарии использования радиального градиента
- **Заполнение фона** для листовок, брошюр или сертификатов, где лёгкий цветовой переход добавляет глубину.  
- **Выделение фокусных точек** таких как логотипы или кнопки призыва к действию в PDF, который позже будет преобразован в PostScript.  
- **Создание реалистичных световых эффектов** в технических схемах или инженерных чертежах.
