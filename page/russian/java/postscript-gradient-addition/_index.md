## Что такое радиальный градиент в PostScript?
Радиальный градиент смешивает цвета от фокусной точки наружу по круговой схеме. В PostScript это достигается определением словаря shading, описывающего начальный и конечный цвета, радиус и центральную точку. Aspose.Page упрощает процесс, предоставляя fluent API, которое генерирует необходимый код PostScript за кулисами.

- **Скорость:** Пишете несколько строк Java вместо ручного написания сложного PostScript.  
- **Точность:** Управляете остановками цвета, радиусом и матрицей трансформации с помощью типобезопасных методов.  
- **Переносимость:** Сгенерированный PostScript работает на принтерах, в просмотрщиках и конвертерах.
