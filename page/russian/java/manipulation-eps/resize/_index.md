# Как изменить размер EPS файлов в Java с помощью Aspose.Page

## Введение
Если вы ищете надёжный способ **как изменить размер EPS** файлов программно, вы попали по адресу. В этом руководстве мы пройдёмся по процессу изменения размеров EPS‑изображений в Java с использованием библиотеки Aspose.Page. Независимо от того, нужно ли удвоить размер, уменьшить его до конкретных измерений или работать с процентами, приведённые ниже шаги дают вам полный контроль над конечными размерами.

## Быстрые ответы
- **Какая библиотека нужна?** Aspose.Page for Java  
- **Можно ли изменять размер в пунктах, дюймах или миллиметрах?** Да — API поддерживает все три единицы измерения плюс проценты.  
- **Нужна ли лицензия для разработки?** Бесплатная пробная версия подходит для тестирования; лицензия требуется для продакшн.  
- **Какая версия Java требуется?** Java 8 или новее.  
- **Является ли код потокобезопасным?** Каждый экземпляр `PsDocument` изолирован, поэтому файлы можно обрабатывать параллельно.

## Предварительные требования
Прежде чем перейти к коду, убедитесь, что у вас есть следующее:

- Установленный Java Development Kit (JDK).  
- Библиотека Aspose.Page for Java. Вы можете скачать её **[здесь](https://releases.aspose.com/page/java/)**.  
- Базовое понимание программирования на Java.  

## Импорт пакетов
В вашем Java‑проекте включите необходимые импорты, чтобы работать с объектами Aspose.Page и стандартными потоками ввода‑вывода.

```java
import java.awt.Dimension;
import java.io.FileInputStream;
import java.io.FileOutputStream;
import com.aspose.eps.PsDocument;
import com.aspose.page.DimensionF;
import com.aspose.page.Units;
```

## Как изменить размер EPS, используя пункты
Ниже представлен полный пошаговый пример, который удваивает размер EPS‑файла, используя пункты в качестве единицы измерения.

### Шаг 1: Настройка входного потока
```java
FileInputStream inputEpsStream = new FileInputStream(dataDir + "input.eps");
```

### Шаг 2: Инициализация объекта `PsDocument`
```java
PsDocument doc = new PsDocument(inputEpsStream);
```

### Шаг 3: Получение текущего размера EPS‑изображения
```java
Dimension oldSize = doc.extractEpsSize();
```

### Шаг 4: Создание выходного потока для изменённого файла
```java
FileOutputStream outputEpsStream = new FileOutputStream(dataDir + "output_resize_points.eps");
```

### Шаг 5: Изменение размера и сохранение EPS с использованием пунктов
```java
doc.resizeEps(outputEpsStream, new Dimension2D.Double(oldSize.width * 2, oldSize.height * 2), Units.Points);
```

## Как изменить размер EPS, используя дюймы
Тот же пятишаговый шаблон применяется; просто замените `Units.Points` на `Units.Inches` и при необходимости скорректируйте коэффициент масштабирования.

## Как изменить размер EPS, используя миллиметры
Снова следуйте тем же шагам, заменив единицу на `Units.Millimeters`. Это удобно, когда нужны метрические размеры для печатных процессов.

## Как изменить размер EPS, используя проценты
Для масштабирования в процентах оставьте единицу как `Units.Percent`. Умножьте исходную ширину и высоту на желаемый процент (например, `0.5` для уменьшения на 50 %).

## Распространённые ошибки и советы
- **Всегда закрывайте потоки** — В продакшн‑коде оборачивайте потоки в `try‑with‑resources`, чтобы избежать блокировок файлов.  
- **Сохраняйте соотношение сторон** — Умножайте как ширину, так и высоту на один и тот же коэффициент, если только вы не хотите искажения.  
- **Проверьте DPI** — Изменение размера не меняет DPI; если нужен иной DPI, скорректируйте его отдельно после изменения размера.

## Заключение
Теперь вы знаете **как изменить размер EPS** файлов в Java с помощью Aspose.Page, будь то пункты, дюймы, миллиметры или проценты. Такая гибкость позволяет интегрировать работу с EPS в автоматизированные конвейеры, настольные утилиты или серверные сервисы.
