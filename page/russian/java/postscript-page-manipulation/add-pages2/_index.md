# Aspose.Page Java Tutorial – установка пользовательского размера страницы при добавлении страниц в PostScript

## Введение
В современных Java‑приложениях часто требуется **установка пользовательского размера страницы** для вывода в формате PostScript — будь то генерация счетов, билетов или пользовательской графики. Aspose.Page for Java упрощает эту задачу. В этом руководстве вы узнаете, как добавлять страницы и задавать пользовательские размеры страниц в документе PostScript шаг за шагом, чтобы каждый раз получать точно нужную разметку.

## Краткие ответы
- **Могу ли я задать разные размеры страниц для каждой страницы?** Да, вы можете открывать страницы с пользовательскими размерами, используя `document.openPage(width, height)`.  
- **Нужна ли лицензия для использования в продакшене?** Для не‑оценочных развертываний требуется действующая лицензия Aspose.Page.  
- **Какие версии Java поддерживаются?** Библиотека работает с Java 8 и новее.  
- **Является ли API потокобезопасным?** Экземпляры Document не являются потокобезопасными; создавайте отдельный `PsDocument` для каждого потока.  
- **Насколько большим может быть файл PostScript?** Aspose.Page эффективно обрабатывает файлы в несколько мегабайт; использование памяти масштабируется с содержимым, а не с количеством страниц.

## Требования
Прежде чем приступить, убедитесь, что у вас есть:

- Базовое понимание программирования на Java.  
- Aspose.Page for Java, добавленный в ваш проект (Maven/Gradle или вручную JAR).  
- Среда разработки Java (IDE, JDK 8+).  

## Импорт пакетов
Для начала импортируйте необходимые классы из библиотеки Aspose.Page.

```java
import java.io.FileOutputStream;
import com.aspose.eps.PsDocument;
import com.aspose.eps.device.PsSaveOptions;
```

## Шаг 1: Создание многостраничного документа PostScript
Сначала создайте новый `PsDocument`, настроенный для нескольких страниц. Это устанавливает поток вывода и сообщает библиотеке, что мы будем работать с многостраничным файлом.

```java
// The path to the documents directory.
String dataDir = "Your Document Directory";
// Create output stream for PostScript document
FileOutputStream outPsStream = new FileOutputStream(dataDir + "AddPages2_outPS.ps");
// Create save options with A4 size
PsSaveOptions options = new PsSaveOptions();
// Set variable that indicates if resulting PostScript document will be multipaged
boolean multiPaged = true;
// Create new multipaged PS Document with one page opened
PsDocument document = new PsDocument(outPsStream, options, multiPaged);
```

## Шаг 2: Добавление содержимого на первую страницу
Когда документ готов, вы можете добавить любое необходимое содержимое на первую страницу. По завершении закройте страницу, чтобы зафиксировать её содержимое.

```java
// Add content to the first page
// Close the first page
document.closePage();
```

## Как задать пользовательский размер страницы
Если размер страницы по умолчанию не подходит, вы можете **задать пользовательский размер страницы** при открытии новой страницы. Это полезно для чеков, этикеток или любой нестандартной разметки.

## Шаг 3: Добавление второй страницы с другим размером
Ниже мы открываем вторую страницу и явно указываем пользовательскую ширину и высоту (в пунктах). Это демонстрирует, как задать пользовательский размер страницы для отдельных страниц.

```java
// Add the second page with a different size
document.openPage(500, 300);
// Add content to the second page
// Close the second page
document.closePage();
```

## Шаг 4: Сохранение документа
Наконец, зафиксируйте изменения, сохранив документ. Все страницы, включая те, что имеют пользовательские размеры, записываются в выходной файл.

```java
// Save the document
document.save();
```

Следуя этим шагам, вы можете без проблем добавлять страницы и **задать пользовательские размеры страниц** в документе Java PostScript с помощью Aspose.Page, получая полный контроль над разметкой каждой страницы.

## Часто задаваемые вопросы
### Могу ли я добавить страницы разных размеров в один документ PostScript?
Да, как показано в этом руководстве, вы можете добавлять страницы с различными размерами в многостраничном документе PostScript.  
### Есть ли ограничения на количество страниц, которые я могу добавить?
Aspose.Page поддерживает добавление практически неограниченного количества страниц в документ.  
### Могу ли я добавить пользовательское содержимое, например изображения или графику, на страницы?
Абсолютно! Aspose.Page позволяет добавлять широкий спектр содержимого, включая текст, изображения и другие графические элементы.  
### Подходит ли Aspose.Page для работы с большими документами?
Да, Aspose.Page разработан для эффективной работы как с небольшими, так и с большими документами.  
### Где я могу найти дополнительные ресурсы и поддержку для Aspose.Page?
Изучите [Aspose.Page documentation](https://reference.aspose.com/page/java/), или посетите [Aspose.Page forum](https://forum.aspose.com/c/page/39) для поддержки сообщества.  

**Additional Q&A**

**В:** *Какие форматы изображений поддерживаются при рисовании на странице PostScript?*  
**О:** Вы можете напрямую встраивать изображения PNG, JPEG, BMP и GIF, используя API рисования.  

**В:** *Как изменить DPI по умолчанию для документа?*  
**О:** Установите `PsSaveOptions.setResolution(int dpi)` перед созданием `PsDocument`.  

**В:** *Могу ли я зашифровать файл PostScript паролем?*  
**О:** Сам PostScript не поддерживает шифрование, но при необходимости вы можете обернуть вывод в PDF и применить настройки безопасности.
