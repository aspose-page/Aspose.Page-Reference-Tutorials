## Как добавить текст с помощью подхода aspose page add text?
`Page` представляет отдельную страницу в XPS‑документе. Класс `TextFragment` инкапсулирует кусок текста, который можно разместить на странице. Загрузите XPS‑документ, создайте объект `Page` и используйте класс `TextFragment` для вставки нужной строки. Этот рабочий процесс **aspose page add text** требует всего три вызова API и стабильно работает на всех поддерживаемых платформах, позволяя быстро добавлять текст, сохраняя существующее расположение и форматирование.
