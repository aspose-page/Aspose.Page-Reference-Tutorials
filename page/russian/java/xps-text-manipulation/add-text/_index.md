# Как добавить текст XPS в Java – Aspose.Page Tutorial

## Введение
Если вам нужно **how to add XPS** текст программно, Aspose.Page for Java предоставляет чистый, высокопроизводительный API для работы с файлами XPS (XML Paper Specification). В этом руководстве мы пройдем процесс создания XPS‑документа, вставки стилизованного текста и сохранения результата — всё с помощью лаконичного, простого кода. Независимо от того, создаете ли вы движок отчетов, генерируете счета‑фактуры или просто хотите наложить текст на страницу, эти шаги быстро помогут вам начать работу.

## Что такое XPS и почему добавлять текст?
XPS (XML Paper Specification) — фиксированный формат документов от Microsoft, похожий на PDF, но основанный на XML. Добавление текста в файл XPS часто требуется, когда нужна точная позиция, например, для этикеток, водяных знаков или динамических данных в отчетах. С помощью Aspose.Page вы можете манипулировать XPS, не вдаваясь в детали низкоуровневой XML‑структуры.

## Импорт пакетов
Начните с импорта классов, необходимых для работы с XPS:

```java
import java.awt.Color;
import com.aspose.xps.XpsDocument;
import com.aspose.xps.XpsFontStyle;
import com.aspose.xps.XpsGlyphs;
import com.aspose.xps.XpsSolidColorBrush;
```

## Шаг 1: Установить каталог документа (create xps file)
Определите, где будет сохранён сгенерированный файл XPS:

```java
String dataDir = "Your Document Directory";
```

## Шаг 2: Создать XPS документ (create xps document)
Создайте новый пустой XPS‑документ:

```java
XpsDocument doc = new XpsDocument();
```

## Шаг 3: Создать кисть для стилизации текста
Сплошная кисть определяет цвет текста. Здесь мы используем чёрный:

```java
XpsSolidColorBrush textFill = doc.createSolidColorBrush(Color.BLACK);
```

## Шаг 4: Добавить глифы — фактический текст (aspose.page add text)
Добавьте текст, который должен появиться в документе. Метод `addGlyphs` позволяет указать шрифт, размер, стиль и точные координаты:

```java
XpsGlyphs glyphs = doc.addGlyphs("Arial", 12, XpsFontStyle.Regular, 300f, 450f, "Hello World!");
glyphs.setFill(textFill);
```

## Шаг 5: Сохранить полученный XPS документ (save xps file)
Наконец, запишите документ на диск:

```java
doc.save(dataDir + "AddText_out.xps");
```

Повторяйте шаг **Add Glyphs** по мере необходимости, чтобы вставлять дополнительные строки, менять шрифты или корректировать позиции.

## Распространённые проблемы и профессиональные советы
- **Неправильные координаты:** XPS использует систему координат, измеряемую в пунктах (1/72 дюйма). Отрегулируйте значения `x` и `y` для точного позиционирования текста.  
- **Шрифт не найден:** Убедитесь, что имя шрифта (например, “Arial”) установлено на машине, где выполняется код.  
- **Исключение лицензии:** Без действующей лицензии сгенерированный XPS может содержать водяной знак. Примените лицензию в начале работы приложения, чтобы избежать этого.
