## Как рисовать фигуры java в XPS документах
Рисование фигур в XPS с помощью Java следует четкому шаблону: создайте объект `Page`, определите геометрию фигуры, примените стилизацию (stroke, fill, gradients) и добавьте её в коллекцию `Graphics` страницы. Такой подход дает вам полный контроль над размером, положением и визуальными эффектами, позволяя создавать документы, которые выделяются.

## Часто задаваемые вопросы

**Q: Нужно ли платить за лицензию, чтобы использовать Aspose.Page для рисования фигур?**  
A: Доступна бесплатная пробная версия для оценки, но для производственного использования требуется коммерческая лицензия.

**Q: Можно ли комбинировать несколько фигур на одной странице XPS?**  
A: Да, можно добавить любое количество фигур; просто добавляйте каждую фигуру в коллекцию graphics страницы.

**Q: Можно ли применять трансформации (поворот, масштаб) к фигурам?**  
A: Абсолютно. API предоставляет матрицы трансформаций, позволяющие вращать, масштабировать или наклонять любую фигуру.

**Q: Какие версии Java поддерживаются?**  
A: Aspose.Page for Java поддерживает Java 8 и более новые среды выполнения.

**Q: Как экспортировать окончательный XPS‑документ после добавления фигур?**  
A: Вызовите метод `save` у объекта `Document`, указав желаемый путь к файлу и формат.
