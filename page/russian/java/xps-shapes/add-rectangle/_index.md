# Установить цвет прямоугольника и добавить прямоугольник в Java XPS

## Введение
В этом руководстве вы узнаете, как **установить цвет прямоугольника** и **добавить прямоугольник** в Java XPS с использованием библиотеки Aspose.Page. Независимо от того, нужно ли вам рисовать простые формы для отчёта или создавать сложную векторную графику, освоение стилизации прямоугольников даёт точный контроль над вашими XPS‑документами.  

## Быстрые ответы
- **Какая библиотека требуется?** Aspose.Page for Java  
- **Можно ли изменить обводку прямоугольника?** Да, используя `setStroke` и `setStrokeThickness`  
- **Поддерживается ли профиль цвета CMYK?** Абсолютно — вы можете загрузить ICC‑профили, как показано  
- **Нужна ли лицензия для продакшн?** Да, коммерческая лицензия требуется для использования не в режиме оценки  
- **Сколько времени занимает реализация?** Обычно менее 10 минут для базового прямоугольника

## Что такое **set rectangle color**?
Установка цвета прямоугольника означает определение цвета заливки или обводки, который будет использоваться при рендеринге формы в окончательном XPS‑документе. С помощью Aspose.Page вы можете использовать RGB, CMYK или даже пользовательские ICC‑цветовые профили для точного соответствия цветов.

## Почему стоит использовать Aspose.Page для **add rectangle java** и **change rectangle stroke**?
- **Полнофункциональный API** — поддерживает как сплошные цвета, так и градиенты.  
- **Кроссплатформенный** — работает на любой Java‑среде без нативных зависимостей.  
- **Широкая поддержка геометрии** — легко создавать сложные контуры, применять трансформации и управлять толщиной обводки.  

## Предварительные требования
Перед тем как погрузиться в код, убедитесь, что у вас есть:

- Базовые знания программирования на Java.  
- Установленная библиотека Aspose.Page. Если нет, скачайте её с [Документация Aspose.Page Java](https://reference.aspose.com/page/java/).  
- Рабочая среда разработки Java (IDE, JDK и т.д.).  

## Импорт пакетов
Чтобы начать, импортируйте необходимые пакеты в ваш Java‑проект. Убедитесь, что библиотека Aspose.Page правильно добавлена в classpath. Вот базовый пример:
```java
import com.aspose.xps.XpsDocument;
import com.aspose.xps.XpsPath;
```

Теперь разберём предоставленный пример на несколько шагов, чтобы **добавить прямоугольник** и **установить его цвет** в Java XPS.

## Шаг 1: Установить каталог документа
```java
// The path to the documents directory.
String dataDir = "Your Document Directory";
```
Замените `"Your Document Directory"` на абсолютный путь, где вы хотите читать/записывать XPS‑файлы.

## Шаг 2: Создать новый XPS‑документ
```java
// Create new XPS Document
XpsDocument doc = new XpsDocument();
```
Эта строка инициализирует новый XPS‑документ, который будет содержать наши фигуры.

## Шаг 3: Добавить прямоугольник с обводкой сплошного цвета CMYK
```java
// CMYK (blue) solid color stroked rectangle in the lower left
XpsPath path = doc.addPath(doc.createPathGeometry("M 20,10 L 220,10 220,100 20,100 Z"));
path.setStroke(doc.createSolidColorBrush(
    doc.createColor(dataDir + "uswebuncoated.icc", 1.0f, 1.000f, 0.000f, 0.000f, 0.000f)));
path.setStrokeThickness(12f);
```
Этот шаг добавляет **прямоугольник с обводкой** сплошного цвета CMYK. Метод `setStroke` задаёт цвет контура прямоугольника, а `setStrokeThickness` управляет толщиной линии — идеально для **изменения обводки прямоугольника**.

### Совет профессионалов:
Если вы предпочитаете цвет RGB, замените вызов `createColor` на `doc.createColor(0, 0, 255)` для сплошной синей заливки.

## Шаг 4: Сохранить полученный XPS‑документ
```java
// Save resultant XPS document
doc.save(dataDir + "AddRectangle_out.xps");
```
Наконец, сохраните XPS‑документ на диск. Файл `AddRectangle_out.xps` теперь содержит прямоугольник с заданными вами цветом и обводкой.

## Распространённые проблемы и решения
| Проблема | Причина | Решение |
|----------|---------|----------|
| **Прямоугольник не виден** | Неправильная геометрия пути или толщина обводки установлена в 0 | Проверьте строку пути (`"M 20,10 L 220,10 220,100 20,100 Z"`) и убедитесь, что `setStrokeThickness` больше 0. |
| **Неправильный цвет** | Использование ICC‑профиля, не соответствующего требуемому цветовому пространству | Используйте `doc.createColor(r, g, b)` для RGB или дважды проверьте путь к ICC‑файлу. |
| **Файл не сохранён** | `dataDir` указывает на несуществующую папку или не имеет прав на запись | Создайте папку заранее или выберите расположение с правом записи. |

## Часто задаваемые вопросы
