# Установка маски непрозрачности в Java XPS с использованием Aspose.Page Java

## Введение
В этом руководстве вы **создадите файлы Java XPS‑документов** и научитесь применять маску непрозрачности, придающую вашим графикам отполованный, полупрозрачный вид. Мы пройдём весь процесс — от инициализации XPS‑документа, добавления холста, рисования прямоугольника до присоединения маски непрозрачности на основе изображения — используя интуитивный API Aspose.Page Java. К концу вы сможете программно генерировать профессиональные XPS‑файлы и повторно использовать маску для любой нужной формы.

- **Можно ли повторно использовать маску для нескольких фигур?** Да, вы можете назначить один и тот же `ImageBrush` нескольким путям.

## Что такое маска непрозрачности в XPS?
Маска непрозрачности — это растровое изображение или вектор, который управляет альфа‑каналом (прозрачностью) каждого пикселя формы. При применении к прямоугольнику части прямоугольника становятся полностью непрозрачными, частично прозрачными или полностью прозрачными, создавая сложные визуальные эффекты без дополнительных слоёв рисования.

## Пошаговое руководство

### Шаг 1: Создать новый XPS‑документ
Сначала создайте новый XPS‑документ, который будет содержать всю нашу графику.

```java
// Create a new XPS document
XpsDocument doc = new XpsDocument();
```

### Шаг 2: Добавить холст
Холст служит поверхностью для рисования внутри страницы XPS.

```java
// New canvas
XpsCanvas canvas = doc.addCanvas();
```

### Шаг 3: Добавить прямоугольник и применить сплошную заливку
Здесь мы создаём путь прямоугольника и задаём ему сплошную красную заливку. Этот прямоугольник позже получит маску непрозрачности.

```java
// Rectangle in the middle left with opacity masked by ImageBrush
XpsPath path = canvas.addPath(doc.createPathGeometry("M 10,180 L 228,180 228,285 10,285"));
path.setFill(doc.createSolidColorBrush(doc.createColor(1.0f, 0.0f, 0.0f)));
```

### Шаг 4: Добавить маску непрозрачности с помощью ImageBrush
Мы загружаем TIFF‑изображение, определяем исходный и целевой прямоугольники и устанавливаем кисть в режим плитки, чтобы маска повторялась при необходимости.

```java
path.setOpacityMask(doc.createImageBrush(dataDir +  "R08SY_NN.tif", 
                    new Rectangle2D.Float(0f, 0f, 128f, 192f), new Rectangle2D.Float(0f, 0f, 64f, 96f)));
((XpsImageBrush)path.getOpacityMask()).setTileMode(XpsTileMode.Tile);
```

### Шаг 5: Сохранить полученный XPS‑документ
Наконец, сохраняем документ на диск. Выходной файл будет содержать прямоугольник с применённой маской непрозрачности.

```java
// Save resultant XPS document
doc.save(dataDir + "OpacityMask_out.xps"); 
```

Тщательно следуйте этим шагам, чтобы внедрить функциональность **add opacity mask** в ваш Java XPS‑документ с помощью Aspose.Page.

## Распространённые проблемы и устранение неполадок
- **Изображение не найдено** — проверьте, что `dataDir` указывает на правильную папку и файл `R08SY_NN.tif` существует.  
- **Маска выглядит сплошной** — убедитесь, что исходное изображение действительно содержит разные альфа‑значения; полностью непрозрачное изображение не покажет прозрачность.  
- **Режим плитки не применяется** — целевой прямоугольник должен быть меньше исходного, чтобы повторение было заметно.
