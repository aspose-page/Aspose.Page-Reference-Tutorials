## Как добавить изображение – пошаговый обзор

Ниже представлена краткая дорожная карта, которой можно следовать после установки библиотеки:

1. **Создайте объект `Document`**, представляющий файл PostScript, который вы хотите отредактировать.  
2. **Создайте объект `Image`** из файла, потока или массива байтов.  
3. **Определите прямоугольник размещения** (X, Y, ширина, высота), где появится изображение.  
4. **Вызовите `document.addImage(image, rect)`**, чтобы внедрить графику.  
5. **Сохраните обновлённый документ** обратно на диск или в поток.

Каждое из этих действий продемонстрировано в связанном учебнике «Add Image in Java PostScript», так что вы можете скопировать‑вставить точные фрагменты кода в свой проект.

## Распространённые ошибки и советы

- **Поддержка форматов изображений** – Убедитесь, что исходное изображение находится в формате, поддерживаемом Aspose (PNG, JPEG, BMP и т.д.).  
- **Система координат** – PostScript использует начало координат в левом нижнем углу; дважды проверьте координаты Y.  
- **Потребление памяти** – Большие изображения могут увеличить использование памяти; рассмотрите возможность снижения разрешения перед вставкой.  
- **Лицензирование** – Запуск без лицензии добавляет водяной знак к результату; всегда применяйте действующую лицензию для продакшна.

## Часто задаваемые вопросы

**В: Могу ли я добавить несколько изображений на одну страницу PostScript?**  
О: Да. Вызывайте метод `addImage` последовательно с разными прямоугольниками размещения.

**В: Поддерживает ли Aspose.Page векторную графику?**  
О: Абсолютно. Вы можете внедрять SVG, EPS или даже сырые команды PostScript рядом с растровыми изображениями.

**В: Какие версии Java совместимы?**  
О: Библиотека работает с Java 8 и новее, включая Java 11, 17 и последующие LTS‑выпуски.

**В: Можно ли вращать изображение при его добавлении?**  
О: Да. Используйте API трансформаций `Matrix`, чтобы задать вращение перед вызовом `addImage`.

**В: Как работать с прозрачными PNG?**  
О: Прозрачные PNG сохраняются автоматически; просто убедитесь, что целевой просмотрщик PostScript поддерживает альфа‑каналы.
