## Что такое текстурный узор в PostScript?

Текстурный узор (также называемый узором‑плиткой) — это переиспользуемый графический объект, который повторяет растровый или векторный тайл по определённой области. В PostScript вы описываете тайл один раз, а затем заполняете формы узором, который интерпретатор автоматически повторяет. Такой подход сохраняет небольшой размер файла, одновременно предоставляя сложные визуальные эффекты.

## Как создать текстурный узор в PostScript

Ниже представлена логическая последовательность, которой вы будете следовать. Каждый шаг включает короткое объяснение; фактический код остаётся без изменений (новые блоки кода не добавляются).

### Шаг 1: Подготовьте окружение
Убедитесь, что у вас в classpath находится последняя версия Aspose.Page for Java JAR и действующий файл лицензии, если вы работаете в продакшн‑режиме.

### Шаг 2: Загрузите растровое изображение, которое будет использоваться как плитка
Используйте класс `Image` для чтения PNG, JPEG или BMP, который будет служить тайлом. Изображение хранится в памяти и позже используется объектом узора.

### Шаг 3: Определите узор‑плитку
Создайте экземпляр `TilingPattern`, задайте его ширину/высоту, соответствующие размерам растрового изображения, и привяжите битмап к потоку содержимого узора. Это сообщает движку PostScript, как повторять тайл, и фактически **определяет узор‑плитку**.

### Шаг 4: Примените узор к графическим объектам
При рисовании фигур (прямоугольников, кругов, путей) установите заливку в виде только что определённого узора‑плитки. Узор автоматически заполнит область фигуры повторяющимся битмапом, позволяя **добавлять текстурный узор** без ручного написания команд PostScript.

### Шаг 5: Сохраните документ PostScript
Вызовите `document.save("output.ps")`, чтобы **сохранить файл PostScript**. Полученный файл содержит определение узора и ссылки на него, готовый к работе с любым совместимым интерпретатором.
