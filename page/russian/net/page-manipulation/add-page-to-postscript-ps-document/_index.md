# Добавление страницы в документ PostScript (PS) с помощью Aspose.Page

## Введение

В разработке на .NET возможность **установить пользовательский размер страницы** и **добавить вторую страницу PS** в документ PostScript (PS) дает вам тонкий контроль над макетом генерируемых печатных материалов, отчетов или графики. Aspose.Page для .NET делает эту задачу простой с чистым, объектно‑ориентированным API. В этом руководстве вы узнаете, как создать многостраничный файл PS, задать пользовательский размер для каждой страницы и сохранить результат — всё это с помощью всего нескольких строк кода на C#.

## Быстрые ответы
- **Могу ли я установить пользовательский размер страницы?** Да — просто передайте ширину и высоту при открытии страницы.  
- **Как добавить вторую страницу PS?** Вызовите `document.OpenPage(width, height)` второй раз.  
- **Какие версии .NET поддерживаются?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6+.  
- **Нужна ли лицензия?** Временная лицензия подходит для тестирования; полная лицензия требуется для продакшна.  
- **Где можно скачать Aspose.Page?** На официальной странице загрузки, ссылка ниже.

## Требования

Прежде чем погрузиться в руководство, убедитесь, что у вас есть следующие требования:

## Шаг 1: Настройте ваш проект

Создайте новый .NET‑проект в Visual Studio и настройте необходимые конфигурации. Убедитесь, что в проекте подключена библиотека Aspose.Page.

## Установка пользовательского размера страницы и добавление второй страницы PS

В этом разделе показано, как **установить пользовательский размер страницы** для каждой страницы и как **добавить вторую страницу ps** в тот же документ.

### Шаг 2: Инициализировать документ

```csharp
// ExStart:1
// The path to the documents directory.
string dataDir = "Your Document Directory";

// Create output stream for PostScript document
using (Stream outPsStream = new FileStream(dataDir + "document1.ps", FileMode.Create))
{
    // Create save options with A4 size
    PsSaveOptions options = new PsSaveOptions();

    // Create a new 2-paged PS Document
    PsDocument document = new PsDocument(outPsStream, options, 2);
```

### Шаг 3: Добавить первую страницу (размер по умолчанию)

```csharp
    // Add the first page
    document.OpenPage();

    // Add content

    // Close the first page
    document.ClosePage();
```

### Шаг 4: Добавить вторую страницу с другим (пользовательским) размером

```csharp
    // Add the second page with a different size
    document.OpenPage(400, 700);

    // Add content

    // Close the second page
    document.ClosePage();
```

### Шаг 5: Сохранить документ

```csharp
    // Save the document
    document.Save();
}
// ExEnd:1
```

Следуйте этим шагам внимательно, и вы успешно **установите пользовательский размер страницы** и добавите **вторую страницу PS** в документ PostScript с помощью Aspose.Page для .NET.

## Почему это важно

- **Точный макет** — Пользовательские размеры страниц позволяют соответствовать спецификациям принтера или создавать уникальные форматы брошюр.  
- **Несколько страниц** — Добавление второй страницы (или более) позволяет создавать многостраничные отчёты без внешних инструментов объединения.  
- **Кросс‑платформенность** — Сгенерированный файл PS может быть отрисован на любом устройстве, совместимом с PostScript, или позже конвертирован в PDF.

## Распространённые ошибки и устранение неполадок

- **Неправильный путь** — Убедитесь, что `dataDir` заканчивается разделителем пути или используйте `Path.Combine`.  
- **Проблемы с лицензией** — Без действующей лицензии библиотека может добавить водяной знак или ограничить количество страниц.  
- **Путаница с единицами** — Ширина и высота измеряются в пунктах (1 пункт = 1/72 дюйма). Регулируйте соответственно.
