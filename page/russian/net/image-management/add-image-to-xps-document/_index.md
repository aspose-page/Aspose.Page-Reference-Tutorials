## Aspose.Page: Добавление изображения в XPS документ

Ниже мы пройдём каждый шаг, необходимый для **create XPS document .NET** и внедрения изображения.

### Шаг 1: Установить каталог документа

Начните с указания пути к каталогу вашего документа. Этот шаг гарантирует, что проект знает, где искать и сохранять файлы.

```csharp
// ExStart:1
string dataDir = "Your Document Directory";
// ExEnd:1
```

### Шаг 2: Создать XPS документ

Создайте новый XPS документ с помощью Aspose.Page для .NET.

```csharp
// ExStart:1
XpsDocument doc = new XpsDocument();
// ExEnd:1
```

### Шаг 3: Добавить изображение в XPS документ

Теперь добавим изображение в XPS документ. В этом примере мы используем образец изображения с именем "QL_logo_color.tif".

```csharp
// ExStart:1
XpsPath path = doc.AddPath(doc.CreatePathGeometry("M 30,20 l 258.24,0 0,56.64 -258.24,0 Z"));
path.RenderTransform = doc.CreateMatrix(0.7f, 0f, 0f, 0.7f, 0f, 20f);
path.Fill = doc.CreateImageBrush(dataDir + "QL_logo_color.tif", new RectangleF(0f, 0f, 258.24f, 56.64f), new RectangleF(50f, 20f, 193.68f, 42.48f));
// ExEnd:1
```

### Шаг 4: Сохранить полученный XPS документ

Сохраните XPS документ с добавленным изображением.

```csharp
// ExStart:1
doc.Save(dataDir + "AddImage_outXPS.xps");
// ExEnd:1
```

Теперь вы успешно добавили изображение в XPS документ с использованием Aspose.Page для .NET!

## Распространённые проблемы и решения

- **Ошибка файл не найден** – проверьте, что `dataDir` указывает на правильную папку и имя файла изображения точно совпадает, включая чувствительность к регистру в Linux.
- **Изображение выглядит искажённым** – скорректируйте коэффициенты масштабирования `CreateMatrix` или параметры `RectangleF`, чтобы управлять размером и соотношением сторон.
- **Неподдерживаемый формат изображения** – Aspose.Page поддерживает распространённые растровые форматы (PNG, JPEG, TIFF). Преобразуйте неподдерживаемые форматы перед использованием `CreateImageBrush`.
