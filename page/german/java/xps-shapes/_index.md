## Was ist eine radial gradient ellipse?
Eine **radial gradient ellipse** kombiniert die geometrische Präzision einer Ellipse mit einem Farbverlauf, der von einem zentralen Punkt nach außen strahlt. Dieser Effekt eignet sich perfekt, um Abschnitte hervorzuheben, schaltflächenähnliche Grafiken zu erstellen oder subtile Branding‑Elemente zu Ihren XPS‑Seiten hinzuzufügen.

## Warum Formen in Java XPS zeichnen?
- Inhalt visuell strukturieren (z. B. Hinweis‑Boxen, Diagramme).

- Rasterbilder durch skalierbare Vektorgrafiken ersetzen, um die Dateigröße klein zu halten.
- Die Berichtserstellung automatisieren, bei der das Layout programmgesteuert definiert wird.

## Häufig gestellte Fragen

**Q: Can I change the gradient colors after the shape is created?**  
A: Ja, Sie können die `GradientStop`‑Sammlung ändern und erneut auf den Strich der Ellipse anwenden.

**Q: Is it possible to add a shadow effect to the rectangle?**  
A: Während XPS keinen nativen Schatten unterstützt, können Sie einen simulieren, indem Sie ein leicht versetztes, halbtransparentes Rechteck hinter dem Original zeichnen.

**Q: Do these methods work with PDF conversion?**  
A: Absolut. Aspose.Page ermöglicht die Konvertierung des XPS-Dokuments zu PDF, wobei alle Vektorformen erhalten bleiben.

**Q: How do I ensure the shapes are DPI‑independent?**  
A: Da XPS Vektorgrafiken verwendet, skalieren Formen automatisch; vermeiden Sie lediglich Rasterbilder innerhalb der Form.

**Q: What if I need to draw more complex polygons?**  
A: Verwenden Sie die `Path`‑Klasse, um benutzerdefinierte Geometrie zu definieren; dieselben Gradient‑ und Strich‑Einstellungen gelten.
