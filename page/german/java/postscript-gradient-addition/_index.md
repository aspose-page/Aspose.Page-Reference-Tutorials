## Was ist ein radialer Farbverlauf in PostScript?
Ein radialer Farbverlauf mischt Farben von einem Fokuspunkt aus nach außen in kreisförmiger Weise. In PostScript wird dies erreicht, indem ein Shading‑Dictionary definiert wird, das die Start‑ und Endfarben, den Radius und den Mittelpunkt beschreibt. Aspose.Page vereinfacht dies, indem es eine fluente API bereitstellt, die den erforderlichen PostScript‑Code im Hintergrund generiert.

- **Präzision:** Steuern Sie Farb‑Stops, Radius und Transformationsmatrix mit typsicheren Methoden.  
- **Portabilität:** Generiertes PostScript funktioniert auf Druckern, Viewern und Konvertern.  
- **Skalierbarkeit:** Kombinieren Sie mehrere Farbverläufe (diagonal, horizontal, vertikal) in einem einzigen Dokument.
