## Cos'è l'unione XPS in PDF?
Unire XPS in PDF significa prendere uno o più file XPS (XML Paper Specification) — il formato di documento a layout fisso di Microsoft — e combinarli in un unico file PDF di alta qualità. Questo è utile per l'archiviazione, la stampa o la condivisione di documenti mantenendo la fedeltà del layout.

- **Elaborazione batch** – Unisce facilmente più file XPS in una singola chiamata.  
- **Cross‑platform** – Funziona su Windows, Linux e macOS con .NET Core/.NET 5+.
