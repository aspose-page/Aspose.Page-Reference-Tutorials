## Introduzione

Pronto a **creare documenti XPS** in modo rapido e affidabile? Questo hub raccoglie tutti i tutorial di Aspose.Page per .NET di cui avrai bisogno—dalle basi del caricamento di un documento agli effetti visivi avanzati. ## Risposte rapide
- **Cosa posso creare?** XPS, PostScript e file EPS con grafiche e testo ricchi.  
- **Quali versioni .NET sono supportate?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Cos'è “creare documento XPS”?
Creare un documento XPS significa generare programmaticamente un file a layout fisso che preserva la fedeltà della pagina su piattaforme Windows. Aspose.Page per .NET fornisce un'API fluida per disegnare forme, aggiungere testo, incorporare immagini e applicare effetti avanzati—tutto senza necessità di strumenti Adobe.

## Perché usare Aspose.Page per .NET?
- **Controllo totale** su grafica vettoriale, sfumature, texture e ticket di stampa.  
- **Nessuna dipendenza esterna** – libreria .NET pura.  
- **Alte prestazioni** per l'elaborazione in blocco di grandi insiemi di documenti.  
- **Supporto multi‑formato** – converti facilmente in PDF, PNG o EPS.

## Domande frequenti

**Q: Come avvio un nuovo documento XPS da zero?**  
A: Usa la classe `XpsDocument`, crea una pagina e aggiungi grafica o testo tramite l'API `XpsPage`.

**Q: Posso applicare un riempimento sfumato alle forme in un documento XPS?**  
A: Sì – la classe `GradientBrush` consente di definire sfumature lineari o radiali e applicarle a qualsiasi oggetto di disegno.

**Q: Qual è il modo migliore per unire più file XPS in un unico PDF?**  
A: Carica ogni XPS con `XpsDocument`, poi usa `PdfSaveOptions` per esportare e combinarli, come mostrato nel tutorial “merge XPS to PDF”.

**Q: Come incorporo un custom print ticket in un file XPS?**  
A: Crea un oggetto `PrintTicket`, configura le impostazioni desiderate e allegalo alla pagina usando `XpsPage.PrintTicket`.

**Q: È possibile ritagliare un'immagine EPS prima di inserirla?**  
A: Assolutamente – la classe `Image` supporta il ritaglio tramite la proprietà `CropRectangle`, come dimostrato nella sezione “crop EPS image”.
