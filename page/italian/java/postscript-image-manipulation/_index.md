## Come aggiungere un'immagine – Panoramica passo‑passo

Di seguito trovi una roadmap concisa da seguire dopo l'installazione della libreria:

1. **Crea un oggetto `Document`** che rappresenta il file PostScript che desideri modificare.  
2. **Istanzia un oggetto `Image`** da un file, stream o array di byte.  
3. **Definisci il rettangolo di posizionamento** (X, Y, larghezza, altezza) dove apparirà l'immagine.  
4. **Chiama `document.addImage(image, rect)`** per incorporare la grafica.  
5. **Salva il documento aggiornato** su disco o su uno stream.  

Ciascuna di queste azioni è dimostrata nel tutorial collegato “Add Image in Java PostScript”, così potrai copiare‑incollare gli snippet di codice esatti nel tuo progetto.

## Problemi comuni e suggerimenti

- **Supporto del formato immagine** – Assicurati che l'immagine di origine sia in un formato supportato da Aspose (PNG, JPEG, BMP, ecc.).  
- **Sistema di coordinate** – PostScript utilizza un'origine in basso‑a‑sinistra; verifica attentamente le coordinate Y.  
- **Utilizzo della memoria** – Immagini grandi possono aumentare il consumo di memoria; considera il down‑sampling prima dell'inserimento.  
- **Licenza** – L'esecuzione senza licenza aggiunge una filigrana all'output; applica sempre una licenza valida per la produzione.

## Domande frequenti

**D: Posso aggiungere più immagini alla stessa pagina PostScript?**  
R: Sì. Chiama il metodo `addImage` più volte con diversi rettangoli di posizionamento.

**D: Aspose.Page supporta anche la grafica vettoriale?**  
R: Assolutamente. Puoi incorporare SVG, EPS o anche comandi PostScript grezzi insieme a immagini raster.

**D: Quali versioni di Java sono compatibili?**  
R: La libreria funziona con Java 8 e versioni successive, inclusi Java 11, 17 e le successive versioni LTS.

**D: È possibile ruotare un'immagine durante l'aggiunta?**  
R: Sì. Usa l'API di trasformazione `Matrix` per impostare la rotazione prima di chiamare `addImage`.

**D: Come gestisco i PNG trasparenti?**  
R: I PNG trasparenti vengono preservati automaticamente; assicurati solo che il visualizzatore PostScript di destinazione supporti i canali alfa.
