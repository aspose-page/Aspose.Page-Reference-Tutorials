## Come aggiungere testo usando l'approccio aspose page add text?
`Page` rappresenta una singola pagina all'interno di un documento XPS. La classe `TextFragment` incapsula un pezzo di testo che può essere posizionato su una pagina. Carica un documento XPS, crea un oggetto `Page` e utilizza la classe `TextFragment` per inserire la stringa desiderata. Questo flusso di lavoro **aspose page add text** richiede solo tre chiamate API e funziona in modo coerente su tutte le piattaforme supportate, permettendoti di aggiungere testo rapidamente mantenendo il layout e la formattazione esistenti.
