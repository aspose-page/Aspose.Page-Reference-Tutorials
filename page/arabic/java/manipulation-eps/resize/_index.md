# كيفية تغيير حجم ملفات EPS في Java باستخدام Aspose.Page

## مقدمة
إذا كنت تبحث عن طريقة موثوقة **كيفية تغيير حجم EPS** برمجياً، فقد وصلت إلى المكان الصحيح. في هذا البرنامج التعليمي سنستعرض عملية تغيير حجم صور EPS في Java باستخدام مكتبة Aspose.Page. سواء كنت تحتاج إلى مضاعفة الحجم، أو تقليصه إلى قياس محدد، أو العمل بالنسبة المئوية، فإن الخطوات أدناه تمنحك التحكم الكامل في أبعاد الإخراج. فهم **كيفية تغيير حجم eps** أمر أساسي عندما تحتاج إلى تعديل الرسومات لتتناسب مع تخطيطات الطباعة أو دقة الشاشات المختلفة.

## إجابات سريعة
- **ما المكتبة المطلوبة؟** Aspose.Page for Java  
- **هل يمكنني تغيير الحجم بالنقاط أو البوصات أو المليمترات؟** نعم – الـ API يدعم الوحدات الثلاث بالإضافة إلى النسب المئوية.  
- **هل أحتاج إلى رخصة للتطوير؟** النسخة التجريبية المجانية تكفي للاختبار؛ الرخصة مطلوبة للإنتاج.  
- **ما نسخة Java المطلوبة؟** Java 8 أو أحدث.  
- **هل الكود آمن للاستخدام المتعدد الخيوط؟** كل كائن `PsDocument` معزول، لذا يمكنك معالجة الملفات بشكل متوازي.  

## ما هو EPS ولماذا نحتاج لتغيير حجمه؟
Encapsulated PostScript (EPS) هو تنسيق رسومات متجهية شائع يستخدم في النشر والطباعة. أحياناً يتم إنشاء ملف EPS الأصلي بحجم لا يتطابق مع الهدف المطلوب – على سبيل المثال، قد يكون الشعار مصممًا بـ 72 نقطة ويحتاج إلى أن يكون 144 نقطة لكتيب أكبر. معرفة **كيفية تغيير حجم eps** تتيح لك الحفاظ على جودة المتجهات مع تعديل الأبعاد لتناسب أي سير عمل.

## لماذا نستخدم Aspose.Page لتغيير حجم EPS؟
- **تحكم كامل في الوحدات** – نقاط، بوصات، مليمترات أو نسب مئوية.  
- **بدون تبعيات خارجية** – API نقي بلغة Java، لا مكتبات أصلية.  
- **أداء عالي** – مناسب للمعالجة الدفعية على الخوادم.  
- **يحافظ على دقة المتجهات** – يبقى الناتج قابلاً للتكبير دون تحويله إلى نقطية.

## استيراد الحزم
في مشروع Java الخاص بك، أضف الاستيرادات المطلوبة لتتمكن من العمل مع كائنات Aspose.Page وتدفقات I/O القياسية.

```java
import java.awt.Dimension;
import java.io.FileInputStream;
import java.io.FileOutputStream;
import com.aspose.eps.PsDocument;
import com.aspose.page.DimensionF;
import com.aspose.page.Units;
```

## كيفية تغيير أبعاد EPS باستخدام وحدات مختلفة
تتيح لك المكتبة **تغيير أبعاد eps** ببساطة عن طريق اختيار تعداد `Units` المناسب. أدناه سنستعرض نمط الخطوات الخمسة نفسه للنقاط، البوصات، المليمترات، والنسب المئوية. الشيء الوحيد الذي يتغير هو الوحدة التي تمررها إلى `resizeEps`.

## كيفية تغيير حجم EPS باستخدام النقاط
فيما يلي مثال كامل خطوة بخطوة يضاعف حجم ملف EPS باستخدام النقاط كوحدة قياس.

### الخطوة 1: إعداد تدفق الإدخال
```java
FileInputStream inputEpsStream = new FileInputStream(dataDir + "input.eps");
```

### الخطوة 2: تهيئة كائن `PsDocument`
```java
PsDocument doc = new PsDocument(inputEpsStream);
```

### الخطوة 3: استخراج الحجم الحالي لصورة EPS
```java
Dimension oldSize = doc.extractEpsSize();
```

### الخطوة 4: إنشاء تدفق إخراج للملف المعدل
```java
FileOutputStream outputEpsStream = new FileOutputStream(dataDir + "output_resize_points.eps");
```

### الخطوة 5: تغيير الحجم وحفظ EPS باستخدام النقاط
```java
doc.resizeEps(outputEpsStream, new Dimension2D.Double(oldSize.width * 2, oldSize.height * 2), Units.Points);
```

## كيفية تغيير حجم EPS باستخدام البوصات
نفس نمط الخطوات الخمسة ينطبق؛ فقط استبدل `Units.Points` بـ `Units.Inches` واضبط عامل التحجيم إلى القيمة بالبوصة المطلوبة.

## كيفية تغيير حجم EPS باستخدام المليمترات
مرة أخرى، اتبع نفس الخطوات مع استبدال الوحدة إلى `Units.Millimeters`. هذا مفيد عندما تحتاج إلى أبعاد مترية لعمليات الطباعة.

## كيفية تغيير حجم EPS باستخدام النسب المئوية
للتكبير أو التصغير بناءً على النسبة، احتفظ بالوحدة كـ `Units.Percent`. اضرب العرض والارتفاع الأصليين في النسبة المطلوبة (مثال: `0.5` لتقليل الحجم بنسبة 50 %).

## الأخطاء الشائعة والنصائح
- **دائمًا أغلق التدفقات** – في الكود الإنتاجي، احwrap التدفقات باستخدام try‑with‑resources لتجنب حجز الملفات.  
- **حافظ على نسبة الأبعاد** – اضرب العرض والارتفاع بنفس العامل ما لم تكن ترغب في تشويه صريح.  
- **تحقق من DPI** – تغيير الحجم لا يغير DPI؛ إذا كنت بحاجة إلى DPI مختلف، عدلها بعد تغيير الحجم.  
- **سلامة الخيوط** – أنشئ كائن `PsDocument` جديد لكل خيط؛ مشاركة نفس الكائن قد تؤدي إلى نتائج غير متوقعة.
