# كيفية إنشاء تدرج دائري – إضافة تدرج في PostScript

## المقدمة

إذا كنت تبحث عن **إنشاء تدرج دائري** في مستندات Java PostScript الخاصة بك، فإن Aspose.Page for Java هو الأداة المثالية. تسلسلات الدروس هذه ترشدك إلى إضافة تدرجات قطرية، أفقية، دائرية، ورأسية، مما يمنحك الثقة لإنتاج مستندات بصرية جذابة تجذب الانتباه. سنبدأ بنظرة سريعة، ثم نتعمق في كل نوع من التدرجات بخطوات واضحة وقابلة للتنفيذ.

## إجابات سريعة
- **ما هو التدرج الدائري؟** انتقال سلس للألوان ينطلق من نقطة مركزية إلى الخارج.  
- **لماذا تستخدم Aspose.Page للتدرجات؟** فهو يج abstracts أوامر PostScript منخفضة المستوى، مما يتيح لك التركيز على التصميم.  
- **هل أحتاج إلى ترخيص؟** النسخة التجريبية المجانية تكفي للتطوير؛ الترخيص التجاري مطلوب للإنتاج.  
- **ما نسخة Java المدعومة؟** Java 8+ (متوافقة مع الإصدارات الأحدث).  
- **كم يستغرق تنفيذ ذلك؟** عادةً أقل من 15 دقيقة لكل نوع من التدرجات.  

## ما هو التدرج الدائري في PostScript؟
التدرج الدائري يدمج الألوان من نقطة محورية إلى الخارج بطريقة دائرية. في PostScript، يتم ذلك عبر تعريف قاموس تظليل يصف ألوان البداية والنهاية، نصف القطر، والنقطة المركزية. Aspose.Page يبسط العملية من خلال توفير API سلس يولد كود PostScript اللازم خلف الكواليس.

## لماذا تستخدم Aspose.Page لإنشاء تدرج دائري؟
- **Speed:** كتابة بضع أسطر من Java بدلاً من كتابة PostScript معقد يدويًا.  
- **Precision:** التحكم في نقاط التوقف اللونية، نصف القطر، ومصفوفة التحويل باستخدام طرق type‑safe.  
- **Portability:** PostScript المُولد يعمل عبر الطابعات، المشاهدين، والمحولات.  
- **Scalability:** دمج عدة تدرجات (قطرية، أفقية، رأسية) في مستند واحد.

## كيفية إنشاء تدرج دائري في PostScript
فيما يلي دليل مختصر للخطوات التي ستتبعها في كل دليل مخصص:

1. **Instantiate the Document** – إنشاء كائن `Page` جديد باستخدام Aspose.Page.  
2. **Define the Gradient Parameters** – تحديد النقطة المركزية، نصف القطر، ونقاط التوقف اللونية.  
3. **Apply the Shading** – استخدام طريقة `addShading` لتضمين التدرج الدائري في الصفحة.  
4. **Save the Output** – تصدير المستند كملف `.ps` جاهز للطباعة أو المعالجة الإضافية.

> **Pro tip:** إعادة استخدام كائن `Shading` نفسه لأشكال متعددة يقلل من استهلاك الذاكرة ويحافظ على نظافة الكود.

## حالات الاستخدام الشائعة للتدرج الدائري
- **Background fills** للمنشورات، الكتيبات، أو الشهادات حيث يضيف تحول لوني خفيف عمقًا.  
- **Highlighting focal points** مثل الشعارات أو أزرار الدعوة إلى اتخاذ إجراء في PDF سيتم تحويله لاحقًا إلى PostScript.  
- **Creating realistic lighting effects** في المخططات التقنية أو الرسومات الهندسية.

## إضافة تدرج عمودي (الكلمة المفتاحية الثانوية)
إذا كنت بحاجة أيضًا إلى **إضافة تدرج عمودي** إلى نفس المستند، يوفر API مساعد `VerticalGradient` يعمل بالتوازي مع تظليل الدائري. سير العمل يطابق خطوات الدائري—فقط استبدل نوع التدرج. هذا يجعل من السهل وضع تدرج عمودي تحت تدرج دائري للحصول على تأثير بصري أغنى.
