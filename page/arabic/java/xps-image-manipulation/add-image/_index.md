## asp asp: إضافة صور إلى مستندات XPS
تدور منهجية **asp asp** حول نمط بسيط وقابل للتكرار: إنشاء مستند، تعريف مسار، تطبيق فرشاة صورة، ثم الحفظ. هذا النمط يحافظ على نظافة الكود ويسهل إدراج رسومات إضافية لاحقًا.

## ما هو Aspose.Page for Java؟
Aspose.Page هو واجهة برمجة تطبيقات مخصصة تتيح لك إنشاء وتعديل وعرض مستندات XPS (XML Paper Specification) دون الحاجة إلى Microsoft XPS Viewer. يقوم بتجريد التفاصيل منخفضة المستوى لتعليمات XPS، بحيث يمكنك التركيز على تخطيط المستندات بصريًا.

## لماذا إضافة صور إلى XPS؟
- **مظهر احترافي:** الصور مثل الشعارات، المخططات، أو صور المنتجات تعطي مستنداتك مظهرًا مصقولًا.  
- **اتساق العلامة التجارية:** تضمين شعار شركتك يضمن أن كل فاتورة أو تقرير يتم إنشاؤه يحمل علامتك التجارية.  
- **محتوى ديناميكي:** يمكنك إدراج رموز QR، الباركود، أو الرسومات التي يولدها المستخدم برمجيًا أثناء التشغيل.  

## المقدمة
إضافة الصور إلى مستندات XPS هي متطلب شائع في العديد من تطبيقات Java، تتراوح من توليد التقارير إلى معالجة المستندات. Aspose.Page for Java يبسط هذه المهمة، مقدماً طرقًا فعّالة لدمج الصور بسلاسة في ملفات XPS الخاصة بك. في هذا الدرس، سنوضح كيفية إضافة صورة إلى مستند XPS باستخدام Aspose.Page for Java.

## استيراد الحزم
في مشروع Java الخاص بك، استورد الحزم اللازمة من Aspose.Page for Java للوصول إلى الفئات والطرق المطلوبة.

```java
import com.aspose.xps.XpsDocument;
import com.aspose.xps.XpsPath;
import java.awt.geom.Rectangle2D;
```

## الخطوة 1: إعداد دليل المستند
حدد المسار إلى دليل المستندات حيث سيتم تخزين مستند XPS وملفات الصور.

```java
String dataDir = "Your Document Directory";
```

## الخطوة 2: إنشاء مستند XPS جديد
ابدأ مستند XPS جديد باستخدام مقتطف الشيفرة التالي:

```java
XpsDocument doc = new XpsDocument();
```

## الخطوة 3: إضافة صورة إلى مستند XPS
لإضافة صورة، أنشئ مسارًا في مستند XPS، واضبط فرشاة الصورة باستخدام ملف الصورة المحدد والإحداثيات.

```java
XpsPath path = doc.addPath(doc.createPathGeometry("M 30,20 l 258.24,0 0,56.64 -258.24,0 Z"));
path.setRenderTransform(doc.createMatrix(0.7f, 0f, 0f, 0.7f, 0f, 20f));
path.setFill(doc.createImageBrush(dataDir + "QL_logo_color.tif", new Rectangle2D.Double(0f, 0f, 258.24f, 56.64f), new Rectangle2D.Double(50f, 20f, 193.68f, 42.48f)));
```

## الخطوة 4: حفظ مستند XPS الناتج
احفظ مستند XPS المعدل إلى الدليل المحدد لديك.

```java
doc.save(dataDir + "AddImage_out.xps");
```

كرر هذه الخطوات لإضافة المزيد من الصور أو لتخصيص الصور الحالية وفقًا لمتطلبات مشروعك.

## المشكلات الشائعة والحلول
- **الصورة تظهر مشوهة:** تحقق من أن المستطيل المصدر (`new Rectangle2D.Double(0f, 0f, 258.24f, 56.64f)`) يطابق الأبعاد الفعلية لصورتك. عدّل المستطيل الهدف للحفاظ على نسبة العرض إلى الارتفاع.  
- **ظهور علامة مائية:** إذا شغلت الكود بدون ترخيص صالح، يضيف Aspose علامة مائية. فعّل الترخيص مبكرًا في التطبيق لتجنب ذلك.  
- **FileNotFoundException:** تأكد من أن `dataDir` يشير إلى المجلد الصحيح وأن اسم ملف الصورة (`QL_logo_color.tif`) يطابق الملف على القرص.
