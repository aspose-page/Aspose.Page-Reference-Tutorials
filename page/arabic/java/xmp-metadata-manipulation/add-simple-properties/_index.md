# إضافة بيانات XMP الوصفية إلى ملفات EPS باستخدام Java

## كيفية إضافة بيانات XMP الوصفية إلى ملفات EPS باستخدام Java؟
حمّل ملف EPS باستخدام `new PsDocument("input.eps")`، استرجع أو أنشئ حزمة XMP الخاصة به، أدخل الخصائص المطلوبة، ثم احفظ المستند مرة أخرى على القرص – كل ذلك بأقل من عشر أسطر من كود Java. يتيح لك هذا النهج تضمين بيانات وصفية قابلة للبحث ومتوافقة مع المعايير دون تعديل يدوي لمصدر EPS.

## ما هي بيانات XMP الوصفية في EPS؟
بيانات XMP الوصفية في EPS هي حزمة XML توجد داخل ملف EPS وتخزن معلومات مثل المؤلف، تاريخ الإنشاء، والعلامات المخصصة. يتيح تضمين هذه الحزمة للأدوات اللاحقة قراءة البيانات والتعامل معها دون الحاجة إلى ملفات جانبية منفصلة.

## لماذا نضيف بيانات XMP الوصفية إلى ملفات EPS؟
يؤدي تضمين بيانات XMP الوصفية إلى جعل أصول EPS قابلة للبحث فورًا، ويضمن الالتزام بتخزين معلومات الترخيص داخل الملف، ويمكّن خطوط الأنابيب الآلية من اتخاذ قرارات بناءً على العلامات المضمنة، ويضمن أن البيانات الوصفية تسافر مع الرسمة عبر جميع المنصات. يمكن لـ Aspose.Page معالجة ملفات تصل إلى 500 ميغابايت دون تحميل المستند بالكامل في الذاكرة، مما يوفر أداءً سريعًا لأعباء العمل الكبيرة.

## استيراد الحزم
أولاً، استورد الفئات التي ستحتاجها. تبقى عبارات الاستيراد كما هي تمامًا كما في المثال الأصلي:

```java
import java.io.FileInputStream;
import java.io.FileOutputStream;
import java.util.Date;
import java.util.TimeZone;
import com.aspose.eps.PsDocument;
import com.aspose.eps.xmp.XmpMetadata;
import com.aspose.eps.xmp.XmpValue;
import com.aspose.page.BaseExamplesTest;
```

## الخطوة 1: تحميل مستند EPS والحصول على بيانات XMP الوصفية
تمثل الفئة `PsDocument` ملف EPS وتوفر طرقًا للوصول إلى محتواه وبياناته الوصفية.  
كائن `XmpMetadata` يحمل حزمة XMP المرتبطة بالمستند.

```java
// The path to the documents directory.
String dataDir = "Your Document Directory";
// Initialize input EPS file stream
FileInputStream psStream = new FileInputStream(dataDir + "xmp3.eps");
PsDocument document = new PsDocument(psStream);
// Get XMP metadata. If EPS file doesn't contain XMP metadata, we get a new one filled with values from PS metadata comments (%%Creator, %%CreateDate, %%Title, etc.)
XmpMetadata xmp = document.getXmpMetadata();
```

## الخطوة 2: إضافة خاصية تاريخ
تمثل الفئة `Date` لحظة معينة في الزمن، بدقة مللي ثانية.

```java
// Add date property "xmp:Date1" value
TimeZone.setDefault(TimeZone.getTimeZone("UTC"));
Date now = new Date();
xmp.put("xmp:Date1", new XmpValue(now));
```

## الخطوة 3: إضافة خاصية عدد صحيح
الفئة `Integer` تغلف قيمة أولية `int` ككائن.

```java
// Add integer property "xmp:Intg1" value
xmp.put("xmp:Intg1", new XmpValue(111));
```

## الخطوة 4: إضافة خاصية عدد عشري
الفئة `Double` تغلف قيمة أولية `double` ككائن.

```java
// Add double property "xmp:Double1" value
xmp.put("xmp:Double1", new XmpValue(111.11D));
```

## الخطوة 5: إضافة خاصية سلسلة نصية
الفئة `String` تمثل تسلسلًا من الأحرف.

```java
// Add string property "xmp:String1" value
xmp.put("xmp:String1", new XmpValue("ABC"));
```

## الخطوة 6: حفظ ملف EPS المحدث
طريقة `save` تكتب المستند المعدل مرة أخرى إلى القرص، مع الحفاظ على بنية EPS.

```java
// Initialize output EPS file stream
FileOutputStream outPsStream = new FileOutputStream(dataDir + "xmp3_changed.eps");
// Save document with changed XMP metadata
try {
    document.save(outPsStream);
} finally {
    outPsStream.close();
}
```

## الخطوة 7: تنظيف الموارد
دائمًا أغلق التدفقات لتجنب تسرب مقبض الملف وتأكد من تفريغ جميع المخازن المؤقتة.

```java
// Close input EPS stream
psStream.close();
```
