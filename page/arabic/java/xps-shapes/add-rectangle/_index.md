# إنشاء مستند XPS Java – إضافة مستطيل

## المقدمة
في هذا الدرس الشامل ستقوم **create XPS document Java** وتتعلم كيفية إضافة المستطيلات باستخدام مكتبة Aspose.Page. سواء كنت تبني تقارير أو فواتير أو رسومات مخصصة، فإن إتقان إنشاء المستطيلات يمنحك تحكمًا دقيقًا في تخطيط XPS. سنستعرض كل خطوة، نشرح السبب وراء كل سطر من الشيفرة، ونظهر لك كيفية تخصيص الألوان والحدود للحصول على نتائج احترافية.

## إجابات سريعة
- **ما المكتبة الموصى بها؟** Aspose.Page for Java  
- **كم من الوقت تستغرق العملية؟** حوالي 10 دقائق لإنشاء مستطيل أساسي  
- **هل أحتاج إلى ترخيص؟** نسخة تجريبية مجانية تكفي للتطوير؛ يلزم ترخيص تجاري للإنتاج  
- **ما نسخة Java المدعومة؟** Java 8 وما بعدها  
- **هل يمكنني إضافة أشكال متعددة؟** نعم – كرّر الخطوات لكل شكل

## المتطلبات المسبقة
قبل أن نبدأ، تأكد من أن لديك:

- معرفة أساسية بلغة برمجة Java.  
- مكتبة Aspose.Page مثبتة. إذا لم تكن كذلك، يمكنك تحميلها من [وثائق Aspose.Page Java](https://reference.aspose.com/page/java/).  
- بيئة تطوير Java تعمل (IDE، JDK، وMaven/Gradle إذا كنت تفضّل).

## استيراد الحزم
للبدء، استورد الحزم الضرورية في مشروع Java الخاص بك. تأكد من إضافة مكتبة Aspose.Page إلى مسار الفئات (classpath). إليك مثالًا أساسيًا:

```java
import com.aspose.xps.XpsDocument;
import com.aspose.xps.XpsPath;
```

الآن، لنقّسم المثال المقدم إلى خطوات متعددة لإضافة مستطيل في XPS باستخدام Java.

## الخطوة 1: تعيين دليل المستند
```java
// The path to the documents directory.
String dataDir = "Your Document Directory";
```

استبدل `"Your Document Directory"` بالمسار إلى المجلد الذي تريد تخزين ملفات XPS فيه.

## الخطوة 2: إنشاء مستند XPS جديد
```java
// Create new XPS Document
XpsDocument doc = new XpsDocument();
```

هذا السطر **creates** مستند XPS جديد يمكنك لاحقًا ملؤه بالأشكال أو النصوص أو الصور.

## الخطوة 3: إضافة مستطيل مخطط بلون صلب CMYK
```java
// CMYK (blue) solid color stroked rectangle in the lower left
XpsPath path = doc.addPath(doc.createPathGeometry("M 20,10 L 220,10 220,100 20,100 Z"));
path.setStroke(doc.createSolidColorBrush(
    doc.createColor(dataDir + "uswebuncoated.icc", 1.0f, 1.000f, 0.000f, 0.000f, 0.000f)));
path.setStrokeThickness(12f);
```

تضيف هذه الخطوة مستطيلًا مخططًا بلون صلب CMYK. يمكنك تعديل سلسلة الهندسة (`"M 20,10 L 220,10 220,100 20,100 Z"`) لتغيير الحجم أو الموقع، وضبط قيم اللون في `createColor` لتناسب تصميمك.

## الخطوة 4: حفظ مستند XPS الناتج
```java
// Save resultant XPS document
doc.save(dataDir + "AddRectangle_out.xps");
```

أخيرًا، احفظ مستند XPS المعدل مع المستطيل المضاف إلى الدليل الذي حددته مسبقًا.

## حالات الاستخدام الشائعة
- **رؤوس التقارير** – استخدم المستطيلات ككتل خلفية للعناوين.  
- **جداول الفواتير** – إبراز الخلايا أو الأقسام بحدود ملونة.  
- **رسومات مخصصة** – دمج مستطيلات متعددة لإنشاء أشكال معقدة.

## نصائح استكشاف الأخطاء وإصلاحها
- **خطأ الملف غير موجود:** تحقق من أن `dataDir` يشير إلى مجلد موجود وأن ملف تعريف ICC (`uswebuncoated.icc`) موجود.  
- **ألوان غير صحيحة:** تأكد من أن ملف تعريف ICC يتطابق مع مساحة اللون التي تنوي استخدامها (CMYK مقابل RGB).  
- **أبعاد غير متوقعة:** عدل سلسلة الهندسة لتعكس الإحداثيات الصحيحة.

## الخاتمة
تهانينا! لقد تعلمت بنجاح كيفية **create XPS document Java** وإضافة المستطيلات باستخدام Aspose.Page. يتيح لك هذا الأساس بناء مستندات XPS أغنى، تخصيص الرسومات، وإدماج الأشكال في سير عمل أكبر.

## الأسئلة المتكررة
### هل يمكنني إضافة مستطيلات متعددة في مستند XPS واحد؟
نعم، يمكنك إضافة عدد غير محدود من المستطيلات بتكرار الخطوات الموضحة في الدرس.

### كيف يمكنني تغيير لون المستطيل؟
قم بتعديل قيم اللون في طريقة `createColor` للحصول على اللون المطلوب.

### هل Aspose.Page مناسبة لمعالجة تعديلات مستندات XPS المعقدة؟
بالطبع! توفر Aspose.Page مجموعة قوية من الميزات للتعامل مع مختلف مهام مستندات XPS.

### أين يمكنني العثور على أمثلة إضافية ودعم؟
استكشف [منتدى Aspose.Page](https://forum.aspose.com/c/page/39) للمزيد من الأمثلة واحصل على المساعدة من المجتمع.

### هل يمكنني تجربة Aspose.Page قبل الشراء؟
نعم، يمكنك تجربة [نسخة تجريبية مجانية](https://releases.aspose.com/) لتجربة قدرات Aspose.Page.

## الأسئلة المتكررة

**س: هل يعمل هذا النهج مع Java 11 أو أحدث؟**  
**ج:** نعم، مكتبة Aspose.Page متوافقة مع Java 8 وما بعدها، بما في ذلك Java 11 و17 وما بعدهما.

**س: هل يمكنني تضمين صور داخل مساحة المستطيل؟**  
**ج:** يمكنك إضافة عنصر `XpsImage` وتحديد موقعه فوق أو داخل المستطيل باستخدام نفس إحداثيات الهندسة.

**س: كيف يمكنني تعيين لون تعبئة بدلاً من الحد فقط؟**  
**ج:** استدعِ `path.setFill(...)` مع فرشاة لون صلب تُنشئها عبر `doc.createSolidColorBrush(...)`.

**س: هل يمكن تدوير المستطيل؟**  
**ج:** طبّق مصفوفة تحويل على المسار باستخدام `path.setTransform(...)` لتحقيق التدوير أو التحجيم.

**س: ما الترخيص المطلوب للاستخدام في الإنتاج؟**  
**ج:** يلزم الحصول على ترخيص تجاري من Aspose.Page للنشر؛ النسخة التجريبية مخصصة للتقييم والتطوير فقط.
