# إنشاء مستند XPS باستخدام Aspose.Page لـ .NET

## مقدمة

مرحبًا بكم في دليلنا خطوة بخطوة حول **كيفية إنشاء مستند XPS** باستخدام Aspose.Page لـ .NET. في هذا البرنامج التعليمي، سنستكشف عملية إنشاء ملفات XPS، وهو تنسيق شائع الاستخدام للمستندات الإلكترونية. سواء كنت مطورًا متمرسًا أو تبدأ للتو مع Aspose.Page، فإن هذا الدليل مُصمم لمساعدتك على إنشاء مستندات XPS بسلاسة مع أمثلة واضحة وتفسيرات مفصلة.

## الإجابات السريعة
- **ما المكتبة التي أحتاجها؟** Aspose.Page for .NET  
- **هل يمكنني تشغيل هذا على .NET Core؟** نعم، المكتبة تدعم بالكامل .NET Core و .NET 5/6  
- **كم عدد أسطر الشيفرة؟** أقل من 20 سطرًا لإنشاء ملف XPS أساسي  
- **هل أحتاج إلى ترخيص للاختبار؟** يتوفر نسخة تجريبية مجانية؛ الترخيص مطلوب للإنتاج  
- **ما هو تنسيق المخرجات؟** XPS (XML Paper Specification)  

## كيفية إنشاء مستند XPS باستخدام Aspose.Page لـ .NET

فيما يلي كل ما تحتاجه قبل بدء كتابة الشيفرة، يليه دليل مختصر مرقم.

## المتطلبات المسبقة

قبل أن نغوص في البرنامج التعليمي، تأكد من توفر المتطلبات المسبقة التالية:

1. مكتبة Aspose.Page لـ .NET: قم بتنزيل وتثبيت مكتبة Aspose.Page من [رابط التحميل](https://releases.aspose.com/page/net/).

2. دليل المستندات الخاص بك: اختر أو أنشئ دليلًا على نظامك حيث تريد حفظ ملفات XPS الناتجة.

الآن، لننقُل إلى البرنامج التعليمي!

## استيراد مساحات الأسماء

من أجل استخدام Aspose.Page لـ .NET، تحتاج إلى استيراد مساحات الأسماء الضرورية إلى مشروعك. اتبع الخطوات التالية:

### الخطوة 1: إضافة مرجع إلى Aspose.Page

في مشروعك، أضف مرجعًا إلى مكتبة Aspose.Page لـ .NET. يمكنك العثور على ملف DLL المطلوب في الحزمة التي تم تنزيلها.

### الخطوة 2: استيراد مساحات الأسماء

قم بتضمين مساحات الأسماء التالية في ملف الشيفرة الخاص بك:

```csharp
using Aspose.Page.XPS;
using Aspose.Page.XPS.XpsModel;
using System.Drawing;
```

الآن بعد أن قمنا بإعداد المتطلبات المسبقة واستيراد مساحات الأسماء اللازمة، لنقسم عملية إنشاء مستند XPS إلى عدة خطوات.

## الخطوة 1: تعيين دليل المستند

```csharp
string dir = "Your Document Directory";
```

تأكد من استبدال `"Your Document Directory"` بالمسار الفعلي حيث تريد حفظ ملف XPS الناتج.

## الخطوة 2: إنشاء مستند XPS

```csharp
XpsDocument xDocs = new XpsDocument();
```

قم بتهيئة مستند XPS جديد باستخدام الفئة `XpsDocument`.

## الخطوة 3: إضافة رموز (Glyphs) إلى المستند

```csharp
var glyphs = xDocs.AddGlyphs("Arial", 12, FontStyle.Regular, 300f, 450f, "Hello World!");
```

استخدم طريقة `AddGlyphs` لإضافة رموز (نص) إلى المستند. خصص الخط، الحجم، النمط، والموقع حسب الحاجة.

## الخطوة 4: تعيين لون تعبئة الرموز

```csharp
glyphs.Fill = xDocs.CreateSolidColorBrush(Color.Black);
```

حدد لون التعبئة للرموز المضافة. في هذا المثال، نستخدم اللون الأسود، لكن يمكنك اختيار أي لون آخر.

## الخطوة 5: حفظ النتيجة

```csharp
xDocs.Save(dir + "output.xps");
```

أخيرًا، احفظ مستند XPS إلى الدليل المحدد مع اسم الملف المطلوب. سيحتوي ملف XPS الناتج على النص "Hello World!".

تهانينا! لقد نجحت في إنشاء مستند XPS باستخدام Aspose.Page لـ .NET.

## نصائح شائعة ومشكلات محتملة

- **مسار الدليل** – استخدم `Path.Combine(dir, "output.xps")` لتجنب فقدان فواصل المسار على أنظمة تشغيل مختلفة.  
- **توفر الخط** – يجب أن يكون الخط الذي تحدده مثبتًا على الجهاز الذي يُشغَّل عليه الكود؛ وإلا سيستبدل Aspose الخط بخط افتراضي.  
- **صفحات متعددة** – لإنشاء ملفات XPS متعددة الصفحات، أنشئ كائنات `XpsPage` إضافية وأضف المحتوى إلى كل صفحة قبل الحفظ.
