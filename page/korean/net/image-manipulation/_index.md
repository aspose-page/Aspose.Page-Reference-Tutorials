## 자주 묻는 질문

**Q: Aspose.Page를 사용해 EPS를 상업용 애플리케이션에서 사용할 수 있나요?**  
A: 네, 유효한 Aspose 라이선스를 보유하면 어떤 상업 프로젝트에도 솔루션을 배포할 수 있습니다.

**Q: 크기 조정 시 벡터 품질이 유지되나요?**  
A: 물론입니다. EPS는 벡터 형식이므로 래스터화 없이 크기 조정이 이루어져 출력이 어떤 크기에서도 선명합니다.

**Q: 시작하기 전에 필요한 사전 조건은 무엇인가요?**  
A: .NET Framework 4.5+ (또는 .NET Core 3.1+), Aspose.Page for .NET NuGet 패키지, 그리고 프로덕션 사용을 위한 유효한 라이선스 키가 필요합니다.

**Q: 여러 페이지가 포함된 EPS 파일은 어떻게 처리하나요?**  
A: API를 사용하면 인덱스로 특정 페이지를 로드할 수 있으며, 해당 페이지를 독립적으로 자르거나 크기 조정할 수 있습니다.

**Q: 많은 EPS 파일을 일괄 처리할 방법이 있나요?**  
A: 예, 디렉터리를 순회하면서 각 EPS를 Aspose.Page로 로드하고 동일한 자르기/크기 조정 로직을 적용한 뒤 결과를 저장하면 됩니다.
