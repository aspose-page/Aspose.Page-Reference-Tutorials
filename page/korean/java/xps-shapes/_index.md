# 도형 - XPS

## 소개

Java로 눈에 띄는 XPS 문서를 만들고 싶다면 **방사형 그라디언트 타원**을 마스터하는 것이 좋은 시작점입니다. Aspose.Page for Java를 사용하면 타원, 사각형 및 기타 기하학적 도형을 손쉽게 그릴 수 있어 문서에 전문적인 마무리를 제공할 수 있습니다. 이 튜토리얼에서는 필수 단계를 안내하고 실제 사용 사례를 보여주며 각 도형에 대한 자세한 하위 튜토리얼을 안내합니다.

## 빠른 답변
- **방사형 그라디언트 타원은 무엇인가요?** 테두리가 방사형 색상 그라디언트로 채워진 벡터 기반 타원입니다.
- **왜 Aspose.Page for Java를 사용하나요?** 저수준 XPS 마크업을 다루지 않고도 도형을 그릴 수 있는 간단한 API를 제공합니다.
- **도형을 추가하는 데 얼마나 걸리나요?** 기본 타원이나 사각형의 경우 일반적으로 10분 이내입니다.
- **라이선스가 필요합니까?** 개발 단계에서는 무료 체험판으로 충분하지만, 운영 환경에서는 상용 라이선스가 필요합니다.
- **지원되는 Java 버전은 무엇인가요?** Java 8 이상.

## 방사형 그라디언트 타원이란?
**방사형 그라디언트 타원**은 타원의 기하학적 정밀도와 중앙점에서 바깥쪽으로 퍼지는 그라디언트를 결합한 형태입니다. 이 효과는 섹션 강조, 버튼과 같은 시각 요소 생성, XPS 페이지에 은은한 브랜드 요소를 추가하는 데 적합합니다.

## Java XPS에서 도형을 그리는 이유
타원 및 사각형과 같은 도형을 그리면 다음과 같은 이점을 얻을 수 있습니다:
- 시각적으로 콘텐츠를 구조화 (예: 강조 박스, 다이어그램).
- 색상으로 채워진 그래픽을 통해 중요한 정보를 강조.
- 래스터 이미지를 확장 가능한 벡터 아트로 교체하여 파일 크기를 작게 유지.
- 레이아웃을 프로그래밍 방식으로 정의하여 보고서 생성을 자동화.

## 일반적인 사용 사례
- **보고서 및 청구서:** 그라디언트가 채워진 타원으로 합계를 강조.
- **기술 매뉴얼:** 사각형을 사용해 코드 스니펫이나 경고를 프레임.
- **마케팅 브로셔:** XPS 내에서 직접 눈에 띄는 배지나 스탬프 생성.

## 자주 묻는 질문

**Q: 도형을 만든 후에도 그라디언트 색상을 변경할 수 있나요?**  
A: 예, `GradientStop` 컬렉션을 수정하고 타원의 스트로크에 다시 적용하면 됩니다.

**Q: 사각형에 그림자 효과를 추가할 수 있나요?**  
A: XPS에는 기본 그림자 기능이 없지만, 원본 뒤에 약간 오프셋된 반투명 사각형을 그려서 시뮬레이션할 수 있습니다.

**Q: 이러한 메서드가 PDF 변환에도 작동하나요?**  
A: 물론입니다. Aspose.Page를 사용하면 모든 벡터 도형을 유지하면서 XPS 문서를 PDF로 변환할 수 있습니다.

**Q: 도형이 DPI에 독립적임을 어떻게 보장하나요?**  
A: XPS는 벡터 그래픽을 사용하므로 도형이 자동으로 스케일됩니다. 도형 내부에 래스터 이미지를 넣지 않으면 됩니다.

**Q: 더 복잡한 다각형을 그려야 하면 어떻게 하나요?**  
A: `Path` 클래스를 사용해 사용자 정의 기하학을 정의하면 됩니다. 동일한 그라디언트와 스트로크 설정을 적용할 수 있습니다.
