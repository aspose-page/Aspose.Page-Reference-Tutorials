## Frequently Asked Questions

**Q: 상업 프로젝트에서 Aspose.Page 변환을 사용할 수 있나요?**  
A: 예. 운영 환경에서는 유효한 Aspose 라이선스가 필요하지만, 평가용 무료 체험판을 사용할 수 있습니다.

**Q: 지원되는 Java 버전은?**  
A: Aspose.Page for Java는 Java 8 이상에서 작동합니다.

**Q: 내보낸 이미지의 해상도를 어떻게 제어하나요?**  
A: `save` 메서드를 호출하기 전에 `ImageSaveOptions` 객체의 DPI 값을 설정합니다.

**Q: 여러 PostScript 파일을 한 번에 배치 변환할 수 있나요?**  
A: 물론입니다—파일 컬렉션을 순회하면서 동일한 변환 로직을 각 문서에 적용하면 됩니다.

**Q: PostScript 파일에 비밀번호가 걸려 있으면 어떻게 하나요?**  
A: 비밀번호를 지정할 수 있는 `LoadOptions` 객체를 받는 `Document` 생성자를 사용하십시오.
