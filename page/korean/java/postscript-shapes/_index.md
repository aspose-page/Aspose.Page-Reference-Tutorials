- **잘못된 치수:** PostScript에서 사용되는 좌표계(포인트)를 확인하세요; 원점은 왼쪽 하단에 있습니다.
- **색상 누락:** 채우기 색상과 선 색상을 모두 설정했는지 확인하세요; 그렇지 않으면 사각형이 보이지 않을 수 있습니다.
