# Aspose.Page .NET – 繪製形狀

## 介紹

Aspose.Page .NET 讓開發人員在 .NET 應用程式中輕鬆繪製形狀，無論是處理 PostScript (PS) 或 XPS 檔案。在本指南中，您將了解如何加入圓形、橢圓形、矩形等，同時學習 **how to add shapes .NET** 風格。讓我們一起探索可能性，看看為何使用 Aspose.Page .NET 繪製形狀既強大又直觀。

## 快速解答
- **What does Aspose.Page .NET do?** 它能以程式方式建立與操作 PS 與 XPS 文件，包含繪製幾何形狀。  
- **Which shapes can I draw?** 圓形、橢圓形、矩形，以及自訂路徑。  
- **Do I need a license?** 提供免費試用版；商業使用需購買授權。  
- **What .NET versions are supported?** 支援 .NET Framework 4.5+、.NET Core 3.1+、.NET 5/6/7。  
- **Is there sample code?** 有 – 每個連結的教學都提供可直接執行的範例。

## Aspose.Page .NET 是什麼？

Aspose.Page .NET 是一套 .NET 函式庫，讓您在不需要 Adobe 工具的情況下產生與編輯 PostScript 與 XPS 文件。它提供功能豐富的 API，可用於繪製形狀、套用顏色、漸層，以及管理頁面版面配置——全部以乾淨的受管理程式碼完成。

## 為何使用 Aspose.Page 在 .NET 中繪製形狀？

- **Cross‑format support:** 一次編寫，可輸出為 PS 或 XPS。  
- **High fidelity:** 向量圖形在任何縮放下皆保持高品質。  
- **No external dependencies:** 純 .NET，無需原生函式庫。  
- **Developer‑friendly API:** 流暢的方法與清晰的命名，使得在 **draw shapes .NET** 應用程式中輕鬆繪製形狀。

## 常見問與答

**Q: Can I use Aspose.Page .NET in a commercial application?**  
A: 是的，持有效的 Aspose 授權即可商業使用；亦提供免費試用版供評估。

**Q: Do I need to install any native components?**  
A: 不需要，Aspose.Page .NET 為純受管理函式庫，只需引用 NuGet 套件即可。

**Q: Is it possible to combine shapes with text in the same page?**  
A: 當然可以。API 允許先繪製形狀，再加入文字物件，並依需求控制 Z‑order。

**Q: How do I handle large documents with many shapes?**  
A: 使用 `Document.Save` 的多載版本搭配串流緩衝，並考慮分割頁面以降低記憶體使用量。

**Q: Does Aspose.Page support transparency and gradients?**  
A: 是的，PS 與 XPS API 都支援漸層筆刷與 alpha 合成，提供豐富的視覺效果。
