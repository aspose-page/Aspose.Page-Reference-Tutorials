# 漸層填充

## 介紹

您是否已準備好將文件創作提升到新層次？**Aspose.Page for .NET** 為您帶來一系列漸層填充教學，讓您能在 PostScript (PS) 與 XPS 文件中注入動態視覺元素。本指南將一步步說明如何 **新增垂直漸層** 填充，以及對角線與水平漸層，提升專案的整體吸引力。

## 快速解答
- **我可以用 Aspose.Page 建立什麼？** 具備進階圖形效果的 PS 與 XPS 文件。  
- **涵蓋哪些漸層類型？** 對角線、水平與 **垂直漸層** 填充。  
- **需要授權嗎？** 提供免費試用版；正式上線需購買商業授權。  
- **支援哪些 .NET 版本？** .NET Framework 4.5+、.NET Core 3.1+、.NET 5/6/7。  
- **實作需要多長時間？** 基本漸層通常在 10 分鐘以內完成。

## 什麼是垂直漸層？

**垂直漸層** 是指從形狀的上緣到下緣，以兩種或多種顏色平滑過渡，為文件元素增添深度與視覺趣味。

## 為什麼在 Aspose.Page 中使用漸層填充？

- **專業外觀：** 漸層讓靜態文件更具現代感。  
- **跨格式支援：** 同一段程式碼可同時產生 PostScript 與 XPS 輸出。  
- **細緻控制：** 可程式化設定顏色、位置與漸層類型。  
- **效能最佳化：** Aspose.Page 能高效渲染漸層，即使是大型文件亦不例外。

## 常見問題

**Q: 我可以在商業應用程式中使用這些漸層技術嗎？**  
A: 可以。只需在正式環境中套用有效的 Aspose.Page 授權即可。

**Q: PS 與 XPS 的漸層 API 是否相同？**  
A: API 呼叫完全相同；Aspose.Page 抽象底層格式，讓您可重複使用相同程式碼。

**Q: 若需要在漸層中使用超過兩種顏色該怎麼辦？**  
A: Aspose.Page 支援多段點漸層，您可以以程式方式定義額外的顏色停點。

**Q: 大量使用漸層會影響效能嗎？**  
A: 漸層渲染已優化，但若文件極大，建議將繪圖操作批次化處理。

**Q: 我可以在哪裡找到完整的 API 參考文件？**  
A: 官方的 Aspose.Page for .NET API 文件可於 Aspose 官網取得。
