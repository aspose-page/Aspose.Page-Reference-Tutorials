## 什麼是 Visual Brush？

Visual Brush 會捕捉視覺元素（例如形狀、文字或影像），並允許您將其作為其他繪圖物件的填充重複使用。此方法可減少重複並確保文件中的樣式一致。

## 為什麼使用 Grid Visual Brush？

A Grid Visual Brush 會產生可平鋪的格線圖案，可作為背景、指引或裝飾元素。當您需要以下情況時特別有用：

- **Consistent alignment** 跨多頁保持一致對齊。  
- **Lightweight styling** 無需嵌入大型影像的輕量樣式。  
- **Dynamic scaling** 可根據不同頁面尺寸動態縮放。

## 常見問題

**Q:** *我可以在同一文件中結合多個 visual brushes 嗎？*  
**A:** 是的，您可以疊加不同的 Visual Brushes 以產生複雜的圖案與效果。

**Q:** *是否可以在執行時變更格線大小？*  
**A:** 當然可以。在渲染前調整 `CellWidth` 與 `CellHeight` 屬性，即可動態修改格線。

**Q:** *Visual Brushes 會影響檔案大小嗎？*  
**A:** 由於它們是向量基礎，相較於點陣圖影像，僅增加極少的負擔。

**Q:** *我可以在 PDF 輸出時使用 Visual Brush 嗎？*  
**A:** 可以，Aspose.Page 支援 PDF、XPS 以及其他多種格式，同時保留 Visual Brush 定義。

**Q:** *如果畫筆未顯示，我該採取哪些故障排除步驟？*  
**A:** 請確認畫筆已指派給可繪製物件，且文件的渲染設定正確配置。
