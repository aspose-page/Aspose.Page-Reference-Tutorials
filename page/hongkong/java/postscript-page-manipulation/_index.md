## 常見問題

**Q: 我可以在不遺失原始內容的情況下，為既有的 PostScript 檔案新增頁面嗎？**  
A: 可以。Aspose.Page 會在保留所有既有內容、字型與圖形的前提下插入新頁面。

**Q: 能否將一個 PostScript 文件的頁面複製到另一個文件中？**  
A: 完全可以。此 API 允許您從任何來源文件匯入頁面，並放入目標檔案。

**Q: 新增頁面後，我可以將最終文件轉換成哪些格式？**  
A: 函式庫支援將結果儲存為 PostScript、PDF 或 XPS，提供下游處理的彈性。

**Q: 函式庫是否支援在新頁面中加入影像或向量圖形？**  
A: 支援。您可以使用相同的 API 在新建頁面上繪製形狀、插入影像與渲染文字。

**Q: 文件在新增頁面時有尺寸限制嗎？**  
A: 函式庫能有效處理大型檔案，但對於特別龐大的文件，建議使用 64 位元 JVM。
