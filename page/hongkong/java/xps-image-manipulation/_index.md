## Frequently Asked Questions

**Q: 我可以在既有的 XPS 檔案中加入影像嗎？**  
A: 可以。使用 `XpsDocument` 開啟現有的 XPS，並使用相同的 `addImage` API 插入新圖形。

**Q: 影像大小有何限制？**  
A: Aspose.Page 支援最高 10 MB 的影像；較大的檔案可能影響效能，但仍可處理。

**Q: 平鋪會影響檔案大小嗎？**  
A: 平鋪會重複相同的影像資料，檔案大小僅會略微增加——只儲存平鋪定義。

**Q: 有方法保留影像的透明度嗎？**  
A: 加入 PNG 影像時會保留 alpha 通道，透明區域在 XPS 中正確呈現。

**Q: 如何偵錯影像放置問題？**  
A: 使用 `saveAsPdf` 方法匯出預覽 PDF；可在最終 XPS 輸出前視覺化座標與縮放。
