## 常見問題

**Q: 我需要付費授權才能使用 Aspose.Page 繪製形狀嗎？**  
A: 提供免費試用版供評估，但在正式上線時需購買商業授權。

**Q: 我可以在同一 XPS 頁面上結合多個形狀嗎？**  
A: 可以，您可以依需求加入任意數量的形狀，只需將每個形狀加入頁面的 graphics 集合即可。

**Q: 是否可以對形狀套用變換（旋轉、縮放）？**  
A: 當然可以。API 提供變換矩陣，讓您旋轉、縮放或傾斜任何形狀。

**Q: 支援哪些 Java 版本？**  
A: Aspose.Page for Java 支援 Java 8 及更新的執行環境。

**Q: 在加入形狀後，如何匯出最終的 XPS 文件？**  
A: 呼叫 `Document` 物件的 `save` 方法，指定目標檔案路徑與格式。
