## 什麼是徑向漸層橢圓？
**radial gradient ellipse** 結合了橢圓的幾何精確度與從中心點向外輻射的漸層。此效果非常適合用於突顯區段、製作類似按鈕的視覺效果，或在 XPS 頁面加入細緻的品牌元素。

## 常見問題

**Q: 我可以在形狀建立後更改漸層顏色嗎？**  
A: 可以，您可以修改 `GradientStop` 集合，並重新套用到橢圓的描邊上。

**Q: 是否可以為矩形加入陰影效果？**  
A: 雖然 XPS 沒有原生的陰影功能，但您可以透過在原始矩形後方繪製稍微偏移且半透明的矩形來模擬。

**Q: 這些方法能用於 PDF 轉換嗎？**  
A: 當然可以。Aspose.Page 允許您將 XPS 文件轉換為 PDF，且保留所有向量形狀。

**Q: 如何確保形狀與 DPI 無關？**  
A: 由於 XPS 使用向量圖形，形狀會自動縮放；只要避免在形狀內使用點陣圖即可。

**Q: 如果需要繪製更複雜的多邊形該怎麼辦？**  
A: 使用 `Path` 類別定義自訂幾何形狀；相同的漸層與描邊設定皆適用。
