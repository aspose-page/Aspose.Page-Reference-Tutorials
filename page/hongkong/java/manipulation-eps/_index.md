## 常見問題與技巧

- **Missing fonts** – 確保 EPS 中引用的字型已安裝於主機，或使用 API 內嵌字型。  
- **Large file size after resize** – 使用 `saveOptions.setCompressionLevel()` 方法降低檔案大小，同時保持品質。  
- **Unexpected clipping** – 再次檢查裁剪矩形座標；座標以點 (pt) 為單位 (1 pt = 1/72 英吋)。

## 常見問答

**Q: Can I process multiple EPS files in a batch?**  
A: 可以。將裁剪或調整大小的程式碼包在迴圈中，並為每個檔案重複使用相同的 `Page` 物件。

**Q: Does Aspose.Page support EPS with embedded raster images?**  
A: 當然支援。向量與點陣圖元素在操作過程中皆會被完整保留。

**Q: Is it possible to convert EPS to other formats (PDF, PNG) after editing?**  
A: 可以。裁剪或調整大小後，您可呼叫 `page.save("output.pdf", SaveFormat.Pdf)` 或 `SaveFormat.Png` 進行轉檔。

**Q: What if the EPS file is corrupted?**  
A: API 會拋出 `FileFormatException`。請捕捉例外並在處理前驗證來源檔案。

**Q: Do I need to close any resources manually?**  
A: `Page` 類別實作了 `Closeable` 介面；請使用 try‑with‑resources 區塊或呼叫 `page.close()` 釋放記憶體。
