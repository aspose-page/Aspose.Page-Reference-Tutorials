- **Why use Aspose.Page for gradients?** 它抽象化低階 PostScript 指令，讓您專注於設計。

- **How long does implementation take?** 每種漸層類型通常在 15 分鐘以內完成。

## 什麼是 PostScript 中的徑向漸層？
徑向漸層以圓形方式從焦點向外混合顏色。在 PostScript 中，透過定義 shading dictionary（陰影字典）來描述起始與結束顏色、半徑以及中心點。Aspose.Page 透過提供流暢的 API，於背後自動產生所需的 PostScript 程式碼，簡化了此過程。

## 為何使用 Aspose.Page 來建立徑向漸層？
- **Speed:** 用少量 Java 程式碼取代手動編寫複雜的 PostScript。  
- **Precision:** 以類型安全的方法控制顏色節點、半徑與變換矩陣。  
- **Portability:** 產生的 PostScript 可在印表機、檢視器與轉換器間通用。  
- **Scalability:** 在同一文件中結合多種漸層（對角線、水平、垂直）。
