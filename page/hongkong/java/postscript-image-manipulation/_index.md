## 如何加入圖片 – 步驟概覽

以下是安裝程式庫後可遵循的簡要路線圖：

1. **建立 `Document` 物件**，代表您要編輯的 PostScript 檔案。  
2. **實例化 `Image` 物件**，來源可以是檔案、串流或位元組陣列。  
3. **定義放置矩形**（X、Y、寬度、高度），決定圖片顯示的位置。  
4. **呼叫 `document.addImage(image, rect)`** 以嵌入圖形。  
5. **將更新後的文件儲存**至磁碟或串流。

上述每個動作皆在「Add Image in Java PostScript」教學中示範，您可以直接複製貼上相同程式碼片段至專案。

## 常見問題與技巧

- **圖片格式支援** – 請確認來源圖片為 Aspose 支援的格式（PNG、JPEG、BMP 等）。  
- **座標系統** – PostScript 使用左下角為原點，務必檢查 Y 座標。  
- **記憶體使用** – 大尺寸圖片會增加記憶體消耗，建議在插入前先降採樣。  
- **授權** – 未授權執行會在輸出檔案加上浮水印，正式環境請務必套用有效授權。

## 常見問答

**Q: 可以在同一個 PostScript 頁面加入多張圖片嗎？**  
A: 可以。重複呼叫 `addImage` 方法，並提供不同的放置矩形。

**Q: Aspose.Page 也支援向量圖形嗎？**  
A: 當然支援。您可以在光柵圖片之外嵌入 SVG、EPS，甚至原始 PostScript 指令。

**Q: 支援哪些 Java 版本？**  
A: 程式庫相容於 Java 8 及以上版本，包括 Java 11、17 以及後續的 LTS 版本。

**Q: 有辦法在加入圖片時同時旋轉嗎？**  
A: 有。使用 `Matrix` 變換 API 設定旋轉角度後，再呼叫 `addImage`。

**Q: 如何處理透明 PNG？**  
A: 透明 PNG 會自動保留；只要目標 PostScript 檢視器支援 alpha 通道即可。
