## 如何使用 Aspose.Page 新增文字方法？
`Page` 代表 XPS 文件中的單一頁面。`TextFragment` 類別封裝可放置於頁面的文字片段。載入 XPS 文件、建立 `Page` 物件，並使用 `TextFragment` 類別插入所需字串。此 **aspose page add text** 工作流程僅需三個 API 呼叫，且在所有支援平台上表現一致，讓您快速新增文字，同時保留原有版面與格式。
