## 常見問題

**Q: 我可以在商業專案中使用 Aspose.Page 轉換嗎？**  
A: 可以。正式使用時需具備有效的 Aspose 授權，但可使用免費試用版進行評估。

**Q: 支援哪個 Java 版本？**  
A: Aspose.Page for Java 支援 Java 8 及以上版本。

**Q: 如何控制匯出影像的解析度？**  
A: 在呼叫 `save` 方法前，於 `ImageSaveOptions` 物件上設定 DPI 值。

**Q: 能否一次批次轉換多個 PostScript 檔案？**  
A: 當然可以——迴圈處理您的檔案集合，對每個文件套用相同的轉換邏輯。

**Q: 如果我的 PostScript 檔案受密碼保護怎麼辦？**  
A: 使用接受 `LoadOptions` 物件的 `Document` 建構子，於其中指定密碼。
