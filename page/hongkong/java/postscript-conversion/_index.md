## 常見問題與解決方案

| 問題 | 解決方案 |
|-------|----------|
| **低解析度影像** | 在呼叫 `save()` 前，提高 `ImageSaveOptions` 的 DPI。 |
| **大型檔案的記憶體消耗** | 一次處理單一頁面，儲存後釋放每個 `Image`。 |
| **受密碼保護的 PostScript** | 使用接受密碼字串的 `LoadOptions` 建構子。 |
| **顏色異常** | 檢查 `PdfSaveOptions` 或 `ImageSaveOptions` 中的色彩空間設定。 |
