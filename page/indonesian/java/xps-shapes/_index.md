## Apa itu elips gradien radial?
Sebuah **radial gradient ellipse** menggabungkan presisi geometris sebuah elips dengan gradien yang memancar dari titik pusat ke luar. Efek ini sempurna untuk menyorot bagian, membuat visual seperti tombol, atau menambahkan elemen branding halus ke halaman XPS Anda.

## Mengapa menggambar bentuk di Java XPS?
Menggambar bentuk seperti elips dan persegi panjang memungkinkan Anda:
- Menyusun konten secara visual (mis., kotak penjelasan, diagram).
- Menekankan informasi penting dengan grafik berisi warna.
- Mengganti gambar raster dengan seni vektor yang dapat diskalakan, menjaga ukuran file tetap kecil.
- Mengotomatiskan pembuatan laporan di mana tata letak didefinisikan secara programatik.

- **Gunakan satuan yang konsisten:** Aspose.Page bekerja dengan poin; 1 pt = 1/72 in.

## Pertanyaan yang Sering Diajukan

**Q: Bisakah saya mengubah warna gradien setelah bentuk dibuat?**  
A: Ya, Anda dapat memodifikasi koleksi `GradientStop` dan menerapkannya kembali ke garis elips.

**Q: Apakah memungkinkan menambahkan efek bayangan pada persegi panjang?**  
A: Meskipun XPS tidak memiliki bayangan bawaan, Anda dapat mensimulasikannya dengan menggambar persegi panjang yang sedikit bergeser dan semi‑transparan di belakang yang asli.

**Q: Apakah metode ini bekerja dengan konversi PDF?**  
A: Tentu saja. Aspose.Page memungkinkan Anda mengonversi dokumen XPS ke PDF sambil mempertahankan semua bentuk vektor.

**Q: Bagaimana saya memastikan bentuk tidak tergantung DPI?**  
A: Karena XPS menggunakan grafik vektor, bentuk secara otomatis dapat diskalakan; cukup hindari gambar raster di dalam bentuk.

**Q: Bagaimana jika saya perlu menggambar poligon yang lebih kompleks?**  
A: Gunakan kelas `Path` untuk mendefinisikan geometri khusus; pengaturan gradien dan garis yang sama tetap berlaku.
