# Cara Membuat Gradien Radial – Penambahan Gradien di PostScript

## Apa Itu Gradien Radial di PostScript?
Gradien radial menggabungkan warna dari titik fokus ke luar secara melingkar. Di PostScript, ini dicapai dengan mendefinisikan kamus shading yang menjelaskan warna awal dan akhir, radius, serta titik pusat. Aspose.Page menyederhanakan proses ini dengan menyediakan API fluida yang menghasilkan kode PostScript yang diperlukan di balik layar.

## Mengapa Menggunakan Aspose.Page untuk Membuat Gradien Radial?
- **Kecepatan:** Tulis beberapa baris Java alih‑alih menulis PostScript yang kompleks secara manual.  
- **Presisi:** Kendalikan color stop, radius, dan matriks transformasi dengan metode yang type‑safe.  
- **Portabilitas:** PostScript yang dihasilkan bekerja di semua printer, penampil, dan konverter.  
- **Skalabilitas:** Gabungkan beberapa gradien (diagonal, horizontal, vertikal) dalam satu dokumen.

## Cara Membuat Gradien Radial di PostScript
Berikut adalah panduan singkat langkah‑langkah yang akan Anda ikuti dalam setiap panduan khusus:

1. **Instansiasi Dokumen** – Buat objek `Page` baru menggunakan Aspose.Page.  
2. **Definisikan Parameter Gradien** – Atur titik pusat, radius, dan color stop.  
3. **Terapkan Shading** – Gunakan metode `addShading` untuk menyisipkan gradien radial ke halaman.  
4. **Simpan Output** – Ekspor dokumen sebagai file `.ps` yang siap dicetak atau diproses lebih lanjut.

> **Tip pro:** Menggunakan kembali objek `Shading` yang sama untuk beberapa bentuk mengurangi penggunaan memori dan membuat kode Anda lebih rapi.

## Kasus Penggunaan Umum untuk Gradien Radial
- **Pengisian latar belakang** untuk selebaran, brosur, atau sertifikat di mana pergeseran warna halus menambah kedalaman.  
- **Menyorot titik fokus** seperti logo atau tombol ajakan‑tindakan dalam PDF yang nantinya akan dikonversi ke PostScript.  
- **Menciptakan efek pencahayaan realistis** dalam diagram teknis atau skema teknik.

## Menambahkan Gradien Vertikal (Kata Kunci Sekunder)
Jika Anda juga perlu **menambahkan gradien vertikal** ke dokumen yang sama, API menyediakan helper `VerticalGradient` yang bekerja paralel dengan shading radial. Alur kerja meniru langkah‑langkah radial—hanya ganti jenis gradien. Ini memudahkan Anda melapisi gradien vertikal di bawah gradien radial untuk efek visual yang lebih kaya.
