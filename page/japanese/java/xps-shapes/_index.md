# 図形 - XPS

## はじめに

Javaで魅力的なXPSドキュメントを作成したい場合、**radial gradient ellipse** をマスターすることが最適な出発点です。Aspose.Page for Java を使用すれば、楕円や矩形、その他の幾何学的形状を簡単に描画でき、ドキュメントにプロフェッショナルな仕上げを施すことができます。このチュートリアルでは、基本的な手順を順に解説し、実際のユースケースを示し、各形状に関する詳細なサブチュートリアルへのリンクをご案内します。

## クイック回答
- **radial gradient ellipseとは何ですか？** 中心から外側へ放射状に色が変化するグラデーションで塗りつぶされた、ベクトルベースの楕円です。
- **Aspose.Page for Javaを使用する理由は？** 低レベルのXPSマークアップを扱うことなく、形状を描画できるシンプルなAPIを提供します。
- **形状の追加にどれくらい時間がかかりますか？** 基本的な楕円や矩形であれば、通常10分未満です。
- **ライセンスは必要ですか？** 開発目的であれば無料トライアルで利用可能です。商用環境では商用ライセンスが必要です。
- **サポートされているJavaバージョンは？** Java 8以降です。

## radial gradient ellipseとは何ですか？
**radial gradient ellipse** は、楕円の幾何学的な精度と、中心点から外側へ放射状に広がるグラデーションを組み合わせたものです。この効果は、セクションの強調、ボタン風のビジュアル作成、またはXPSページへのさりげないブランディング要素の追加に最適です。

## Java XPSで形状を描画する理由は？
- 視覚的にコンテンツを構造化する（例：コールアウトボックス、図）
- 色で塗りつぶしたグラフィックで重要情報を強調する
- ラスタ画像をスケーラブルなベクターアートに置き換え、ファイルサイズを小さく保つ
- レイアウトをプログラムで定義し、レポート生成を自動化する

## 主な使用例
- **レポートと請求書:** 合計金額をグラデーション塗りの楕円で強調する
- **技術マニュアル:** 矩形でコードスニペットや警告を枠付けする
- **マーケティングブローシャ:** XPS内で目を引くバッジやスタンプを作成する

## よくある質問

**Q: 形状作成後にグラデーションの色を変更できますか？**  
A: はい、`GradientStop` コレクションを変更し、楕円のストロークに再適用できます。

**Q: 矩形に影効果を追加することは可能ですか？**  
A: XPS にはネイティブな影機能はありませんが、元の矩形の背後に少しオフセットした半透明の矩形を描画することでシミュレートできます。

**Q: これらのメソッドはPDF変換でも機能しますか？**  
A: はい。Aspose.Page を使用すれば、XPS ドキュメントを PDF に変換し、すべてのベクトル形状を保持できます。

**Q: 形状を DPI に依存しないようにするにはどうすればよいですか？**  
A: XPS はベクトルグラフィックを使用しているため、形状は自動的にスケーリングされます。形状内部にラスタ画像を入れないようにしてください。

**Q: より複雑な多角形を描画する必要がある場合はどうすればよいですか？**  
A: `Path` クラスを使用してカスタムジオメトリを定義します。同じグラデーションとストローク設定が適用されます。
