## Aspose Page のテキスト追加アプローチでテキストを追加する方法は？

`Page` は XPS ドキュメント内の単一ページを表します。`TextFragment` クラスはページ上に配置できるテキストの断片をカプセル化します。XPS ドキュメントをロードし、`Page` オブジェクトを作成し、`TextFragment` クラスを使用して目的の文字列を挿入します。この **aspose page add text** ワークフローは API 呼び出しが 3 回だけで済み、すべてのサポートプラットフォームで一貫して動作し、既存のレイアウトと書式設定を保持しながらテキストを迅速に追加できます。
