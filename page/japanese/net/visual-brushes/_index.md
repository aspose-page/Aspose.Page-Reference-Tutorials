## Visual Brush とは何ですか？

Visual Brush は、図形、テキスト、画像などのビジュアル要素をキャプチャし、他の描画オブジェクトの塗りとして再利用できる機能です。このアプローチにより重複を減らし、ドキュメント全体で一貫したスタイルを確保できます。

## Grid Visual Brush を使用する理由は？

A Grid Visual Brush は、背景、ガイド、装飾要素として使用できるタイル状のグリッドパターンを作成します。以下のような場合に特に有用です：

- **一貫した配置** 複数ページにわたって。  
- **軽量なスタイリング** 大きな画像を埋め込まずに。  
- **動的スケーリング** 異なるページサイズに適応。

## よくある質問

**Q:** *単一のドキュメントで複数の visual brush を組み合わせることはできますか？*  
**A:** はい、異なる Visual Brushes を重ねて複雑なパターンや効果を作成できます。

**Q:** *実行時にグリッドサイズを変更することは可能ですか？*  
**A:** もちろんです。レンダリング前に `CellWidth` と `CellHeight` プロパティを調整して、グリッドを動的に変更できます。

**Q:** *Visual Brushes はファイルサイズに影響しますか？*  
**A:** ベクターベースであるため、ラスタ画像に比べてオーバーヘッドは最小限です。

**Q:** *PDF 出力で Visual Brush を使用できますか？*  
**A:** はい、Aspose.Page は PDF、XPS、その他複数のフォーマットをサポートし、Visual Brush の定義を保持します。

**Q:** *ブラシが表示されない場合、どのようなトラブルシューティング手順を取るべきですか？*  
**A:** ブラシが描画可能オブジェクトに割り当てられているか、ドキュメントのレンダリング設定が正しく構成されているかを確認してください。
