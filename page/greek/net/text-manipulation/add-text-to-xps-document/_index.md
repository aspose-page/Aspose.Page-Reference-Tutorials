# Δημιουργία εγγράφου XPS .NET και προσθήκη κειμένου με Aspose.Page

Στη σύγχρονη ανάπτυξη .NET, η δυνατότητα **δημιουργίας εγγράφου XPS .NET** προγραμματιστικά αποτελεί πολύτιμη δεξιότητα, ειδικά όταν χρειάζεται να δημιουργήσετε εκτυπώσιμες αναφορές, τιμολόγια ή προσαρμοσμένα γραφικά. Αυτό το tutorial σας οδηγεί στη χρήση του Aspose.Page για **aspose.page add text** σε ένα έγγραφο XPS, δίνοντάς σας πλήρη έλεγχο πάνω στη διάταξη και το στυλ—όλα από την .NET εφαρμογή σας.

## Γρήγορες Απαντήσεις
- **Τι καλύπτει αυτό το tutorial;** Προσθήκη κειμένου σε ένα νεοδημιουργημένο έγγραφο XPS χρησιμοποιώντας το Aspose.Page για .NET.  
- **Πόσο χρόνο διαρκεί;** Περίπου 5‑10 λεπτά για μια βασική υλοποίηση.  
- **Ποιες είναι οι προαπαιτήσεις;** Περιβάλλον ανάπτυξης .NET και η βιβλιοθήκη Aspose.Page.  
- **Απαιτείται άδεια;** Ναι, απαιτείται έγκυρη άδεια Aspose.Page για παραγωγική χρήση.  
- **Μπορεί να τρέξει σε .NET Core / .NET 6+;** Απόλυτα – το Aspose.Page υποστηρίζει όλες τις πρόσφατες εκδόσεις .NET.

## Τι είναι η δημιουργία εγγράφου XPS .NET;

Η δημιουργία ενός εγγράφου XPS σε .NET σημαίνει την παραγωγή ενός αρχείου σταθερής διάταξης που διατηρεί την ακριβή εμφάνιση του περιεχομένου σας σε διαφορετικές συσκευές και εκτυπωτές. Το XPS (XML Paper Specification) είναι το ανοιχτό πρότυπο της Microsoft για περιγραφή σελίδων, παρόμοιο με το PDF αλλά πλήρως βασισμένο σε XML.

## Γιατί να χρησιμοποιήσετε το Aspose.Page για προσθήκη κειμένου;

Το Aspose.Page προσφέρει ένα υψηλού επιπέδου, αντικειμενοστραφές API που αφαιρεί την ανάγκη χειρισμού χαμηλού επιπέδου XPS markup. Μπορείτε να προσθέσετε κείμενο, γραφικά και σχήματα χωρίς να ασχοληθείτε με ακατέργαστο XML, καθιστώντας τη διαδικασία ανάπτυξης πιο γρήγορη και λιγότερο επιρρεπή σε σφάλματα.

## Προαπαιτήσεις

- Aspose.Page for .NET: Βεβαιωθείτε ότι έχετε εγκαταστήσει τη βιβλιοθήκη Aspose.Page. Μπορείτε να τη κατεβάσετε από την [Aspose.Page for .NET documentation](https://reference.aspose.com/page/net/).
- Περιβάλλον Ανάπτυξης: Ρυθμίστε το περιβάλλον ανάπτυξης .NET. Αν δεν το έχετε κάνει ακόμη, ακολουθήστε τις οδηγίες εγκατάστασης που παρέχονται στην [documentation](https://reference.aspose.com/page/net/).
- Κατάλογος Εγγράφων: Δημιουργήστε έναν φάκελο όπου θα αποθηκεύετε τα έγγραφά σας. Αντικαταστήστε το «Your Document Directory» στο παρεχόμενο απόσπασμα κώδικα με την πραγματική διαδρομή.

Τώρα που καλύψαμε τα βασικά, ας προχωρήσουμε στον κώδικα.

## Εισαγωγή Namespaces

Πρώτα, εισάγετε τα απαραίτητα namespaces για να ξεκινήσετε το έργο:

```csharp
using Aspose.Page.XPS;
using Aspose.Page.XPS.XpsModel;
using System.Drawing;
```

## Βήμα 1: Δημιουργία εγγράφου XPS .NET

Δημιουργήστε ένα νέο έγγραφο XPS που θα λειτουργήσει ως καμβάς για το κείμενό μας.

```csharp
// ExStart:3
string dataDir = "Your Document Directory";
XpsDocument doc = new XpsDocument();
// ExEnd:3
```

## Βήμα 2: Δημιουργία πινέλου για το κείμενο

Ορίστε ένα πινέλο στερεού χρώματος που καθορίζει το χρώμα του κειμένου. Εδώ χρησιμοποιούμε το μαύρο.

```csharp
// ExStart:4
XpsSolidColorBrush textFill = doc.CreateSolidColorBrush(Color.Black);
// ExEnd:4
```

## Βήμα 3: Προσθήκη glyphs (aspose.page add text)

Τα glyphs είναι η χαμηλού επιπέδου αναπαράσταση χαρακτήρων σε ένα έγγραφο XPS. Αυτή η κλήση προσθέτει το κείμενο «Hello World!» στις καθορισμένες συντεταγμένες.

```csharp
// ExStart:5
XpsGlyphs glyphs = doc.AddGlyphs("Arial", 12, FontStyle.Regular, 300f, 450f, "Hello World!");
glyphs.Fill = textFill;
// ExEnd:5
```

## Βήμα 4: Αποθήκευση του τελικού εγγράφου XPS

Αποθηκεύστε το έγγραφο στο δίσκο ώστε να μπορείτε να το δείτε ή να το εκτυπώσετε αργότερα.

```csharp
// ExStart:6
doc.Save(dataDir + "AddText_out.xps");
// ExEnd:6
```

Ακολουθώντας αυτά τα βήματα, έχετε δημιουργήσει επιτυχώς **create XPS document .NET** και προσθέσει προσαρμοσμένο κείμενο χρησιμοποιώντας το Aspose.Page.

## Συνηθισμένα Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| **File not found** κατά την αποθήκευση | `dataDir` δείχνει σε φάκελο που δεν υπάρχει | Βεβαιωθείτε ότι ο φάκελος υπάρχει ή χρησιμοποιήστε `Directory.CreateDirectory(dataDir)` πριν την αποθήκευση. |
| **Text not visible** | Το χρώμα του πινέλου ταιριάζει με το φόντο | Αλλάξτε το `Color.Black` σε άλλο χρώμα με αντίθεση. |
| **Unsupported font** | Η γραμματοσειρά δεν είναι εγκατεστημένη στο σύστημα | Χρησιμοποιήστε μια γραμματοσειρά που είναι σίγουρα διαθέσιμη, ή ενσωματώστε τη γραμματοσειρά χρησιμοποιώντας τις δυνατότητες ενσωμάτωσης γραμματοσειρών του Aspose.Page. |
