# Πώς να Χρησιμοποιήσετε το Aspose.Page για Προσθήκη Επικαλυπτόμενης Εικόνας σε Έγγραφο XPS

## Εισαγωγή

Αν αναρωτιέστε **πώς να χρησιμοποιήσετε το Aspose** για να δώσετε στα αρχεία XPS σας πιο πλούσιο οπτικό στυλ, βρίσκεστε στο σωστό σημείο. Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα τις ακριβείς ενέργειες που απαιτούνται για **επίστρωση (tile) μιας εικόνας** μέσα σε ένα έγγραφο XPS χρησιμοποιώντας το Aspose.Page για .NET. Στο τέλος θα έχετε ένα επαναχρησιμοποιήσιμο snippet που μπορείτε να ενσωματώσετε σε οποιοδήποτε .NET project για δημιουργία γραφικών με επαναλαμβανόμενη εικόνα «on the fly».

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη χρειάζεται;** Aspose.Page για .NET  
- **Ποια μέθοδος δημιουργεί το tiled brush;** `CreateImageBrush` με `TileMode = XpsTileMode.Tile`  
- **Μπορώ να ελέγξω την αδιαφάνεια;** Ναι – ορίστε `path.Fill.Opacity` (π.χ., 0.5f)  
- **Χρειάζεται άδεια για δοκιμή;** Μια προσωρινή άδεια λειτουργεί για αξιολόγηση· απαιτείται πλήρης άδεια για παραγωγή.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6+

## Τι είναι το Aspose.Page και Γιατί να Επικαλύπτετε Εικόνες;

Το Aspose.Page είναι ένα ισχυρό API που επιτρέπει στους προγραμματιστές να δημιουργούν, να επεξεργάζονται και να αποδίδουν XPS, PDF και άλλες μορφές βασισμένες σε σελίδες χωρίς να εξαρτώνται από το Microsoft Office. Η επικάλυψη μιας εικόνας—η επανάληψη ενός bitmap σε ένα σχήμα—σας βοηθά να γεμίσετε μεγάλες περιοχές με μοτίβα, υδατογραφήματα ή υφές φόντου, διατηρώντας το μέγεθος του αρχείου μικρό.

## Πώς να Χρησιμοποιήσετε το Aspose.Page για Επικάλυψη Εικόνων σε Έγγραφα XPS

Παρακάτω θα βρείτε ένα πλήρες, έτοιμο‑για‑εκτέλεση παράδειγμα. Κάθε βήμα εξηγείται με απλή γλώσσα πριν από το αντίστοιχο μπλοκ κώδικα, ώστε να καταλάβετε **γιατί** κάθε γραμμή είναι σημαντική.

### Προαπαιτούμενα

- **Aspose.Page για .NET** – κατεβάστε και προσθέστε τη βιβλιοθήκη από την επίσημη σελίδα [εδώ](https://reference.aspose.com/page/net/).  
- **Περιβάλλον ανάπτυξης** – Visual Studio (οποιαδήποτε έκδοση) ή άλλο .NET IDE της προτίμησής σας.

### Εισαγωγή Ονομάτων Χώρων (Namespaces)

Πρώτα, φέρτε τα απαιτούμενα namespaces στο scope ώστε ο μεταγλωττιστής να γνωρίζει πού βρίσκονται οι κλάσεις XPS.

```csharp
using Aspose.Page.XPS;
using Aspose.Page.XPS.XpsModel;
using System.Drawing;
```

### Βήμα 1: Ορισμός Καταλόγου Εγγράφου

Καθορίστε πού θα αποθηκευτεί το παραγόμενο αρχείο XPS και η πηγαία εικόνα. Αντικαταστήστε το placeholder με έναν πραγματικό φάκελο στον υπολογιστή σας.

```csharp
// The path to the documents directory.
string dataDir = "Your Document Directory";
```

### Βήμα 2: Δημιουργία Νέου Εγγράφου XPS

Δημιουργήστε ένα κενό έγγραφο XPS που θα φιλοξενήσει το επικάλυπτο γραφικό.

```csharp
// Create new XPS Document
XpsDocument doc = new XpsDocument();
```

### Βήμα 3: Προσθήκη Επικαλυπτόμενης Εικόνας

Εδώ δημιουργούμε ένα ορθογώνιο path, το γεμίζουμε με ένα `ImageBrush` και ορίζουμε το brush σε λειτουργία tile. Η ιδιότητα `TileMode` λέει στη μηχανή να επαναλαμβάνει την εικόνα οριζόντια και κάθετα. Προσαρμόστε τις συντεταγμένες του ορθογωνίου ή την πηγή εικόνας ανάλογα με το σενάριό σας.

```csharp
// Tile image
// ImageBrush filled rectangle in the right top below
XpsPath path = doc.AddPath(doc.CreatePathGeometry("M 10,160 L 228,160 228,305 10,305"));
path.Fill = doc.CreateImageBrush(dataDir + "R08LN_NN.jpg", new RectangleF(0f, 0f, 128f, 96f), new RectangleF(0f, 0f, 64f, 48f));
((XpsImageBrush)path.Fill).TileMode = XpsTileMode.Tile;
path.Fill.Opacity = 0.5f;
```

### Βήμα 4: Αποθήκευση του Τελικού Εγγράφου XPS

Τέλος, γράψτε το έγγραφο στο δίσκο. Το αρχείο εξόδου μπορεί να ανοιχθεί με οποιονδήποτε XPS viewer ή να υποστεί περαιτέρω επεξεργασία με το Aspose.Page.

```csharp
// Save resultant XPS document
doc.Save(dataDir + "AddTiledImage_outXPS.xps");
```

## Συνηθισμένα Προβλήματα & Συμβουλές

- **Σφάλματα διαδρομής** – Βεβαιωθείτε ότι το `dataDir` τελειώνει με κάθετο ή χρησιμοποιήστε `Path.Combine` για να αποφύγετε προβλήματα έλλειψης διαχωριστικού.  
- **Ασυμφωνίες μεγέθους εικόνας** – Η πηγαία εικόνα πρέπει να είναι αρκετά μεγάλη για την περιοχή επικάλυψης· διαφορετικά το μοτίβο μπορεί να εμφανιστεί τεντωμένο.  
- **Η αδιαφάνεια δεν φαίνεται** – Κάποιοι viewers αγνοούν την αδιαφάνεια στα XPS· δοκιμάστε με έναν viewer που υποστηρίζει πλήρως την απόδοση XPS (π.χ., XPS Viewer στα Windows).
