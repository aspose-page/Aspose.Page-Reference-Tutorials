## Aspose.Page Προσθήκη Εικόνας σε Έγγραφο XPS

Παρακάτω περιγράφονται τα βήματα που απαιτούνται για **create XPS document .NET** και την ενσωμάτωση μιας εικόνας.

### Βήμα 1: Ορισμός Καταλόγου Εγγράφου

Καθορίστε τη διαδρομή προς τον κατάλογο του εγγράφου σας. Αυτό το βήμα διασφαλίζει ότι το project γνωρίζει πού θα βρει και θα αποθηκεύσει τα αρχεία.

```csharp
// ExStart:1
string dataDir = "Your Document Directory";
// ExEnd:1
```

### Βήμα 2: Δημιουργία Εγγράφου XPS

Δημιουργήστε ένα νέο έγγραφο XPS χρησιμοποιώντας το Aspose.Page για .NET.

```csharp
// ExStart:1
XpsDocument doc = new XpsDocument();
// ExEnd:1
```

### Βήμα 3: Προσθήκη Εικόνας στο Έγγραφο XPS

Τώρα, προσθέτουμε την εικόνα στο έγγραφο XPS. Στο παράδειγμα αυτό, χρησιμοποιούμε μια εικόνα δείγματος με όνομα "QL_logo_color.tif".

```csharp
// ExStart:1
XpsPath path = doc.AddPath(doc.CreatePathGeometry("M 30,20 l 258.24,0 0,56.64 -258.24,0 Z"));
path.RenderTransform = doc.CreateMatrix(0.7f, 0f, 0f, 0.7f, 0f, 20f);
path.Fill = doc.CreateImageBrush(dataDir + "QL_logo_color.tif", new RectangleF(0f, 0f, 258.24f, 56.64f), new RectangleF(50f, 20f, 193.68f, 42.48f));
// ExEnd:1
```

### Βήμα 4: Αποθήκευση του Τελικού Εγγράφου XPS

Αποθηκεύστε το έγγραφο XPS με την προστιθέμενη εικόνα.

```csharp
// ExStart:1
doc.Save(dataDir + "AddImage_outXPS.xps");
// ExEnd:1
```

Τώρα έχετε προσθέσει επιτυχώς μια εικόνα σε έγγραφο XPS χρησιμοποιώντας το Aspose.Page για .NET!

## Συχνά Προβλήματα και Λύσεις

- **Σφάλμα “File not found”** – Επαληθεύστε ότι το `dataDir` δείχνει στον σωστό φάκελο και ότι το όνομα του αρχείου εικόνας ταιριάζει ακριβώς, συμπεριλαμβανομένης της ευαισθησίας σε πεζά/κεφαλαία σε Linux.  
- **Η εικόνα εμφανίζεται παραμορφωμένη** – Προσαρμόστε τους παράγοντες κλιμάκωσης του `CreateMatrix` ή τις παραμέτρους του `RectangleF` για να ελέγξετε το μέγεθος και την αναλογία.  
- **Μη υποστηριζόμενη μορφή εικόνας** – Το Aspose.Page υποστηρίζει κοινές μορφές raster (PNG, JPEG, TIFF). Μετατρέψτε μη υποστηριζόμενες μορφές πριν χρησιμοποιήσετε το `CreateImageBrush`.
