# Προσθήκη Κατακόρυφης Διαβάθμισης σε XPS με Aspose.Page για .NET

## Εισαγωγή

Σε αυτό το tutorial θα **δημιουργήσετε ένα έγγραφο XPS** που περιλαμβάνει μια ομαλή κατακόρυφη διαβάθμιση. Η προσθήκη διαβαθμίσεων είναι ένας κοινός τρόπος να κάνετε τα αρχεία XPS να φαίνονται πιο επαγγελματικά — ιδανικό για εκθέσεις, φυλλάδια ή οποιοδήποτε εκτυπώσιμο γραφικό. Θα περάσουμε από κάθε βήμα, από τη ρύθμιση του έργου μέχρι την αποθήκευση του τελικού αρχείου XPS, ώστε να μπορείτε γρήγορα να εφαρμόσετε μια γραμμική διαβάθμιση σε οποιοδήποτε μονοπάτι.

## Γρήγορες Απαντήσεις
- **Τι καλύπτει αυτό το tutorial;** Προσθήκη κατακόρυφης γραμμικής διαβάθμισης σε μονοπάτι σε έγγραφο XPS.  
- **Ποια βιβλιοθήκη απαιτείται;** Aspose.Page για .NET.

## Τι είναι μια κατακόρυφη διαβάθμιση σε XPS;

Μια κατακόρυφη διαβάθμιση είναι μια μετάβαση χρώματος που τρέχει από την κορυφή ενός σχήματος μέχρι το κάτω μέρος. Στο XPS, αυτό επιτυγχάνεται με ένα **linear gradient brush** που ορίζει σημεία εκκίνησης και λήξης, καθώς και μια συλλογή από gradient stops που ελέγχουν το χρώμα σε συγκεκριμένες θέσεις.

- **Εύκολη συντήρηση** – σαφές αντικειμενοστραφές μοντέλο για paths, brushes και colors.

Ορίστε το φάκελο όπου θα αποθηκευτεί το παραγόμενο αρχείο XPS.

```csharp
// ExStart:3
string dataDir = "Your Document Directory";
// ExEnd:3
```

## Βήμα 2: Δημιουργία Νέου Εγγράφου XPS

Δημιουργήστε ένα νέο έγγραφο XPS που θα γεμίσει αργότερα με ένα μονοπάτι γεμάτο διαβάθμιση.

```csharp
// ExStart:4
XpsDocument doc = new XpsDocument();
// ExEnd:4
```

## Βήμα 3: Ορισμός Gradient Stops

Τα gradient stops καθορίζουν τα χρώματα και τις θέσεις τους κατά μήκος της γραμμής διαβάθμισης. Εδώ δημιουργούμε πέντε stops για μια ομαλή κατακόρυφη μετάβαση.

```csharp
// ExStart:5
List<XpsGradientStop> stops = new List<XpsGradientStop>();
stops.Add(doc.CreateGradientStop(doc.CreateColor(253, 255, 12, 0), 0f));
stops.Add(doc.CreateGradientStop(doc.CreateColor(252, 255, 154, 0), 0.359375f));
stops.Add(doc.CreateGradientStop(doc.CreateColor(252, 255, 56, 0), 0.424805f));
stops.Add(doc.CreateGradientStop(doc.CreateColor(253, 255, 229, 0), 0.879883f));
stops.Add(doc.CreateGradientStop(doc.CreateColor(252, 255, 255, 234), 1f));
// ExEnd:5
```

## Βήμα 4: Δημιουργία Path με Διαβάθμιση

Σχεδιάζουμε ένα ορθογώνιο path και εφαρμόζουμε ένα **linear gradient brush** που τρέχει κατακόρυφα από το σημείο (10, 110) στο σημείο (10, 200). Το brush λαμβάνει τα gradient stops που ορίσαμε νωρίτερα.

```csharp
// ExStart:6
XpsPath path = doc.AddPath(doc.CreatePathGeometry("M 10,110 L 228,110 228,200 10,200"));
path.RenderTransform = doc.CreateMatrix(1f, 0f, 0f, 1f, 20f, 70f);
path.Fill = doc.CreateLinearGradientBrush(new PointF(10f, 110f), new PointF(10f, 200f));
((XpsGradientBrush)path.Fill).GradientStops.AddRange(stops);
// ExEnd:6
```

## Βήμα 5: Αποθήκευση του Τελικού Εγγράφου XPS

Τέλος, γράψτε το έγγραφο XPS στο δίσκο. Αυτό το βήμα **save XPS file** παράγει το `AddVerticalGradient_outXPS.xps` στον φάκελο που καθορίσατε.

```csharp
// ExStart:7
doc.Save(dataDir + "AddVerticalGradient_outXPS.xps");
// ExEnd:7
```

**Συμβουλή:** Επαληθεύστε το αποτέλεσμα ανοίγοντας το αρχείο XPS στον Windows XPS Viewer ή σε οποιονδήποτε τρίτο viewer για να βεβαιωθείτε ότι η διαβάθμιση εμφανίζεται όπως αναμένεται.
