# Τροποποίηση εγγράφου XPS με Aspose.Page για .NET

## Εισαγωγή

Καλώς ήρθατε στον βήμα‑βήμα οδηγό μας για **πώς να τροποποιήσετε αρχεία xps** με το Aspose.Page για .NET. Είτε χρειάζεστε να εισάγετε μια υπογραφή, να προσθέσετε υδατογράφημα, ή απλώς να τοποθετήσετε προσαρμοσμένο κείμενο σε μια σελίδα, αυτό το tutorial σας δείχνει ακριβώς **πώς να προσθέσετε κείμενο** σε ένα έγγραφο XPS σε λίγα λεπτά. Θα περάσουμε από κάθε γραμμή κώδικα, θα εξηγήσουμε γιατί κάθε βήμα είναι σημαντικό, και θα σας δώσουμε συμβουλές για να αποφύγετε κοινά προβλήματα.

### Γρήγορες Απαντήσεις
- **Τι καλύπτει αυτό το tutorial;** Προσθήκη κειμένου υπογραφής ("Confirmed") σε επιλεγμένες σελίδες ενός αρχείου XPS.  
- **Ποια βιβλιοθήκη απαιτείται;** Aspose.Page για .NET (τελευταία έκδοση).  
- **Χρειάζομαι άδεια;** Μια προσωρινή άδεια λειτουργεί για δοκιμές· απαιτείται πλήρης άδεια για παραγωγή.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6+.  
- **Πόσο χρόνο διαρκεί η υλοποίηση;** Περίπου 10 λεπτά για μια βασική εισαγωγή υπογραφής.

## Τι είναι η τροποποίηση ενός εγγράφου XPS;

Το XPS (XML Paper Specification) είναι το σταθερό‑διάταξης φορμά εγγράφων της Microsoft, παρόμοιο με το PDF. Η τροποποίηση ενός εγγράφου XPS σημαίνει προγραμματιστική αλλαγή του οπτικού του περιεχομένου—προσθήκη κειμένου, εικόνων ή σχημάτων—χωρίς μετατροπή του αρχείου σε άλλο φορμά. Το Aspose.Page παρέχει ένα πλούσιο αντικειμενικό μοντέλο που σας επιτρέπει να επεξεργάζεστε αρχεία XPS απευθείας από τον κώδικα .NET.

## Γιατί να χρησιμοποιήσετε το Aspose.Page για την τροποποίηση εγγράφων XPS;

* **Πλήρης έλεγχος** – εργασία με σελίδες, γλύφους, πινέλα και μετασχηματισμούς σε χαμηλό επίπεδο.  
* **Χωρίς εξωτερικές εξαρτήσεις** – καθαρή βιβλιοθήκη .NET, χωρίς ανάγκη για Office ή Adobe components.  
* **Διασυστημική** – λειτουργεί σε Windows, Linux και macOS μέσω .NET Core.  
* **Αξιόπιστη απόδοση** – διαχειρίζεται μεγάλα έγγραφα αποδοτικά και υποστηρίζει προχωρημένη τυπογραφία.

## Προαπαιτούμενα

Πριν ξεκινήσετε, βεβαιωθείτε ότι διαθέτετε τα εξής:

- **Aspose.Page για .NET** – Εγκαταστήστε το πακέτο NuGet ή κατεβάστε τη βιβλιοθήκη από την επίσημη τεκμηρίωση **[εδώ](https://reference.aspose.com/page/net/)**.  
- **Αρχείο εισόδου XPS** – Αποκτήστε ένα δείγμα εγγράφου XPS (π.χ. `input1.xps`) από τη **[σελίδα εκδόσεων Aspose](https://releases.aspose.com/page/net/)**.  
- **Φάκελος εργασίας** – Δημιουργήστε έναν φάκελο στον υπολογιστή σας για να αποθηκεύσετε τα αρχεία εισόδου και εξόδου και σημειώστε τη πλήρη διαδρομή του· θα την αντιστοιχίσετε στη μεταβλητή `dir` στον κώδικα.  
- **Περιβάλλον ανάπτυξης** – Visual Studio 2019/2022, .NET Framework 4.7.2 ή νεότερο, ή οποιοδήποτε έργο .NET Core/5/6.

Τώρα που όλα είναι έτοιμα, ας βουτήξουμε στον κώδικα.

## Εισαγωγή Namespaces

Στο .NET project σας, ξεκινήστε εισάγοντας τα απαιτούμενα namespaces για το Aspose.Page:

```csharp
using Aspose.Page.XPS;
using Aspose.Page.XPS.XpsModel;
using System.Drawing;
using System.IO;
```

## Βήμα 1: Άνοιγμα ροής εγγράφου XPS

Θα ανοίξουμε το πηγαίο αρχείο XPS ως ροή και θα δημιουργήσουμε ένα αντικείμενο `XpsDocument` που αντιπροσωπεύει ολόκληρο το έγγραφο.

```csharp
// ExStart:3
// The path to the documents directory.
string dir = "Your Document Directory";
// Open a stream of XPS file
using (FileStream xpsStream = File.Open(dir + "input1.xps", FileMode.Open, FileAccess.Read))
{
    // Create PS document from stream
    XpsDocument document = new XpsDocument(xpsStream, new XpsLoadOptions());
    // Continue to the next step...
}
// ExEnd:3
```

*Συμβουλή:* Τυλίξτε τη ροή μέσα σε ένα `using` block ώστε να απελευθερώνεται αυτόματα το handle του αρχείου.

## Βήμα 2: Δημιουργία κειμένου υπογραφής

Στη συνέχεια, δημιουργούμε ένα πινέλο μονής απόχρωσης που θα χρησιμοποιηθεί για τη βαφή των γλύφων της υπογραφής.

```csharp
// ExStart:4
// Create fill of the signature text
XpsSolidColorBrush textFill = document.CreateSolidColorBrush(Color.BlueViolet);
// Continue to the next step...
// ExEnd:4
```

Μπορείτε να αλλάξετε το `Color.BlueViolet` σε οποιοδήποτε `System.Drawing.Color` ταιριάζει με το branding σας.

## Βήμα 3: Ορισμός σελίδων και προσθήκη υπογραφής

Θα καθορίσουμε ποιες σελίδες πρέπει να λάβουν την υπογραφή και στη συνέχεια θα προσθέσουμε τους γλύφους σε κάθε σελίδα.

```csharp
// ExStart:5
// Define pages where signature will be set
int[] pageNumbers = new int[] {1, 2, 3};

// For every defined page set signature "Confirmed" at coordinates x=650 and y=950
for (int i = 0; i < pageNumbers.Length; i++)
{
    // Define active page
    document.SelectActivePage(pageNumbers[i]);

    // Create glyphs object
    XpsGlyphs glyphs = document.AddGlyphs("Arial", 24, FontStyle.Bold, 650, 900, "Confirmed");

    // Define fill for glyphs
    glyphs.Fill = textFill;
}
// Continue to the next step...
// ExEnd:5
```

*Γιατί αυτές οι συντεταγμένες;* Οι τιμές X και Y μετρώνται σε points (1/72 inch). Προσαρμόστε τις ώστε να τοποθετήσετε το κείμενο ακριβώς εκεί που χρειάζεστε στο layout της σελίδας.

## Βήμα 4: Αποθήκευση αλλαγών στο έγγραφο XPS

Τέλος, γράψτε το τροποποιημένο έγγραφο ξανά στο δίσκο.

```csharp
// ExStart:6
// Save changed XPS document
document.Save(dir + "input1_out.xps");
// ExEnd:6
```

Το νέο αρχείο `input1_out.xps` περιέχει τώρα την υπογραφή “Confirmed” στις σελίδες 1‑3.

## Συνηθισμένα Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Λύση |
|----------|-------|------|
| **Η υπογραφή δεν είναι ορατή** | Λανθασμένες συντεταγμένες ή μη επιλεγμένη σελίδα | Επαληθεύστε ότι καλείται `SelectActivePage` για κάθε σελίδα και προσαρμόστε τις τιμές X/Y. |
| **Εξαίρεση στο `AddGlyphs`** | Η γραμματοσειρά δεν είναι εγκατεστημένη στον server | Βεβαιωθείτε ότι η συγκεκριμένη γραμματοσειρά (π.χ. Arial) είναι διαθέσιμη, ή ενσωματώστε προσαρμοσμένη γραμματοσειρά με `document.AddFont`. |
| **Το αρχείο εξόδου είναι κατεστραμμένο** | Η ροή δεν κλείνει σωστά | Χρησιμοποιήστε δηλώσεις `using` για όλες τις ροές και καλέστε `document.Dispose()` αν χρειάζεται. |
| **Μείωση απόδοσης σε μεγάλα αρχεία** | Φόρτωση ολόκληρου εγγράφου στη μνήμη | Επεξεργαστείτε τις σελίδες σε παρτίδες ή χρησιμοποιήστε `XpsLoadOptions` με επιλογές streaming (αν είναι διαθέσιμες σε νεότερες εκδόσεις). |
