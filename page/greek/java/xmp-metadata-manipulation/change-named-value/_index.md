## Τι είναι τα μεταδεδομένα XMP;
Το XMP (Extensible Metadata Platform) είναι μια τυποποιημένη μορφή βασισμένη σε XML για την ενσωμάτωση πλούσιων μεταδεδομένων—όπως δημιουργός, τίτλος και προσαρμοσμένες ιδιότητες—απευθείας μέσα σε αρχεία. Σε έγγραφα EPS, το XMP συνυπάρχει με τα παραδοσιακά σχόλια PostScript, επιτρέποντας στις εφαρμογές να διαβάζουν και να τροποποιούν τα μεταδεδομένα χωρίς να αλλάζουν το οπτικό περιεχόμενο.

Το Aspose.Page σας παρέχει **πλήρη έλεγχο πάνω από 150 ιδιότητες σχήματος XMP** και λειτουργεί σταθερά σε μορφές EPS, PDF και SVG. Επεξεργάζεται αρχεία έως **500 MB** χωρίς να φορτώνει ολόκληρο το έγγραφο στη μνήμη, και περιλαμβάνει ενσωματωμένη επικύρωση που εγγυάται ότι το παραγόμενο EPS παραμένει σύμφωνο με τα πρότυπα. ## Εισαγωγή Πακέτων
Οι δηλώσεις `import` φέρνουν τις απαραίτητες κλάσεις του Aspose.Page στο πεδίο ορατότητας.

Η κλάση `PsDocument` είναι το βασικό αντικείμενο του Aspose.Page που αντιπροσωπεύει ένα έγγραφο EPS στη μνήμη.  
Η κλάση `XmpMetadata` παρέχει πρόσβαση στο πακέτο XMP που είναι ενσωματωμένο στο αρχείο EPS.  

```java
import java.io.FileInputStream;
import java.io.FileOutputStream;
import com.aspose.eps.PsDocument;
import com.aspose.eps.xmp.XmpMetadata;
import com.aspose.eps.xmp.XmpValue;
import com.aspose.page.BaseExamplesTest;
import com.aspose.page.License;
```

## Πώς να επεξεργαστείτε τα μεταδεδομένα XMP;
Φορτώστε το αρχείο EPS, ανακτήστε το πακέτο XMP, τροποποιήστε την επιθυμητή ιδιότητα και αποθηκεύστε το έγγραφο—όλα σε λίγες απλές γραμμές. Αυτή η προσέγγιση λειτουργεί για οποιαδήποτε ονομαστική τιμή XMP, είτε πρόκειται για τυπικό πεδίο Dublin Core είτε για προσαρμοσμένο namespace που έχετε ορίσει.

## Βήμα 1: Αρχικοποίηση Ροής Εισόδου Αρχείου EPS
`FileInputStream` ανοίγει μια ροή μόνο για ανάγνωση προς το πηγαίο αρχείο EPS, επιτρέποντας στο Aspose.Page να επεξεργαστεί το έγγραφο χωρίς να κλειδώνει το σύστημα αρχείων.

```java
// The path to the documents directory.
String dataDir = "Your Document Directory";
        
// Initialize input EPS file stream
FileInputStream psStream = new FileInputStream(dataDir + "xmp4.eps");
```

## Βήμα 2: Αρχικοποίηση PsDocument
`PsDocument` είναι το κύριο αντικείμενο που περιλαμβάνει το περιεχόμενο EPS και παρέχει πρόσβαση στα μεταδεδομένα, τις σελίδες και τους πόρους του.

```java
PsDocument document = new PsDocument(psStream);
```

## Βήμα 3: Λήψη Μεταδεδομένων XMP
`XmpMetadata` αντιπροσωπεύει το πακέτο XMP μέσα στο αρχείο EPS και προσφέρει μεθόδους για ανάγνωση, τροποποίηση ή διαγραφή μεμονωμένων ιδιοτήτων.

```java
// Get XMP metadata. If EPS file doesn't contain XMP metadata, we get a new one filled with values from PS metadata comments (%%Creator, %%CreateDate, %%Title, etc.)
XmpMetadata xmp = document.getXmpMetadata();
```

## Βήμα 4: Ορισμός Νέας Τιμής στο XMP
`setProperty` ορίζει την τιμή μιας συγκεκριμένης ιδιότητας XMP στο πακέτο μεταδεδομένων.

```java
// Set new string value "Inches" for named value "stDim:unit" of structure "xmpTPg:MaxPageSize" 
xmp.setNamedValue("xmpTPg:MaxPageSize", "stDim:unit", new XmpValue("Inches"));
```

## Βήμα 5: Αρχικοποίηση Ροής Εξόδου Αρχείου EPS
`FileOutputStream` δημιουργεί μια ροή εγγραφής για την αποθήκευση του τροποποιημένου αρχείου EPS.

```java
// Initialize output EPS file stream
FileOutputStream outPsStream = new FileOutputStream(dataDir + "xmp4_changed.eps");
```

## Βήμα 6: Αποθήκευση Εγγράφου με Τροποποιημένα Μεταδεδομένα XMP
`save` γράφει το PsDocument στη μνήμη, συμπεριλαμβανομένου του ενημερωμένου XMP, στην έξοδο ροής.

```java
// Save document with changed XMP metadata
try {			
    document.save(outPsStream);
} finally {
    outPsStream.close();
}
```

## Βήμα 7: Κλείσιμο Ροής Εισόδου EPS
`close` απελευθερώνει το χειριστήριο του αρχείου και ελευθερώνει τους πόρους που σχετίζονται με τη ροή εισόδου.

```java
// Close input EPS stream
psStream.close();
```

## Συνηθισμένα Προβλήματα και Λύσεις
- **FileNotFoundException** – Επαληθεύστε ότι το `dataDir` δείχνει στο σωστό φάκελο και ότι το `xmp4.eps` υπάρχει.  
- **LicenseException** – Εάν εμφανίζονται σφάλματα άδειας, βεβαιωθείτε ότι ένα έγκυρο αρχείο άδειας Aspose.Page έχει φορτωθεί πριν δημιουργήσετε το `PsDocument`.  
- **Metadata Not Updated** – Μετά την αποθήκευση, ανοίξτε το παραγόμενο EPS σε προβολέα μεταδεδομένων (π.χ., Adobe Bridge) για να επιβεβαιώσετε ότι η νέα τιμή εμφανίζεται.
