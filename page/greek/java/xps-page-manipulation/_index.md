## Πώς να προσθέσετε xps σελίδες java

Φορτώστε ένα υπάρχον έγγραφο XPS, καλέστε τη μέθοδο `addPage()` στο αντικείμενο `Document` και, στη συνέχεια, αποθηκεύστε τις αλλαγές. Η κλάση `Document` αντιπροσωπεύει ένα πακέτο XPS, εκθέτοντας τις σελίδες και τους πόρους του. Η `addPage()` δημιουργεί μια νέα κενή σελίδα και επιστρέφει ένα αντικείμενο `Page`. Αυτή η απλή τρι‑βήμα ροή — **load → add → save** — καλύπτει το 95 % των πραγματικών σεναρίων, όπως η δημιουργία πολυ‑σελίδων τιμολογίων, η προσθήκη αναφορών ή η κατασκευή σύνθετων εγγράφων από πρότυπα. Το API ενημερώνει αυτόματα τις αναφορές σελίδων, τα λεξικά πόρων και το εσωτερικό μανιφέστο του εγγράφου, ώστε να μην χρειάζεται ποτέ να επεμβείτε σε χαμηλού επιπέδου XML.

### Βήμα 1: Φόρτωση του πηγαίου αρχείου XPS

Αρχικά, δημιουργήστε μια παρουσία της κλάσης `Document` με τη διαδρομή προς το πηγαίο αρχείο σας. Ο κατασκευαστής αναλύει το πακέτο και δημιουργεί μια αναπαράσταση στη μνήμη.

### Βήμα 2: Προσθήκη νέας κενής σελίδας

Καλέστε `document.getPages().add()` για να προσαρτήσετε μια νέα σελίδα στο τέλος, ή χρησιμοποιήστε την υπερφόρτωση που δέχεται δείκτη για να εισάγετε σε συγκεκριμένη θέση. Μπορείτε επίσης να περάσετε ένα αντικείμενο `Page` αν θέλετε να κλωνοποιήσετε μια υπάρχουσα διάταξη σελίδας.

### Βήμα 3: Αποθήκευση του ενημερωμένου εγγράφου

Τέλος, καλέστε `document.save("output.xps")`. Η βιβλιοθήκη γράφει ένα πλήρως συμμορφωμένο πακέτο XPS, διατηρώντας το υπάρχον περιεχόμενο και ενσωματώνοντας τυχόν νέους πόρους που προσθέσατε (γραμματοσειρές, εικόνες κ.λπ.).

## Αντιμετώπιση Προβλημάτων & Συμβουλές

- **Προβλήματα μνήμης σε πολύ μεγάλα αρχεία:** Ενεργοποιήστε `Document.setLoadOptions(LoadOptions.withMemoryOptimization(true))` για να μεταφέρετε τις σελίδες αντί να φορτώνετε ολόκληρο το έγγραφο στη RAM.  
- **Λείπουν γραμματοσειρές:** Εάν μια νέα σελίδα αναφέρεται σε γραμματοσειρά που δεν είναι ενσωματωμένη στο αρχικό αρχείο, χρησιμοποιήστε `FontRepository.addFont("path/to/font.ttf")` πριν από την αποθήκευση.  
- **Σφάλματα σειράς σελίδων:** Θυμηθείτε ότι οι δείκτες σελίδων είναι μηδενικής βάσης· η εισαγωγή στο index 0 τοποθετεί τη νέα σελίδα στην αρχή.
