# Πώς να αλλάξετε το μέγεθος αρχείων EPS σε Java με το Aspose.Page

## Εισαγωγή
Αν ψάχνετε για έναν αξιόπιστο τρόπο **πώς να αλλάξετε το μέγεθος EPS** προγραμματιστικά, βρίσκεστε στο σωστό μέρος. Σε αυτό το tutorial θα περάσουμε από τη διαδικασία αλλαγής μεγέθους εικόνων EPS σε Java χρησιμοποιώντας τη βιβλιοθήκη Aspose.Page. Είτε χρειάζεστε να διπλασιάσετε το μέγεθος, να το μει διάσταση, είτε να δουλέψετε με ποσοστά, τα παρακάτω βήματα σας δίνουν πλήρη έλεγχο στις διαστάσεις εξόδου.

## Γρήγορες απαντήσεις
- **Ποια βιβλιοθήκη χρειάζεται;** Aspose.Page for Java  
- **Μπορώ να αλλάξω το μέγεθος χρησιμοποιώντας σημεία, ίντσες ή χιλιοστά;** Ναι – το API υποστηρίζει και τις τρεις μονάδες συν τα ποσοστά.  
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια δωρεάν δοκιμή λειτουργεί για δοκιμές· απαιτείται άδεια για παραγωγή.  
- **Ποια έκδοση Java απαιτείται;** Java 8 ή νεότερη.  
- **Είναι ο κώδικας ασφαλής για νήματα;** Κάθε αντικείμενο `PsDocument` είναι απομονωμένο, έτσι μπορείτε να επεξεργάζεστε αρχεία παράλληλα.

## Προαπαιτούμενα
Πριν βυθιστούμε στον κώδικα, βεβαιωθείτε ότι έχετε τα παρακάτω:

- Java Development Kit (JDK) εγκατεστημένο στον υπολογιστή σας.  
- Βιβλιοθήκη Aspose.Page for Java. Μπορείτε να την κατεβάσετε **[here](https://releases.aspose.com/page/java/)**.  
- Βασική κατανόηση του προγραμματισμού Java.  

## Εισαγωγή πακέτων
Στο έργο Java, συμπεριλάβετε τις απαιτούμενες εισαγωγές ώστε να μπορείτε να εργάζεστε με αντικείμενα Aspose.Page και τυπικές ροές I/O.

```java
import java.awt.Dimension;
import java.io.FileInputStream;
import java.io.FileOutputStream;
import com.aspose.eps.PsDocument;
import com.aspose.page.DimensionF;
import com.aspose.page.Units;
```

## Πώς να αλλάξετε το μέγεθος EPS χρησιμοποιώντας σημεία
Παρακάτω υπάρχει ένα πλήρες, βήμα‑βήμα παράδειγμα που διπλασιάζει το μέγεθος ενός αρχείου EPS χρησιμοποιώντας τα σημεία ως μονάδα μέτρησης.

### Βήμα 1: Ρύθμιση της εισόδου ροής
```java
FileInputStream inputEpsStream = new FileInputStream(dataDir + "input.eps");
```

### Βήμα 2: Αρχικοποίηση του αντικειμένου `PsDocument`
```java
PsDocument doc = new PsDocument(inputEpsStream);
```

### Βήμα 3: Εξαγωγή του τρέχοντος μεγέθους της εικόνας EPS
```java
Dimension oldSize = doc.extractEpsSize();
```

### Βήμα 4: Δημιουργία ροής εξόδου για το αρχείο με νέο μέγεθος
```java
FileOutputStream outputEpsStream = new FileOutputStream(dataDir + "output_resize_points.eps");
```

### Βήμα 5: Αλλαγή μεγέθους και αποθήκευση του EPS χρησιμοποιώντας σημεία
```java
doc.resizeEps(outputEpsStream, new Dimension2D.Double(oldSize.width * 2, oldSize.height * 2), Units.Points);
```

## Πώς να αλλάξετε το μέγεθος EPS χρησιμοποιώντας ίντσες
Το ίδιο μοτίβο πέντε βημάτων ισχύει· απλώς αντικαταστήστε το `Units.Points` με `Units.Inches` και προσαρμόστε τον συντελεστή κλιμάκωσης όπως απαιτείται.

## Πώς να αλλάξετε το μέγεθος EPS χρησιμοποιώντας χιλιοστά
Ξανά, ακολουθήστε τα ίδια βήματα, αλλάζοντας τη μονάδα σε `Units.Millimeters`. Αυτό είναι χρήσιμο όταν χρειάζεστε διαστάσεις σε μετρικό σύστημα για διαδικασίες εκτύπωσης.

## Πώς να αλλάξετε το μέγεθος EPS χρησιμοποιώντας ποσοστά
Για κλιμάκωση βάσει ποσοστών, διατηρήστε τη μονάδα ως `Units.Percent`. Πολλαπλασιάστε το αρχικό πλάτος και ύψος με το επιθυμητό ποσοστό (π.χ., `0.5` για μείωση 50 %).

## Συνηθισμένα λάθη & Συμβουλές
- **Πάντα κλείστε τις ροές** – Σε κώδικα παραγωγής, τυλίξτε τις ροές σε try‑with‑resources για να αποφύγετε κλειδώματα αρχείων.  
- **Διατηρήστε την αναλογία διαστάσεων** – Πολλαπλασιάστε τόσο το πλάτος όσο και το ύψος με τον ίδιο συντελεστή εκτός αν θέλετε σκόπιμα παραμόρφωση.  
- **Ελέγξτε το DPI** – Η αλλαγή μεγέθους δεν αλλάζει το DPI· εάν χρειάζεστε διαφορετικό DPI, προσαρμόστε το ξεχωριστά μετά την αλλαγή μεγέθους.
