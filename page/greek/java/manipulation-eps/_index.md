## Συνηθισμένα Προβλήματα & Συμβουλές

- **Missing fonts** – Βεβαιωθείτε ότι οι γραμματοσειρές που αναφέρονται στο EPS είναι εγκατεστημένες στο σύστημα ή ενσωματώστε τις χρησιμοποιώντας το API.

## Συχνές Ερωτήσεις

**Q: Μπορώ να επεξεργαστώ πολλαπλά αρχεία EPS σε batch;**  
A: Ναι. Τυλίξτε τη λογική περικοπής ή αλλαγής μεγέθους μέσα σε βρόχο και επαναχρησιμοποιήστε το ίδιο αντικείμενο `Page` για κάθε αρχείο.

**Q: Υποστηρίζει το Aspose.Page EPS με ενσωματωμένες ραστερ εικόνες;**  
A: Απόλυτα. Και τα διανυσματικά και τα ραστερ στοιχεία διατηρούνται κατά τη διάρκεια της επεξεργασίας.

**Q: Είναι δυνατόν να μετατρέψω το EPS σε άλλες μορφές (PDF, PNG) μετά την επεξεργασία;**  
A: Ναι. Μετά την περικοπή ή την αλλαγή μεγέθους, μπορείτε να καλέσετε `page.save("output.pdf", SaveFormat.Pdf)` ή `SaveFormat.Png`.

**Q: Τι γίνεται αν το αρχείο EPS είναι κατεστραμμένο;**  
A: Το API ρίχνει μια `FileFormatException`. Πιάστε την εξαίρεση και επικυρώστε το αρχείο προέλευσης πριν την επεξεργασία.

**Q: Πρέπει να κλείσω κάποιους πόρους χειροκίνητα;**  
A: Η κλάση `Page` υλοποιεί το `Closeable`; χρησιμοποιήστε ένα μπλοκ try‑with‑resources ή καλέστε `page.close()` για να ελευθερώσετε μνήμη.
