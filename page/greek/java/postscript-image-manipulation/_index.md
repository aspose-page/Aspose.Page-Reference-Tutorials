## Πώς να προσθέσετε εικόνα – Επισκόπηση βήμα‑βήμα

Ακολουθεί ένας σύντομος χάρτης πορείας που μπορείτε να ακολουθήσετε μετά την εγκατάσταση της βιβλιοθήκης:

1. **Create a `Document` object** που αντιπροσωπεύει το αρχείο PostScript που θέλετε να επεξεργαστείτε.  
2. **Instantiate an `Image` object** από αρχείο, ροή ή byte array.  
3. **Define the placement rectangle** (X, Y, width, height) όπου θα εμφανιστεί η εικόνα.  
4. **Call `document.addImage(image, rect)`** για την ενσωμάτωση του γραφικού.  
5. **Save the updated document** ξανά στο δίσκο ή σε ροή.  

Κάθε μία από αυτές τις ενέργειες παρουσιάζεται στο συνδεδεμένο tutorial “Add Image in Java PostScript”, ώστε να μπορείτε να αντιγράψετε‑και‑επικολλήσετε τα ακριβή αποσπάσματα κώδικα στο έργο σας.

**Q: Μπορώ να προσθέσω πολλές εικόνες στην ίδια σελίδα PostScript;**  
A: Ναι. Καλέστε τη μέθοδο `addImage` επανειλημμένα με διαφορετικά ορθογώνια τοποθέτησης.

**Q: Υποστηρίζει το Aspose.Page επίσης γραφικά διανυσματικά;**  
A: Απόλυτα. Μπορείτε να ενσωματώσετε SVG, EPS ή ακόμη και ακατέργαστες εντολές PostScript μαζί με εικόνες raster.

**Q: Ποιες εκδόσεις της Java είναι συμβατές;**  
A: Η βιβλιοθήκη λειτουργεί με Java 8 και νεότερες, συμπεριλαμβανομένων των Java 11, 17 και μεταγενέστερων εκδόσεων LTS.

**Q: Υπάρχει τρόπος να περιστρέψετε μια εικόνα κατά την προσθήκη της;**  
A: Ναι. Χρησιμοποιήστε το API μετασχηματισμού `Matrix` για να ορίσετε την περιστροφή πριν καλέσετε το `addImage`.
