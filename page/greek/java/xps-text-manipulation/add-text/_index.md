# Πώς να Προσθέσετε Κείμενο XPS σε Java – Εγχειρίδιο Aspose.Page

## Εισαγωγή
Αν χρειάζεστε να **προσθέσετε κείμενο XPS** προγραμματιστικά, το Aspose.Page for Java σας παρέχει ένα καθαρό, υψηλής απόδοσης API για εργασία με αρχεία XPS (XML Paper Specification). Σε αυτό το εγχειρίδιο θα περάσουμε από τη δημιουργία ενός εγγράφου XPS, την εισαγωγή μορφοποιημένου κειμένου και την αποθήκευση του αποτελέσματος — όλα με σύντομο, εύκολο στην κατανόηση κώδικα. Είτε δημιουργείτε μηχανή αναφορών, παράγετε τιμολόγια, είτε απλώς χρειάζεστε να τοποθετήσετε κείμενο πάνω σε μια σελίδα, αυτά τα βήματα θα σας θέσουν σε λειτουργία γρήγορα.

## Συχνές Απαντήσεις
- **Ποια βιβλιοθήκη είναι η καλύτερη για χειρισμό XPS σε Java;** Aspose.Page for Java.
- **Μπορώ να δημιουργήσω και να αποθηκεύσω αρχεία XPS χωρίς άδεια;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται άδεια για παραγωγή.
- **Ποια IDE υποστηρίζονται;** IntelliJ IDEA, Eclipse, NetBeans ή οποιοδήποτε IDE που υποστηρίζει Java.
- **Πόσες γραμμές κώδικα χρειάζονται για την προσθήκη απλού κειμένου;** Περίπου 10 γραμμές συν την προετοιμασία.
- **Είναι δυνατή η μορφοποίηση γραμματοσειράς;** Ναι – μπορείτε να ορίσετε οικογένεια γραμματοσειράς, μέγεθος, στυλ και χρώμα.

## Τι είναι το XPS και γιατί να προσθέσετε κείμενο;
Το XPS (XML Paper Specification) είναι η σταθερή μορφή εγγράφου της Microsoft, παρόμοια με το PDF αλλά βασισμένη σε XML. Η προσθήκη κειμένου σε αρχείο XPS είναι συχνή όταν χρειάζεται ακριβής τοποθέτηση, όπως ετικέτες, υδατογραφήματα ή δυναμικά δεδομένα σε αναφορές. Χρησιμοποιώντας το Aspose.Page μπορείτε να χειριστείτε XPS χωρίς να ασχοληθείτε με χαμηλού επιπέδου δομές XML.

## Γιατί να χρησιμοποιήσετε το Aspose.Page για την προσθήκη κειμένου XPS;
- **Πλήρης έλεγχος** της τοποθέτησης των γλύφων, των στυλ γραμματοσειράς και των πινέλων.  
- **Χωρίς εξωτερικές εξαρτήσεις** – καθαρό Java API.  
- **Υψηλή πιστότητα** απόδοση που διατηρεί τη διάταξη σε όλες τις πλατφόρμες.  
- **Πλήρης τεκμηρίωση** και δείγμα κώδικα για γρήγορη εκμάθηση.

## Προαπαιτούμενα
Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε:

- **Java Development Kit (JDK)** – εγκατεστημένη πρόσφατη έκδοση (8 ή νεότερη).
- **Aspose.Page for Java** – κατεβάστε τη βιβλιοθήκη από την επίσημη ιστοσελίδα [εδώ](https://releases.aspose.com/page/java/).
- **Ένα Java IDE** – IntelliJ IDEA, Eclipse ή οποιονδήποτε επεξεργαστή προτιμάτε.

## Εισαγωγή Πακέτων
Ξεκινήστε εισάγοντας τις κλάσεις που θα χρειαστείτε για τη διαχείριση XPS:

```java
import java.awt.Color;
import com.aspose.xps.XpsDocument;
import com.aspose.xps.XpsFontStyle;
import com.aspose.xps.XpsGlyphs;
import com.aspose.xps.XpsSolidColorBrush;
```

## Βήμα 1: Ορισμός Καταλόγου Εγγράφου (δημιουργία αρχείου xps)
Ορίστε πού θα αποθηκευτεί το παραγόμενο αρχείο XPS:

```java
String dataDir = "Your Document Directory";
```

## Βήμα 2: Δημιουργία Εγγράφου XPS (δημιουργία εγγράφου xps)
Δημιουργήστε ένα νέο, κενό έγγραφο XPS:

```java
XpsDocument doc = new XpsDocument();
```

## Βήμα 3: Δημιουργία Πινέλου για Μορφοποίηση Κειμένου
Ένα πινέλο μονής χρώματος καθορίζει το χρώμα του κειμένου. Εδώ χρησιμοποιούμε μαύρο:

```java
XpsSolidColorBrush textFill = doc.createSolidColorBrush(Color.BLACK);
```

## Βήμα 4: Προσθήκη Γλύφων – το Πραγματικό Κείμενο (aspose.page add text)
Προσθέστε το κείμενο που θέλετε να εμφανιστεί στο έγγραφο. Η μέθοδος `addGlyphs` σας επιτρέπει να καθορίσετε γραμματοσειρά, μέγεθος, στυλ και ακριβείς συντεταγμένες:

```java
XpsGlyphs glyphs = doc.addGlyphs("Arial", 12, XpsFontStyle.Regular, 300f, 450f, "Hello World!");
glyphs.setFill(textFill);
```

## Βήμα 5: Αποθήκευση του Τελικού Εγγράφου XPS (αποθήκευση αρχείου xps)
Τέλος, γράψτε το έγγραφο στο δίσκο:

```java
doc.save(dataDir + "AddText_out.xps");
```

Επαναλάβετε το βήμα **Add Glyphs** όσο χρειάζεται για να εισάγετε πρόσθετες γραμμές, να αλλάξετε γραμματοσειρές ή να προσαρμόσετε θέσεις.

## Κοινά Προβλήματα & Συμβουλές
- **Λανθασμένες συντεταγμένες:** Το XPS χρησιμοποιεί σύστημα συντεταγμένων σε σημεία (1/72 ίντσα). Προσαρμόστε τις τιμές `x` και `y` για ακριβή τοποθέτηση του κειμένου.
- **Γραμματοσειρά δεν βρέθηκε:** Βεβαιωθείτε ότι το όνομα της γραμματοσειράς (π.χ., “Arial”) είναι εγκατεστημένο στο μηχάνημα που εκτελεί τον κώδικα.
- **Απόκλιση άδειας:** Χωρίς έγκυρη άδεια, το παραγόμενο XPS μπορεί να περιέχει υδατογράφημα. Εφαρμόστε την άδειά σας νωρίς στην εφαρμογή για να το αποφύγετε.
