# Μετατροπή Aspose Page – Πώς να Μετατρέψετε XPS σε PDF σε Java

Η μετατροπή αρχείων XPS σε PDF είναι μια συχνή απαίτηση όταν χρειάζεστε μια μορφή εγγράφου που μπορεί να προβληθεί παντού. Σε αυτό το σεμινάριο θα ανακαλύψετε **aspose page conversion** σε δράση, χρησιμοποιώντας το Aspose.Page for Java για **render PDF from XPS** με λίγες μόνο γραμμές κώδικα. Είτε δημιουργείτε σύστημα διαχείρισης εγγράφων, παράγετε εκτυπώσιμες αναφορές ή αρχειοθετείτε αρχεία, αυτός ο οδηγός σας καθοδηγεί σε όλη τη διαδικασία, από τις προαπαιτήσεις μέχρι την αντιμετώπιση προβλημάτων.

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη διαχειρίζεται τη μετατροπή;** Aspose.Page for Java.  
- **Πόσο διαρκεί η υλοποίηση;** Περίπου 10‑15 λεπτά για μια βασική ρύθμιση.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται άδεια για παραγωγή.  
- **Μπορώ να μετατρέψω μόνο επιλεγμένες σελίδες;** Ναι – χρησιμοποιήστε την επιλογή *specify PDF page numbers*.  
- **Η μετατροπή είναι χωρίς απώλειες;** Η βιβλιοθήκη διατηρεί τα διανυσματικά γραφικά και την πιστότητα του κειμένου.  

## Τι είναι η Μετατροπή Aspose Page από XPS σε PDF;
Η μετατροπή Aspose Page μετατρέπει έγγραφα XPS (XML Paper Specification) — τη σταθερή μορφή διάταξης της Microsoft — σε PDF, το de‑facto πρότυπο για κοινή χρήση και εκτύπωση. Η μετατροπή διατηρεί τη διάταξη, τις γραμματοσειρές και τα διανυσματικά γραφικά, εξασφαλίζοντας ότι το παραγόμενο PDF φαίνεται ακριβώς όπως το αρχικό XPS.

## Γιατί να Χρησιμοποιήσετε το Aspose.Page for Java για Render PDF από XPS;
- **Υψηλή πιστότητα** – διατηρεί τα διανυσματικά γραφικά, τις γραμματοσειρές και τη διάταξη.  
- **Λεπτομερής έλεγχος** – μπορείτε να ορίσετε την ποιότητα εικόνας, τη συμπίεση και ακόμη να επιλέξετε συγκεκριμένες σελίδες.  
- **Χωρίς εξωτερικές εξαρτήσεις** – καθαρή Java, λειτουργεί σε οποιαδήποτε πλατφόρμα που υποστηρίζει JDK.  
- **Ενσωματωμένη επιλογή σελίδων** – εύκολα **specify PDF page numbers** για μείωση του μεγέθους του αρχείου.

## Προαπαιτήσεις
Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε:

- **Java Development Kit (JDK)** – οποιαδήποτε πρόσφατη έκδοση (συνίσταται 8+).  
- **Aspose.Page for Java** – κατεβάστε τη βιβλιοθήκη από την επίσημη [documentation](https://reference.aspose.com/page/java/).  
- Ένα αρχείο XPS που θέλετε να μετατρέψετε.

## Εισαγωγή Πακέτων
Στο έργο Java σας, εισάγετε τα απαιτούμενα πακέτα για εργασία με το Aspose.Page for Java. Αυτό το βήμα είναι κρίσιμο για την πρόσβαση στις λειτουργίες που απαιτούνται για τη μετατροπή XPS σε PDF.

```java
import com.aspose.xps.XpsDocument;
import java.io.FileOutputStream;
```

## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Ορισμός Καταλόγου Εγγράφου
Ορίστε τη διαδρομή προς το φάκελο που περιέχει το πηγαίο αρχείο XPS.

```java
String dataDir = "Your Document Directory";
```

### Βήμα 2: Αρχικοποίηση Ροής Εξόδου PDF
Δημιουργήστε ένα `FileOutputStream` που θα λάβει το παραγόμενο PDF.

```java
FileOutputStream pdfStream = new FileOutputStream(dataDir + "XPStoPDF.pdf");
```

### Βήμα 3: Φόρτωση Εγγράφου XPS
Φορτώστε το αρχείο XPS χρησιμοποιώντας το Aspose.Page.

```java
XpsDocument document = new XpsDocument(dataDir + "input.xps");
```

### Βήμα 4: Αρχικοποίηση Επιλογών Αποθήκευσης PDF  
Δημιουργήστε επιλογές για τη μετατροπή. Εδώ μπορείτε **specify PDF page numbers**, να ρυθμίσετε την ποιότητα εικόνας και να ορίσετε συμπίεση.

```java
PdfSaveOptions options = new PdfSaveOptions();
options.setJpegQualityLevel(100);
options.setImageCompression(PdfImageCompression.Jpeg);
options.setTextCompression(PdfTextCompression.Flate);
options.setPageNumbers(new int[] { 1, 2, 6 });
```

### Βήμα 5: Δημιουργία Συσκευής Απόδοσης PDF  
Ρυθμίστε μια συσκευή απόδοσης που θα γράψει το PDF αποτέλεσμα.

```java
PdfDevice device = new PdfDevice(pdfStream);
```

### Βήμα 6: Αποθήκευση Εγγράφου  
Τέλος, αποθηκεύστε το έγγραφο XPS ως PDF χρησιμοποιώντας τις επιλογές και τη συσκευή που διαμορφώσατε.

```java
document.save(device, options);
```

Επαναλάβετε αυτά τα βήματα, προσαρμόζοντας τις διαδρομές αρχείων και τις επιλογές ανάλογα με το συγκεκριμένο σενάριό σας.

## Πώς να Καθορίσετε Αριθμούς Σελίδων PDF Κατά τη Μετατροπή XPS
Εάν χρειάζεστε μόνο ένα υποσύνολο σελίδων από το αρχικό XPS, γεμίστε τον πίνακα `setPageNumbers` με τους επιθυμητούς δείκτες σελίδων (ξεκινώντας από 1). Αυτό βοηθά στη μείωση του μεγέθους του αρχείου και του χρόνου επεξεργασίας.

## Συνηθισμένα Προβλήματα & Αντιμετώπιση
- **FileNotFoundException** – Επαληθεύστε ότι το `dataDir` δείχνει στο σωστό φάκελο και ότι το όνομα του αρχείου XPS ταιριάζει.  
- **LicenseException** – Απαιτείται έγκυρη άδεια Aspose.Page για παραγωγική χρήση· διαφορετικά, η βιβλιοθήκη λειτουργεί σε λειτουργία αξιολόγησης με υδατογράφημα.  
- **Low image quality** – Αυξήστε το `setJpegQualityLevel` ή μεταβείτε σε συμπίεση χωρίς απώλειες αν χρειάζεται.
