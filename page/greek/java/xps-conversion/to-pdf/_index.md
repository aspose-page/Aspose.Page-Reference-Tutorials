# Πώς να Μετατρέψετε XPS σε PDF σε Java

## Πώς να Μετατρέψετε XPS σε PDF σε Java
Στον χώρο της ανάπτυξης Java, **πώς να μετατρέψετε XPS σε PDF** είναι μια συχνή ερώτηση. Είτε δημιουργείτε σύστημα διαχείρισης εγγράφων είτε χρειάζεστε να δημιουργήσετε εκτυπώσιμες αναφορές, η αξιόπιστη μετατροπή αρχείων XPS μπορεί να είναι καθοριστική. Ευτυχώς, το Aspose.Page for Java το καθιστά απλό να **δημιουργήσετε PDF από XPS** με λίγες μόνο γραμμές κώδικα.

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη διαχειρίζεται τη μετατροπή;** Aspose.Page for Java.  
- **Πόσο χρόνο παίρνει η υλοποίηση;** Περίπου 10‑15 λεπτά για μια βασική ρύθμιση.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται άδεια για παραγωγική χρήση.  
- **Μπορώ να μετατρέψω μόνο επιλεγμένες σελίδες;** Ναι – χρησιμοποιήστε την επιλογή *specify PDF page numbers*.  
- **Είναι η μετατροπή χωρίς απώλειες;** Η βιβλιοθήκη διατηρεί την ακρίβεια των διανυσματικών γραφικών και του κειμένου.

## Τι είναι η μετατροπή XPS σε PDF;
Το XPS (XML Paper Specification) είναι το σταθερό μορφότυπο εγγράφων της Microsoft. Η μετατροπή του σε PDF σας επιτρέπει να μοιράζεστε, να εκτυπώνετε ή να αρχειοθετείτε έγγραφα χρησιμοποιώντας το παγκοσμίως αποδεκτό πρότυπο PDF.

## Γιατί να χρησιμοποιήσετε το Aspose.Page for Java για τη δημιουργία PDF από XPS;
- **Υψηλή πιστότητα** – διατηρεί διανυσματικά γραφικά, γραμματοσειρές και διάταξη.  
- **Λεπτομερής έλεγχος** – μπορείτε να ορίσετε την ποιότητα εικόνας, τη συμπίεση και ακόμη να επιλέξετε συγκεκριμένες σελίδες.  
- **Χωρίς εξωτερικές εξαρτήσεις** – καθαρά Java, λειτουργεί σε οποιαδήποτε πλατφόρμα υποστηρίζει JDK.

## Προαπαιτούμενα
Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε:

- **Java Development Kit (JDK)** – οποιαδήποτε πρόσφατη έκδοση (συνιστάται 8+).  
- **Aspose.Page for Java** – κατεβάστε τη βιβλιοθήκη από την επίσημη [documentation](https://reference.aspose.com/page/java/).  
- Ένα αρχείο XPS που θέλετε να μετατρέψετε.

## Εισαγωγή Πακέτων
Στο έργο Java σας, εισάγετε τα απαιτούμενα πακέτα για εργασία με το Aspose.Page for Java. Αυτό το βήμα είναι κρίσιμο για την πρόσβαση στις λειτουργίες που απαιτούνται για τη μετατροπή XPS σε PDF.

```java
import com.aspose.xps.XpsDocument;
import java.io.FileOutputStream;
```

## Οδηγός Βήμα‑βήμα

### Βήμα 1: Ορισμός Καταλόγου Εγγράφου
Ορίστε τη διαδρομή προς το φάκελο που περιέχει το πηγαίο αρχείο XPS.

```java
String dataDir = "Your Document Directory";
```

### Βήμα 2: Αρχικοποίηση Ροής Εξόδου PDF
Δημιουργήστε ένα `FileOutputStream` που θα λάβει το παραγόμενο PDF.

```java
FileOutputStream pdfStream = new FileOutputStream(dataDir + "XPStoPDF.pdf");
```

### Βήμα 3: Φόρτωση Εγγράφου XPS
Φορτώστε το αρχείο XPS χρησιμοποιώντας το Aspose.Page.

```java
XpsDocument document = new XpsDocument(dataDir + "input.xps");
```

### Βήμα 4: Αρχικοποίηση Επιλογών Αποθήκευσης PDF  
Δημιουργήστε επιλογές για τη μετατροπή. Εδώ μπορείτε να **specify PDF page numbers**, να ρυθμίσετε την ποιότητα εικόνας και να ορίσετε συμπίεση.

```java
PdfSaveOptions options = new PdfSaveOptions();
options.setJpegQualityLevel(100);
options.setImageCompression(PdfImageCompression.Jpeg);
options.setTextCompression(PdfTextCompression.Flate);
options.setPageNumbers(new int[] { 1, 2, 6 });
```

### Βήμα 5: Δημιουργία Συσκευής Απόδοσης PDF  
Ρυθμίστε μια συσκευή απόδοσης που θα γράφει το PDF.

```java
PdfDevice device = new PdfDevice(pdfStream);
```

### Βήμα 6: Αποθήκευση Εγγράφου  
Τέλος, αποθηκεύστε το έγγραφο XPS ως PDF χρησιμοποιώντας τις επιλογές και τη συσκευή που διαμορφώσατε.

```java
document.save(device, options);
```

Επαναλάβετε αυτά τα βήματα, προσαρμόζοντας τις διαδρομές αρχείων και τις επιλογές ανάλογα με το συγκεκριμένο σενάριό σας.

## Πώς να Καθορίσετε Αριθμούς Σελίδων PDF Κατά τη Μετατροπή XPS
Εάν χρειάζεστε μόνο ένα υποσύνολο σελίδων από το αρχικό XPS, γεμίστε τον πίνακα `setPageNumbers` με τους επιθυμητούς δείκτες σελίδων (ξεκινώντας από 1). Αυτό βοηθά στη μείωση του μεγέθους του αρχείου και του χρόνου επεξεργασίας.

## Συχνά Προβλήματα & Αντιμετώπιση
- **FileNotFoundException** – Επαληθεύστε ότι το `dataDir` δείχνει στον σωστό φάκελο και ότι το όνομα του αρχείου XPS ταιριάζει.  
- **LicenseException** – Απαιτείται έγκυρη άδεια Aspose.Page για παραγωγική χρήση· διαφορετικά, η βιβλιοθήκη λειτουργεί σε λειτουργία αξιολόγησης με υδατογράφημα.  
- **Χαμηλή ποιότητα εικόνας** – Αυξήστε το `setJpegQualityLevel` ή μεταβείτε σε συμπίεση χωρίς απώλειες εάν χρειάζεται.
