# Δημιουργία XPS Document Java – Προσθήκη Ορθογωνίου

## Εισαγωγή
Σε αυτό το ολοκληρωμένο tutorial θα **δημιουργήσετε αρχεία XPS document Java** και θα μάθετε πώς να προσθέτετε ορθογώνια χρησιμοποιώντας τη βιβλιοθήκη Aspose.Page. Είτε δημιουργείτε αναφορές, τιμολόγια ή προσαρμοσμένα γραφικά, η εξοικείωση με τη δημιουργία ορθογωνίων σας δίνει ακριβή έλεγχο πάνω στη διάταξη του XPS. Θα περάσουμε βήμα-βήμα, θα εξηγήσουμε το «γιατί» πίσω από κάθε γραμμή κώδικα και θα σας δείξουμε πώς να προσαρμόζετε χρώματα και στυλ γραμμής για επαγγελματικά αποτελέσματα.

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη συνιστάται;** Aspose.Page for Java  
- **Πόσο χρόνο διαρκεί η υλοποίηση;** Περίπου 10 λεπτά για ένα βασικό ορθογώνιο  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για ανάπτυξη· απαιτείται εμπορική άδεια για παραγωγή  
- **Ποια έκδοση Java υποστηρίζεται;** Java 8 και νεότερες  
- **Μπορώ να προσθέσω πολλαπλά σχήματα;** Ναι – επαναλάβετε τα βήματα για κάθε σχήμα

## Προαπαιτούμενα
Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε:

- Βασικές γνώσεις της γλώσσας προγραμματισμού Java.  
- Εγκατεστημένη τη βιβλιοθήκη Aspose.Page. Αν όχι, μπορείτε να τη κατεβάσετε από την [τεκμηρίωση Aspose.Page Java](https://reference.aspose.com/page/java/).  
- Ένα λειτουργικό περιβάλλον ανάπτυξης Java (IDE, JDK και Maven/Gradle αν προτιμάτε).

## Εισαγωγή Πακέτων
Για να ξεκινήσετε, εισάγετε τα απαραίτητα πακέτα στο έργο Java σας. Βεβαιωθείτε ότι η βιβλιοθήκη Aspose.Page έχει προστεθεί σωστά στο classpath. Ακολουθεί ένα βασικό παράδειγμα:

```java
import com.aspose.xps.XpsDocument;
import com.aspose.xps.XpsPath;
```

Τώρα, ας αναλύσουμε το παραπάνω παράδειγμα σε πολλαπλά βήματα για την προσθήκη ενός ορθογωνίου σε XPS Java.

## Βήμα 1: Ορισμός Καταλόγου Εγγράφου
```java
// The path to the documents directory.
String dataDir = "Your Document Directory";
```

Αντικαταστήστε το `"Your Document Directory"` με τη διαδρομή του φακέλου όπου θέλετε να αποθηκεύσετε τα αρχεία XPS.

## Βήμα 2: Δημιουργία Νέου XPS Εγγράφου
```java
// Create new XPS Document
XpsDocument doc = new XpsDocument();
```

Αυτή η γραμμή **δημιουργεί** ένα νέο XPS έγγραφο που μπορείτε αργότερα να γεμίσετε με σχήματα, κείμενο ή εικόνες.

## Βήμα 3: Προσθήκη Ορθογωνίου με CMYK Στερεό Χρώμα Περιγράμματος
```java
// CMYK (blue) solid color stroked rectangle in the lower left
XpsPath path = doc.addPath(doc.createPathGeometry("M 20,10 L 220,10 220,100 20,100 Z"));
path.setStroke(doc.createSolidColorBrush(
    doc.createColor(dataDir + "uswebuncoated.icc", 1.0f, 1.000f, 0.000f, 0.000f, 0.000f)));
path.setStrokeThickness(12f);
```

Αυτό το βήμα προσθέτει ένα ορθογώνιο με περίγραμμα σε στερεό χρώμα CMYK. Μπορείτε να αλλάξετε το geometry string (`"M 20,10 L 220,10 220,100 20,100 Z"`) για να τροποποιήσετε το μέγεθος ή τη θέση, και να προσαρμόσετε τις τιμές χρώματος στο `createColor` ώστε να ταιριάζει στο σχέδιό σας.

## Βήμα 4: Αποθήκευση του Τελικού XPS Εγγράφου
```java
// Save resultant XPS document
doc.save(dataDir + "AddRectangle_out.xps");
```

Τέλος, αποθηκεύστε το τροποποιημένο XPS έγγραφο με το προστιθέμενο ορθογώνιο στον κατάλογο που ορίσατε νωρίτερα.

## Συνηθισμένες Περιπτώσεις Χρήσης
- **Κεφαλίδες αναφορών** – Χρησιμοποιήστε ορθογώνια ως φόντο για τίτλους.  
- **Πίνακες τιμολογίων** – Επισημάνετε κελιά ή ενότητες με χρωματιστά περιγράμματα.  
- **Προσαρμοσμένα γραφικά** – Συνδυάστε πολλαπλά ορθογώνια για να δημιουργήσετε σύνθετα σχήματα.

## Συμβουλές Επίλυσης Προβλημάτων
- **Σφάλμα «αρχείο δεν βρέθηκε»:** Επαληθεύστε ότι το `dataDir` δείχνει σε υπάρχον φάκελο και ότι το ICC προφίλ (`uswebuncoated.icc`) υπάρχει.  
- **Λάθος χρώματα:** Βεβαιωθείτε ότι το ICC προφίλ ταιριάζει με το χρωματικό χώρο που θέλετε να χρησιμοποιήσετε (CMYK vs. RGB).  
- **Απρόσμενες διαστάσεις:** Προσαρμόστε το geometry string ώστε να αντανακλά τις σωστές συντεταγμένες.

## Συχνές Ερωτήσεις

**Ε: Λειτουργεί αυτή η προσέγγιση με Java 11 ή νεότερη;**  
Α: Ναι, η βιβλιοθήκη Aspose.Page είναι συμβατή με Java 8 και νεότερες, συμπεριλαμβανομένων των Java 11, 17 κ.ά.

**Ε: Μπορώ να ενσωματώσω εικόνες μέσα στην περιοχή του ορθογωνίου;**

Α: Εφαρμόστε έναν μετασχηματιστικό πίνακα στο path χρησιμοποιώντας `path.setTransform(...)` για περιστροφή ή κλιμάκωση.
