# Ορισμός Χρώματος Ορθογωνίου και Προσθήκη Ορθογωνίου σε Java XPS

## Εισαγωγή
Σε αυτόν τον οδηγό, θα μάθετε πώς να **set rectangle color** και **add a rectangle** σε Java XPS χρησιμοποιώντας τη βιβλιοθήκη Aspose.Page. Είτε χρειάζεστε να σχεδιάσετε απλά σχήματα για μια αναφορά είτε να δημιουργήσετε πολύπλογα διανυσματικά γραφικά, η εξοικείωση με το στυλ των ορθογωνίων σας παρέχει ακριβή έλεγχο στα έγγραφα XPS.  

## Σύντομες Απαντήσεις
- **Ποια βιβλιοθήκη απαιτείται;** Aspose.Page for Java  
- **Μπορώ να αλλάξω το περίγραμμα του ορθογωνίου;** Yes, using `setStroke` and `setStrokeThickness`  
- **Υποστηρίζεται προφίλ χρώματος CMYK;** Απολύτως – μπορείτε να φορτώσετε προφίλ ICC όπως φαίνεται  
- **Χρειάζομαι άδεια για παραγωγή;** Ναι, απαιτείται εμπορική άδεια για χρήση εκτός αξιολόγησης  
- **Πόσο διαρκεί η υλοποίηση;** Συνήθως λιγότερο από 10 λεπτά για ένα βασικό ορθογώνιο

## Τι είναι **set rectangle color**;
Ο ορισμός του χρώματος ενός ορθογωνίου σημαίνει τον καθορισμό του χρώματος γεμίσματος ή περιγράμματος που θα αποδώσει το σχήμα στο τελικό έγγραφο XPS. Με το Aspose.Page μπορείτε να χρησιμοποιήσετε RGB, CMYK ή ακόμη και προσαρμοσμένα προφίλ χρώματος ICC για ακριβή αντιστοίχιση χρώματος.

## Γιατί να χρησιμοποιήσετε το Aspose.Page για **add rectangle java** και **change rectangle stroke**;
- **Full‑featured API** – υποστηρίζει τόσο στερεά χρώματα όσο και διαβαθμίσεις.  
- **Cross‑platform** – λειτουργεί σε οποιοδήποτε περιβάλλον Java χωρίς εγγενείς εξαρτήσεις.  
- **Rich geometry support** – δημιουργήστε πολύπλοκες διαδρομές, εφαρμόστε μετασχηματισμούς και διαχειριστείτε εύκολα το πάχος του περιγράμματος.  

## Προαπαιτούμενα
Πριν βουτήξουμε στον κώδικα, βεβαιωθείτε ότι έχετε:

- Βασικές γνώσεις προγραμματισμού Java.  
- Εγκατεστημένη τη βιβλιοθήκη Aspose.Page. Αν όχι, κατεβάστε την από την [Τεκμηρίωση Aspose.Page Java](https://reference.aspose.com/page/java/).  
- Ένα λειτουργικό περιβάλλον ανάπτυξης Java (IDE, JDK, κ.λπ.).  

## Εισαγωγή Πακέτων
Για να ξεκινήσετε, εισάγετε τα απαραίτητα πακέτα στο έργο Java σας. Βεβαιωθείτε ότι η βιβλιοθήκη Aspose.Page έχει προστεθεί σωστά στο classpath. Ακολουθεί ένα βασικό παράδειγμα:
```java
import com.aspose.xps.XpsDocument;
import com.aspose.xps.XpsPath;
```

Τώρα, ας αναλύσουμε το παρεχόμενο παράδειγμα σε πολλαπλά βήματα για **add a rectangle** και **set its color** σε Java XPS.

## Βήμα 1: Ορισμός Καταλόγου Εγγράφου
```java
// The path to the documents directory.
String dataDir = "Your Document Directory";
```
Αντικαταστήστε το `"Your Document Directory"` με την απόλυτη διαδρομή όπου θέλετε να διαβάζετε/γράφετε αρχεία XPS.

## Βήμα 2: Δημιουργία Νέου Εγγράφου XPS
```java
// Create new XPS Document
XpsDocument doc = new XpsDocument();
```
Αυτή η γραμμή αρχικοποιεί ένα νέο έγγραφο XPS που θα περιέχει τα σχήματά μας.

## Βήμα 3: Προσθήκη Ορθογωνίου με Στερεό Χρώμα CMYK και Περιγράμματα
```java
// CMYK (blue) solid color stroked rectangle in the lower left
XpsPath path = doc.addPath(doc.createPathGeometry("M 20,10 L 220,10 220,100 20,100 Z"));
path.setStroke(doc.createSolidColorBrush(
    doc.createColor(dataDir + "uswebuncoated.icc", 1.0f, 1.000f, 0.000f, 0.000f, 0.000f)));
path.setStrokeThickness(12f);
```
Αυτό το βήμα προσθέτει ένα **stroked rectangle** με στερεό χρώμα CMYK. Η μέθοδος `setStroke` ορίζει το χρώμα του περιγράμματος του ορθογωνίου, ενώ η `setStrokeThickness` ελέγχει το πάχος της γραμμής — ιδανικό για **changing rectangle stroke**.

### Pro tip:
Αν προτιμάτε χρώμα RGB, αντικαταστήστε την κλήση `createColor` με `doc.createColor(0, 0, 255)` για στερεό μπλε γέμισμα.

## Βήμα 4: Αποθήκευση του Παραγόμενου Εγγράφου XPS
```java
// Save resultant XPS document
doc.save(dataDir + "AddRectangle_out.xps");
```
Τέλος, αποθηκεύστε το έγγραφο XPS στο δίσκο. Το αρχείο `AddRectangle_out.xps` περιέχει πλέον ένα ορθογώνιο του οποίου το χρώμα και το περίγραμμα έχετε ορίσει.

## Συχνά Προβλήματα και Λύσεις
| Πρόβλημα | Αιτία | Λύση |
|----------|-------|------|
| **Το ορθογώνιο δεν είναι ορατό** | Λανθασμένη γεωμετρία διαδρομής ή πάχος περιγράμματος ορισμένο σε 0 | Επαληθεύστε τη συμβολοσειρά διαδρομής (`"M 20,10 L 220,10 220,100 20,100 Z"`) και βεβαιωθείτε ότι το `setStrokeThickness` είναι μεγαλύτερο από 0. |
| **Λάθος χρώμα** | Χρήση προφίλ ICC που δεν ταιριάζει με τον επιθυμητό χώρο χρώματος | Χρησιμοποιήστε `doc.createColor(r, g, b)` για RGB ή ελέγξτε ξανά τη διαδρομή του αρχείου ICC. |
| **Το αρχείο δεν αποθηκεύεται** | `dataDir` δείχνει σε μη υπάρχον φάκελο ή δεν έχει άδεια εγγραφής | Δημιουργήστε το φάκελο εκ των προτέρων ή επιλέξτε μια τοποθεσία με δικαιώματα εγγραφής. |

## Συχνές Ερωτήσεις

**Q:** *Μπορώ να προσθέσω πολλαπλά ορθογώνια σε ένα μόνο έγγραφο XPS;*  
A: Ναι. Απλώς επαναλάβετε τα βήματα δημιουργίας γεωμετρίας και στυλ για κάθε ορθογώνιο που χρειάζεστε.

**Q:** *Πώς μπορώ να αλλάξω το χρώμα γεμίσματος του ορθογωνίου αντί για το περίγραμμα;*  
A: Χρησιμοποιήστε `path.setFill(...)` με ένα στερεό χρώμα πινέλου, παρόμοια με τη χρήση του `setStroke`.

**Q:** *Είναι το Aspose.Page κατάλληλο για σύνθετες επεμβάσεις σε έγγραφα XPS;*

**Q:** *Πού μπορώ να βρω πρόσθετα παραδείγματα και υποστήριξη από την κοινότητα;*

**Q:** *Μπορώ να δοκιμάσω το Aspose.Page πριν το αγοράσω;*  
A: Ναι, μπορείτε να εξερευνήσετε μια [δωρεάν δοκιμή](https://releases.aspose.com/) για να αξιολογήσετε τις δυνατότητες του API.
