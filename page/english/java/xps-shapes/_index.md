## What is a radial gradient ellipse?
A **radial gradient ellipse** combines the geometric precision of an ellipse with a gradient that radiates from a central point outward. This effect is perfect for highlighting sections, creating button‑like visuals, or adding subtle branding elements to your XPS pages.

## Tips & Best Practices
- **Use consistent units:** Aspose.Page works with points; 1 pt = 1/72 in.

## Frequently Asked Questions

**Q: Can I change the gradient colors after the shape is created?**  
A: Yes, you can modify the `GradientStop` collection and re‑apply it to the ellipse’s stroke.

**Q: Is it possible to add a shadow effect to the rectangle?**  
A: While XPS does not have a native shadow, you can simulate one by drawing a slightly offset, semi‑transparent rectangle behind the original.

**Q: Do these methods work with PDF conversion?**  
A: Absolutely. Aspose.Page lets you convert the XPS document to PDF while preserving all vector shapes.

**Q: How do I ensure the shapes are DPI‑independent?**  
A: Because XPS uses vector graphics, shapes scale automatically; just avoid raster images inside the shape.

**Q: What if I need to draw more complex polygons?**  
A: Use the `Path` class to define custom geometry; the same gradient and stroke settings apply.
