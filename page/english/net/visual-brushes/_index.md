## What is a Visual Brush?

A Visual Brush captures a visual element—such as shapes, text, or images—and lets you reuse it as a fill for other drawing objects. This approach reduces duplication and ensures consistent styling across your document.

## Why use a Grid Visual Brush?

A Grid Visual Brush creates a tiled grid pattern that can serve as a background, a guide, or a decorative element. It’s especially useful when you need:

- **Consistent alignment** across multiple pages.  
- **Lightweight styling** without embedding large images.  
- **Dynamic scaling** that adapts to different page sizes.

## Frequently Asked Questions

**Q:** *Can I combine multiple visual brushes in a single document?*  
**A:** Yes, you can layer different Visual Brushes to create complex patterns and effects.

**Q:** *Is it possible to change the grid size at runtime?*  
**A:** Absolutely. Adjust the `CellWidth` and `CellHeight` properties before rendering to modify the grid dynamically.

**Q:** *Do Visual Brushes affect file size?*  
**A:** Since they are vector‑based, they add minimal overhead compared to raster images.

**Q:** *Can I use a Visual Brush with PDF output?*  
**A:** Yes, Aspose.Page supports PDF, XPS, and several other formats while preserving Visual Brush definitions.

**Q:** *What troubleshooting steps should I take if the brush doesn’t appear?*  
**A:** Verify that the brush is assigned to a drawable object and that the document’s rendering settings are correctly configured.
