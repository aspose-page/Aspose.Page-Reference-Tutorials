# สร้างเอกสาร XPS – เพิ่ม Glyph Clone และเปลี่ยนสีด้วย Aspose.Page สำหรับ .NET

## บทนำ

ยินดีต้อนรับสู่คู่มือแบบขั้นตอนต่อขั้นตอนนี้ที่จะแสดงให้คุณ **สร้างไฟล์เอกสาร XPS** , คัดลอก glyphs, และเปลี่ยนสีของพวกมันโดยใช้ Aspose.Page สำหรับ .NET ไม่ว่าคุณจะกำลังสร้างรายงานแบบไดนามิก, ใบแจ้งหนี้, หรือกราฟิกแบบกำหนดเอง การเชี่ยวชาญเทคนิคเหล่านี้จะทำให้คุณผลิตผลลัพธ์ XPS ที่มีภาพสวยงามโดยไม่ต้องออกจากระบบนิเวศ .NET

## คำตอบสั้น
- **เป้าหมายหลักคืออะไร?** สร้างเอกสาร XPS, เพิ่ม glyph clone, และเปลี่ยนสีของพวกมัน  
- **ใช้ไลบรารีใด?** Aspose.Page สำหรับ .NET  
- **ต้องการไลเซนส์หรือไม่?** มีไลเซนส์ชั่วคราวสำหรับการประเมิน; จำเป็นต้องมีไลเซนส์เต็มสำหรับการใช้งานจริง  
- **รองรับเวอร์ชัน .NET ใด?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6+  
- **บันทึกผลลัพธ์อย่างไร?** ใช้เมธอด `Save` เพื่อเขียนไฟล์ XPS ลงดิสก์

## ข้อกำหนดเบื้องต้น

ก่อนเริ่มทำตามบทเรียน โปรดตรวจสอบว่าคุณมีสิ่งต่อไปนี้:

- ความเข้าใจพื้นฐานเกี่ยวกับภาษาโปรแกรม C#  
- Visual Studio หรือสภาพแวดล้อมการพัฒนา C# ที่คุณชื่นชอบติดตั้งแล้ว  
- ไลบรารี Aspose.Page สำหรับ .NET คุณสามารถดาวน์โหลดได้จาก [ที่นี่](https://releases.aspose.com/page/net/)  
- ความคุ้นเคยกับรูปแบบเอกสาร XPS

## นำเข้า Namespaces

เพื่อเริ่มต้น ให้เพิ่ม namespaces ที่จำเป็นในโปรเจกต์ C# ของคุณ:

```csharp
using Aspose.Page.XPS;
using Aspose.Page.XPS.XpsModel;
using System.Drawing;
```

## วิธีสร้างเอกสาร XPS และเพิ่ม Glyph Clone

### ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสารของคุณ

เริ่มต้นโดยตั้งค่าไดเรกทอรีที่ไฟล์เอกสารของคุณจะถูกจัดเก็บ:

```csharp
string dataDir = "Your Document Directory";
```

### ขั้นตอนที่ 2: สร้างเอกสาร XPS แรก

ตอนนี้มาสร้างเอกสาร XPS แรก:

```csharp
XpsDocument doc1 = new XpsDocument();
```

### ขั้นตอนที่ 3: เพิ่ม Glyphs ลงในเอกสารแรก

เพิ่ม glyphs ลงในเอกสารแรกโดยใช้พารามิเตอร์ที่ระบุ:

```csharp
XpsGlyphs glyphs = doc1.AddGlyphs("Times New Roman", 200, FontStyle.Bold, 50, 250, "Test");
```

### ขั้นตอนที่ 4: เติม Glyphs ด้วย Solid Color Brush

เติม glyphs ในเอกสารแรกด้วย **solid color brush** ตัวอย่างเช่นสีเขียว:

```csharp
glyphs.Fill = doc1.CreateSolidColorBrush(Color.Green);
```

### ขั้นตอนที่ 5: สร้างเอกสาร XPS ที่สอง

สร้างเอกสาร XPS ที่สอง:

```csharp
XpsDocument doc2 = new XpsDocument();
```

### ขั้นตอนที่ 6: วิธีเพิ่ม Glyph Clone ไปยังเอกสารอื่น

คัดลอก glyphs จากเอกสารแรกและเพิ่มลงในเอกสารที่สอง:

```csharp
glyphs = doc2.Add(glyphs.Clone());
```

### ขั้นตอนที่ 7: วิธีเปลี่ยนสีของ Glyph ที่คัดลอกแล้ว

เปลี่ยนสีของ glyph ที่คัดลอกในเอกสารที่สอง ตัวอย่างเช่นเป็นสีแดง สิ่งนี้แสดง **วิธีเปลี่ยนสี** ของ glyph หลังจากการคัดลอก:

```csharp
((XpsSolidColorBrush)glyphs.Fill).Color = doc2.CreateColor(Color.Red);
```

### ขั้นตอนที่ 8: บันทึกไฟล์ XPS – เอกสารแรก

บันทึกเอกสาร XPS แรกลงในโฟลเดอร์ที่คุณกำหนดไว้ก่อนหน้า:

```csharp
doc1.Save(dataDir + "out1.xps");
```

### ขั้นตอนที่ 9: บันทึกไฟล์ XPS – เอกสารที่สอง

บันทึกเอกสาร XPS ที่สอง:

```csharp
doc2.Save(dataDir + "out2.xps");
```

ขอแสดงความยินดี! คุณได้ **สร้างไฟล์เอกสาร XPS** สำเร็จ, เพิ่ม glyph clone, และเปลี่ยนสีของพวกมันโดยใช้ Aspose.Page สำหรับ .NET

## ทำไมต้องใช้ Glyph Cloning และการเปลี่ยนสี?

- **การนำกลับมาใช้ใหม่:** คัดลอก glyph ที่มีอยู่เพื่อใช้รูปเวกเตอร์ที่ซับซ้อนโดยไม่ต้องกำหนดใหม่  
- **ประสิทธิภาพ:** ลดเวลาในการประมวลผลเมื่อเทียบกับการสร้าง glyph จากศูนย์  
- **ความยืดหยุ่นในการออกแบบ:** ใช้ `SolidColorBrush` ที่แตกต่างกันกับ glyph เดียวกันเพื่อให้ได้ธีมภาพที่หลากหลาย  
- **การแก้ไขข้ามเอกสาร:** คัดลอก glyph ข้ามหลายไฟล์ XPS ทำให้สามารถรักษาแบรนด์เดียวกันในรายงานต่าง ๆ ได้

## ปัญหาทั่วไปและการแก้ไขปัญหา

| ปัญหา | วิธีแก้ |
|-------|----------|
| **Clone คืนค่า null** | ตรวจสอบให้แน่ใจว่า glyph แหล่ง (`glyphs`) ถูกเพิ่มลงในเอกสารแรกก่อนทำการคัดลอก |
| **สีไม่เปลี่ยน** | แคสต์ `glyphs.Fill` เป็น `XpsSolidColorBrush` ก่อนตั้งค่า property `Color` (ตามที่แสดงในขั้นตอน 7) |
| **ไฟล์ไม่ถูกบันทึก** | ตรวจสอบว่า `dataDir` ชี้ไปยังโฟลเดอร์ที่มีสิทธิ์เขียนและคุณมีสิทธิ์เข้าถึงระบบไฟล์ที่เหมาะสม |
| **ขนาด glyph ไม่คาดคิด** | ปรับพารามิเตอร์ขนาดฟอนต์ (อาร์กิวเมนต์ที่สองใน `AddGlyphs`) เพื่อสเกล glyph ให้เหมาะสม |

## คำถามที่พบบ่อย

**Q1: ฉันสามารถใช้ Aspose.Page สำหรับ .NET กับรูปแบบเอกสารอื่นได้หรือไม่?**  
A1: Aspose.Page สำหรับ .NET ถูกออกแบบมาเฉพาะสำหรับการทำงานกับเอกสาร XPS หากต้องการจัดการรูปแบบอื่น คุณอาจต้องสำรวจไลบรารี Aspose อื่นที่เหมาะกับรูปแบบนั้น

**Q2: มีไลเซนส์ชั่วคราวสำหรับ Aspose.Page สำหรับ .NET หรือไม่?**  
A2: มี คุณสามารถรับไลเซนส์ชั่วคราวสำหรับการทดสอบได้ เยี่ยมชม [ที่นี่](https://purchase.aspose.com/temporary-license/) เพื่อดูรายละเอียดเพิ่มเติม

**Q3: ฉันจะขอรับการสนับสนุนหรือขอความช่วยเหลือเมื่อเจอปัญหาได้อย่างไร?**  
A3: สามารถเยี่ยมชม [ฟอรั่ม Aspose.Page](https://forum.aspose.com/c/page/39) เพื่อเชื่อมต่อกับชุมชนและขอความช่วยเหลือ

**Q4: มีข้อจำกัดใดบ้างในรุ่นทดลองฟรี?**  
A4: รุ่นทดลองฟรีมีข้อจำกัดบางประการ แนะนำให้ตรวจสอบเอกสารเพื่อดูรายละเอียดก่อนใช้งาน

**Q5: ฉันจะหาเอกสารอ้างอิงที่ครอบคลุมสำหรับ Aspose.Page สำหรับ .NET ได้จากที่ไหน?**  
A5: คุณสามารถอ้างอิงเอกสารได้ที่ [นี่](https://reference.aspose.com/page/net/) เพื่อรับข้อมูลและตัวอย่างอย่างละเอียด

**Q6: ฉันจะเปลี่ยนสีเส้นขอบ (stroke) ของ glyph แทนการเติมสีได้อย่างไร?**  
A6: ใช้ `glyphs.Stroke = doc.CreateSolidColorBrush(Color.Blue);` เพื่อกำหนด brush สำหรับเส้นขอบ

**Q7: ฉันสามารถเพิ่มหลาย glyph clone ลงในเอกสารเดียวกันได้หรือไม่?**  
A7: ได้ คุณสามารถเรียก `doc2.Add(glyphs.Clone());` ซ้ำ ๆ และปรับตำแหน่งตามต้องการ
