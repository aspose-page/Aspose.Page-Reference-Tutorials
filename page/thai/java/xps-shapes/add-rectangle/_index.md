# สร้างเอกสาร XPS ด้วย Java – เพิ่มสี่เหลี่ยม

## บทนำ
ในบทแนะนำที่ครอบคลุมนี้ คุณจะ **สร้างไฟล์เอกสาร XPS ด้วย Java** และเรียนรู้วิธีเพิ่มสี่เหลี่ยมโดยใช้ไลบรารี Aspose.Page ไม่ว่าคุณจะสร้างรายงาน ใบแจ้งหนี้ หรือกราฟิกแบบกำหนดเอง การเชี่ยวชาญการสร้างสี่เหลี่ยมจะให้การควบคุมที่แม่นยำต่อการจัดวาง XPS เราจะเดินผ่านแต่ละขั้นตอน อธิบายเหตุผลของแต่ละบรรทัดโค้ด และแสดงวิธีปรับแต่งสีและเส้นขอบเพื่อผลลัพธ์ระดับมืออาชีพ

## คำตอบสั้น
- **ไลบรารีที่แนะนำคืออะไร?** Aspose.Page for Java  
- **ใช้เวลานานเท่าไหร่ในการทำงานนี้?** ประมาณ 10 นาทีสำหรับสี่เหลี่ยมพื้นฐาน  
- **ต้องมีลิขสิทธิ์หรือไม่?** ทดลองใช้ฟรีได้สำหรับการพัฒนา; ต้องมีลิขสิทธิ์เชิงพาณิชย์สำหรับการใช้งานจริง  
- **รองรับเวอร์ชัน Java ใด?** Java 8 ขึ้นไป  
- **สามารถเพิ่มหลายรูปทรงได้หรือไม่?** ได้ – ทำซ้ำขั้นตอนสำหรับแต่ละรูปทรง

## ข้อกำหนดเบื้องต้น
ก่อนที่เราจะดำเนินการต่อ โปรดตรวจสอบว่าคุณมี:

- ความรู้พื้นฐานเกี่ยวกับภาษาโปรแกรม Java  
- ไลบรารี Aspose.Page ติดตั้งแล้ว หากยังไม่มี คุณสามารถดาวน์โหลดได้จาก [เอกสาร Aspose.Page Java](https://reference.aspose.com/page/java/)  
- สภาพแวดล้อมการพัฒนา Java ที่ทำงานได้ (IDE, JDK, และ Maven/Gradle หากต้องการ)

## นำเข้าแพ็กเกจ
เพื่อเริ่มต้น ให้นำเข้าแพ็กเกจที่จำเป็นเข้าสู่โปรเจกต์ Java ของคุณ ตรวจสอบให้แน่ใจว่าไลบรารี Aspose.Page ถูกเพิ่มลงใน classpath อย่างถูกต้อง ตัวอย่างพื้นฐานมีดังนี้:

```java
import com.aspose.xps.XpsDocument;
import com.aspose.xps.XpsPath;
```

ตอนนี้ เราจะวิเคราะห์ตัวอย่างที่ให้ไว้เป็นหลายขั้นตอนเพื่อเพิ่มสี่เหลี่ยมใน XPS ด้วย Java

## ขั้นตอนที่ 1: ตั้งค่าไดเรกทอรีเอกสาร
```java
// The path to the documents directory.
String dataDir = "Your Document Directory";
```

แทนที่ `"Your Document Directory"` ด้วยเส้นทางไปยังโฟลเดอร์ที่คุณต้องการเก็บไฟล์ XPS

## ขั้นตอนที่ 2: สร้างเอกสาร XPS ใหม่
```java
// Create new XPS Document
XpsDocument doc = new XpsDocument();
```

บรรทัดนี้ **สร้าง** เอกสาร XPS ใหม่ที่คุณสามารถเติมรูปทรง ข้อความ หรือรูปภาพต่อไปได้

## ขั้นตอนที่ 3: เพิ่มสี่เหลี่ยมที่มีเส้นขอบสี CMYK
```java
// CMYK (blue) solid color stroked rectangle in the lower left
XpsPath path = doc.addPath(doc.createPathGeometry("M 20,10 L 220,10 220,100 20,100 Z"));
path.setStroke(doc.createSolidColorBrush(
    doc.createColor(dataDir + "uswebuncoated.icc", 1.0f, 1.000f, 0.000f, 0.000f, 0.000f)));
path.setStrokeThickness(12f);
```

ขั้นตอนนี้เพิ่มสี่เหลี่ยมที่มีเส้นขอบสี CMYK คุณสามารถเปลี่ยนสตริงเรขาคณิต (`"M 20,10 L 220,10 220,100 20,100 Z"`) เพื่อปรับขนาดหรือตำแหน่ง และปรับค่าของสีใน `createColor` ให้ตรงกับการออกแบบของคุณ

## ขั้นตอนที่ 4: บันทึกเอกสาร XPS ที่ได้
```java
// Save resultant XPS document
doc.save(dataDir + "AddRectangle_out.xps");
```

สุดท้ายบันทึกเอกสาร XPS ที่แก้ไขแล้วพร้อมสี่เหลี่ยมที่เพิ่มเข้าไปไปยังไดเรกทอรีที่คุณระบุไว้ก่อนหน้า

## กรณีการใช้งานทั่วไป
- **หัวเรื่องรายงาน** – ใช้สี่เหลี่ยมเป็นบล็อกพื้นหลังสำหรับหัวข้อ  
- **ตารางใบแจ้งหนี้** – เน้นเซลล์หรือส่วนต่าง ๆ ด้วยเส้นขอบสี  
- **กราฟิกแบบกำหนดเอง** – รวมหลายสี่เหลี่ยมเพื่อสร้างรูปทรงซับซ้อน

## เคล็ดลับการแก้ไขปัญหา
- **ข้อผิดพลาดไฟล์ไม่พบ:** ตรวจสอบว่า `dataDir` ชี้ไปยังโฟลเดอร์ที่มีอยู่และโปรไฟล์ ICC (`uswebuncoated.icc`) มีอยู่จริง  
- **สีไม่ถูกต้อง:** ตรวจสอบว่าโปรไฟล์ ICC ตรงกับพื้นที่สีที่คุณต้องการใช้ (CMYK vs. RGB)  
- **ขนาดไม่ตรงตามคาด:** ปรับสตริงเรขาคณิตให้สะท้อนพิกัดที่ถูกต้อง

## สรุป
ขอแสดงความยินดี! คุณได้เรียนรู้วิธี **สร้างไฟล์เอกสาร XPS ด้วย Java** และเพิ่มสี่เหลี่ยมโดยใช้ Aspose.Page พื้นฐานนี้จะช่วยให้คุณสร้างเอกสาร XPS ที่หลากหลาย ปรับแต่งกราฟิก และรวมรูปทรงเข้ากับกระบวนการทำงานที่ใหญ่ขึ้นได้

## คำถามที่พบบ่อย
### ฉันสามารถเพิ่มหลายสี่เหลี่ยมในเอกสาร XPS เดียวได้หรือไม่?
ได้ คุณสามารถเพิ่มสี่เหลี่ยมได้ตามจำนวนที่ต้องการโดยทำซ้ำขั้นตอนที่อธิบายในบทแนะนำ

### ฉันจะเปลี่ยนสีของสี่เหลี่ยมได้อย่างไร?
แก้ไขค่าของสีในเมธอด `createColor` เพื่อให้ได้สีที่ต้องการ

### Aspose.Page เหมาะกับการจัดการเอกสาร XPS ที่ซับซ้อนหรือไม่?
แน่นอน! Aspose.Page มีชุดคุณสมบัติที่แข็งแกร่งสำหรับการทำงานกับงานเอกสาร XPS หลากหลาย

### ฉันจะหา ตัวอย่างเพิ่มเติมและการสนับสนุนได้จากที่ไหน?
สำรวจ [ฟอรั่ม Aspose.Page](https://forum.aspose.com/c/page/39) เพื่อดูตัวอย่างเพิ่มเติมและขอความช่วยเหลือจากชุมชน

### ฉันสามารถทดลองใช้ Aspose.Page ก่อนซื้อได้หรือไม่?
ได้ คุณสามารถสำรวจ [การทดลองใช้ฟรี](https://releases.aspose.com/) เพื่อสัมผัสความสามารถของ Aspose.Page

## Frequently Asked Questions

**Q: วิธีนี้ทำงานกับ Java 11 หรือใหม่กว่าได้หรือไม่?**  
A: ใช่ ไลบรารี Aspose.Page รองรับ Java 8 ขึ้นไป รวมถึง Java 11, 17 และรุ่นต่อ ๆ ไป

**Q: ฉันสามารถฝังรูปภาพภายในพื้นที่สี่เหลี่ยมได้หรือไม่?**  
A: คุณสามารถเพิ่มองค์ประกอบ `XpsImage` และวางตำแหน่งไว้เหนือหรือภายในสี่เหลี่ยมโดยใช้พิกัดเรขาคณิตเดียวกัน

**Q: จะตั้งค่าสีเติมแทนการวางเส้นขอบอย่างไร?**  
A: เรียก `path.setFill(...)` พร้อมกับแปรงสีทึบที่สร้างโดย `doc.createSolidColorBrush(...)`

**Q: สามารถหมุนสี่เหลี่ยมได้หรือไม่?**  
A: ใช้เมทริกซ์การแปลงกับพาธโดยใช้ `path.setTransform(...)` เพื่อทำการหมุนหรือสเกล

**Q: ต้องการลิขสิทธิ์แบบใดสำหรับการใช้งานในผลิตภัณฑ์?**  
A: จำเป็นต้องมีลิขสิทธิ์ Aspose.Page เชิงพาณิชย์สำหรับการปรับใช้ในสภาพแวดล้อมการผลิต; การทดลองใช้ฟรีจำกัดเฉพาะการประเมินและพัฒนา
