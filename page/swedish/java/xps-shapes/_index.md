## Vanliga frågor

**Q: Behöver jag en betald licens för att använda Aspose.Page för att rita former?**  
A: En gratis provversion finns tillgänglig för utvärdering, men en kommersiell licens krävs för produktionsdistributioner.

**Q: Kan jag kombinera flera former på en enda XPS-sida?**  
A: Ja, du kan lägga till så många former som behövs; lägg bara till varje form i sidans graphics‑samling.

**Q: Är det möjligt att applicera transformationer (rotera, skala) på former?**  
A: Absolut. API‑et tillhandahåller transformationsmatriser som låter dig rotera, skala eller skeva vilken form som helst.

**Q: Vilka Java‑versioner stöds?**  
A: Aspose.Page for Java stöder Java 8 och nyare körmiljöer.

**Q: Hur exporterar jag det slutgiltiga XPS-dokumentet efter att ha lagt till former?**  
A: Anropa `save`‑metoden på `Document`‑objektet och ange önskad filsökväg och format.
