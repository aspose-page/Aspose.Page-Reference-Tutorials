## O que é Aspose.Page para Java?
Aspose.Page para Java é uma API poderosa que permite a criação, edição e conversão de documentos XPS programaticamente. Ela fornece mais de 30 classes para manipular páginas, texto, gráficos e fontes, suportando mais de 50 formatos de entrada e saída. Com esta biblioteca você pode adicionar, remover ou estilizar texto em arquivos XPS sem edição manual.

## Como adicionar texto usando a abordagem aspose page add text?
`Page` representa uma única página dentro de um documento XPS. A classe `TextFragment` encapsula um trecho de texto que pode ser colocado em uma página. Carregue um documento XPS, crie um objeto `Page` e use a classe `TextFragment` para inserir a string desejada. Este fluxo de trabalho **aspose page add text** requer apenas três chamadas de API e funciona de forma consistente em todas as plataformas suportadas, permitindo que você adicione texto rapidamente enquanto preserva o layout e a formatação existentes.
