## Perguntas Frequentes

**Q: Posso usar Aspose.Page para conversão de XPS para PDF em uma aplicação web?**  
A: Absolutamente. A biblioteca é thread‑safe e funciona bem em contêineres servlet ou serviços Spring Boot.

**Q: Existe alguma limitação de tamanho para os arquivos XPS que posso converter?**  
A: A API em si não tem limite rígido, mas você deve considerar o tamanho do heap da JVM para documentos muito grandes.

**Q: Preciso instalar fontes adicionais no servidor?**  
A: Aspose.Page usa as fontes do sistema por padrão. Se um XPS de origem referenciar uma fonte personalizada, certifique‑se de que ela esteja instalada ou incorporada no XPS.

**Q: Como lidar com arquivos XPS protegidos por senha?**  
A: Use a classe `LoadOptions` para fornecer a senha ao carregar o documento XPS.

**Q: Posso converter XPS para PDF sem perder gráficos vetoriais?**  
A: Sim. Aspose.Page preserva os gráficos vetoriais, garantindo que o PDF resultante seja idêntico ao XPS original.
