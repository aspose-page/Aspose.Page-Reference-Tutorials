## Como desenhar formas java em documentos XPS
Desenhar formas em XPS usando Java segue um padrão claro: criar um objeto `Page`, definir a geometria da forma, aplicar estilo (stroke, fill, gradients) e adicioná‑la à coleção `Graphics` da página. Essa abordagem oferece controle total sobre tamanho, posição e efeitos visuais, permitindo criar documentos que se destacam.

## Perguntas Frequentes

**Q: Preciso de uma licença paga para usar Aspose.Page para desenhar formas?**  
A: Uma versão de avaliação gratuita está disponível para avaliação, mas uma licença comercial é necessária para implantações em produção.

**Q: Posso combinar várias formas em uma única página XPS?**  
A: Sim, você pode adicionar quantas formas precisar; basta adicionar cada forma à coleção de gráficos da página.

**Q: É possível aplicar transformações (rotacionar, escalar) às formas?**  
A: Absolutamente. A API fornece matrizes de transformação que permitem rotacionar, escalar ou inclinar qualquer forma.

**Q: Quais versões do Java são suportadas?**  
A: Aspose.Page for Java suporta Java 8 e ambientes de tempo de execução mais recentes.

**Q: Como exporto o documento XPS final após adicionar as formas?**  
A: Chame o método `save` no objeto `Document`, especificando o caminho e o formato de arquivo desejados.
