## Problemas comuns & Dicas

- **Fontes ausentes** – Certifique‑se de que as fontes referenciadas no EPS estejam instaladas na máquina host ou incorpore‑as usando a API.  
- **Tamanho de arquivo grande após redimensionamento** – Use o método `saveOptions.setCompressionLevel()` para reduzir o tamanho sem perder qualidade.  
- **Corte inesperado** – Verifique novamente as coordenadas do retângulo de recorte; elas são expressas em pontos (1 pt = 1/72 polegada).  

## Perguntas Frequentes

**P: Posso processar vários arquivos EPS em lote?**  
R: Sim. Envolva a lógica de recorte ou redimensionamento dentro de um loop e reutilize o mesmo objeto `Page` para cada arquivo.

**P: O Aspose.Page suporta EPS com imagens raster incorporadas?**  
R: Absolutamente. Tanto elementos vetoriais quanto raster são preservados durante a manipulação.

**P: É possível converter EPS para outros formatos (PDF, PNG) após a edição?**  
R: Sim. Após recortar ou redimensionar, você pode chamar `page.save("output.pdf", SaveFormat.Pdf)` ou `SaveFormat.Png`.

**P: E se o arquivo EPS estiver corrompido?**  
R: A API lança uma `FileFormatException`. Capture a exceção e valide o arquivo de origem antes do processamento.

**P: Preciso fechar algum recurso manualmente?**  
R: A classe `Page` implementa `Closeable`; use um bloco try‑with‑resources ou chame `page.close()` para liberar memória.
