## O que é um Gradiente Radial em PostScript?
Um gradiente radial mistura cores a partir de um ponto focal para fora em forma circular. No PostScript, isso é conseguido definindo um dicionário de sombreamento que descreve as cores inicial e final, o raio e o ponto central. O Aspose.Page simplifica isso expondo uma API fluente que gera o código PostScript necessário nos bastidores.

## Por que usar o Aspose.Page para criar Gradiente Radial?

- **Portabilidade:** O PostScript gerado funciona em impressoras, visualizadores e conversores.  
- **Escalabilidade:** Combine múltiplos gradientes (diagonal, horizontal, vertical) em um único documento.

## Perguntas Frequentes

**Q: Posso combinar vários tipos de gradiente em um único arquivo PostScript?**  
A: Sim, o Aspose.Page permite empilhar gradientes diagonal, horizontal, radial e vertical em qualquer ordem.

**Q: Existe um limite para o número de pontos de cor em um gradiente radial?**  
A: Praticamente não; você pode definir quantos pontos precisar, mas tenha em mente o desempenho para sombreamentos muito complexos.

**Q: Preciso calcular manualmente a matriz do gradiente?**  
A: Não, a API fornece métodos auxiliares para definir o ponto central, o raio e a matriz de transformação automaticamente.

**Q: Como visualizo o gradiente antes de imprimir?**  
A: Salve o PostScript gerado como um arquivo .ps e abra-o com qualquer visualizador de PostScript (por exemplo, Ghostscript ou Adobe Distiller).
