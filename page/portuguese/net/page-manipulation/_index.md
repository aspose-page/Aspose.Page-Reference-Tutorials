## Perguntas Frequentes

**Q: Existe um limite para o número de páginas que posso adicionar a um documento PostScript?**  
A: O Aspose.Page não impõe limite rígido; as únicas restrições são considerações de memória e tamanho do arquivo.

**Q: Preciso converter XPS para PDF antes de adicionar páginas?**  
A: Não, você pode adicionar páginas diretamente a arquivos XPS usando os mesmos métodos de API que para PS.

**Q: Posso preservar anotações existentes ao remover uma página?**  
A: Sim, a biblioteca mantém as anotações nas páginas restantes automaticamente.

**Q: Quais passos de solução de problemas devo seguir se uma página recém‑adicionada aparecer em branco?**  
A: Verifique se o fluxo de conteúdo está corretamente preenchido e se o tamanho da página corresponde às configurações de DPI do documento.

**Q: Existem projetos de exemplo que eu possa consultar?**  
A: A documentação do Aspose.Page inclui exemplos prontos para execução tanto para manipulação de páginas PS quanto XPS.
