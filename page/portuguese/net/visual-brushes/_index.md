## O que é um Visual Brush?

Um Visual Brush captura um elemento visual — como formas, texto ou imagens — e permite reutilizá‑lo como preenchimento para outros objetos de desenho. Essa abordagem reduz duplicação e garante consistência de estilo em todo o documento.

## Por que usar um Grid Visual Brush?

Um Grid Visual Brush cria um padrão de grade em mosaico que pode servir como fundo, guia ou elemento decorativo. É especialmente útil quando você precisa de:

- **Alinhamento consistente** em várias páginas.  
- **Estilização leve** sem incorporar imagens grandes.  
- **Escala dinâmica** que se adapta a diferentes tamanhos de página.

## Perguntas Frequentes

**Q:** *Posso combinar múltiplos visual brushes em um único documento?*  
**A:** Sim, você pode sobrepor diferentes Visual Brushes para criar padrões e efeitos complexos.

**Q:** *É possível alterar o tamanho da grade em tempo de execução?*  
**A:** Absolutamente. Ajuste as propriedades `CellWidth` e `CellHeight` antes da renderização para modificar a grade dinamicamente.

**Q:** *Visual Brushes afetam o tamanho do arquivo?*  
**A:** Como são baseados em vetor, adicionam sobrecarga mínima comparada a imagens raster.

**Q:** *Posso usar um Visual Brush com saída PDF?*  
**A:** Sim, Aspose.Page suporta PDF, XPS e vários outros formatos preservando as definições de Visual Brush.

**Q:** *Quais passos de solução de problemas devo seguir se o brush não aparecer?*  
**A:** Verifique se o brush está atribuído a um objeto desenhável e se as configurações de renderização do documento estão corretas.
