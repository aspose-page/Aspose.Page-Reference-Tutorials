Pronto para **criar documentos XPS** de forma rápida e confiável? ## Respostas rápidas
- **O que posso criar?** Arquivos XPS, PostScript e EPS com gráficos e texto ricos.  
- **Quais versões do .NET são suportadas?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## O que é “criar documento XPS”?
Criar um documento XPS significa gerar programaticamente um arquivo de layout fixo que preserva a fidelidade das páginas em plataformas Windows. Aspose.Page para .NET fornece uma API fluente para desenhar formas, adicionar texto, incorporar imagens e aplicar efeitos avançados — tudo sem precisar de ferramentas Adobe.

## Por que usar Aspose.Page para .NET?
- **Controle total** sobre gráficos vetoriais, gradientes, texturas e tickets de impressão.  
- **Sem dependências externas** – biblioteca .NET pura.  
- **Alto desempenho** para processamento em massa de grandes conjuntos de documentos.  
- **Suporte a múltiplos formatos** – converta facilmente para PDF, PNG ou EPS.

## Perguntas Frequentes

**Q: Como iniciar um novo documento XPS do zero?**  
A: Use a classe `XpsDocument`, crie uma página e adicione gráficos ou texto via a API `XpsPage`.

**Q: Posso aplicar um preenchimento em gradiente a formas em um documento XPS?**  
A: Sim – a classe `GradientBrush` permite definir gradientes lineares ou radiais e aplicá‑los a qualquer objeto de desenho.

**Q: Qual a melhor forma de mesclar vários arquivos XPS em um único PDF?**  
A: Carregue cada XPS com `XpsDocument`, então use `PdfSaveOptions` para exportar e combiná‑los, como mostrado no tutorial “merge XPS to PDF”.

**Q: Como incorporar um print ticket personalizado em um arquivo XPS?**  
A: Crie um objeto `PrintTicket`, configure as definições desejadas e anexe‑o à página usando `XpsPage.PrintTicket`.

**Q: É possível recortar uma imagem EPS antes de inseri‑la?**  
A: Absolutamente – a classe `Image` suporta recorte via a propriedade `CropRectangle`, demonstrado na seção “crop EPS image”.
