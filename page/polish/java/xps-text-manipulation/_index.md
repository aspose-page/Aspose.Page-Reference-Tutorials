## Jak dodać tekst przy użyciu podejścia aspose page add text?
`Page` reprezentuje pojedynczą stronę w dokumencie XPS. Klasa `TextFragment` kapsułkuje fragment tekstu, który może być umieszczony na stronie. Załaduj dokument XPS, utwórz obiekt `Page` i użyj klasy `TextFragment`, aby wstawić żądany ciąg znaków. Ten przepływ pracy **aspose page add text** wymaga tylko trzech wywołań API i działa konsekwentnie na wszystkich obsługiwanych platformach, pozwalając szybko dodać tekst przy zachowaniu istniejącego układu i formatowania.
