# Aspose.Page .NET – Rysowanie Kształtów

## Wprowadzenie

Aspose.Page .NET ułatwia programistom rysowanie kształtów w aplikacjach .NET, niezależnie od tego, czy pracujesz z plikami PostScript (PS) lub XPS. W tym przewodniku dowiesz się, jak dodawać koła, elipsy, prostokąty i inne, ucząc się **how to add shapes .NET** style. Poznajmy możliwości i zobaczmy, dlaczego rysowanie kształtów przy użyciu Aspose.Page .NET jest zarówno potężne, jak i intuicyjne.

## Szybkie odpowiedzi
- **What does Aspose.Page .NET do?** Umożliwia programowe tworzenie i manipulację dokumentami PS i XPS, w tym rysowanie kształtów geometrycznych.  
- **Which shapes can I draw?** Koła, elipsy, prostokąty i własne ścieżki.  
- **Do I need a license?** Dostępna jest darmowa wersja próbna; licencja komercyjna jest wymagana do użytku produkcyjnego.  
- **What .NET versions are supported?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.  
- **Is there sample code?** Tak – każdy powiązany tutorial dostarcza gotowe do uruchomienia przykłady.

## Co to jest Aspose.Page .NET?
Aspose.Page .NET to biblioteka .NET, która pozwala generować i edytować dokumenty PostScript i XPS bez potrzeby używania narzędzi Adobe. Oferuje bogate API do rysowania kształtów, stosowania kolorów, gradientów oraz zarządzania układem strony — wszystko z czystego, zarządzanego kodu.

## Dlaczego rysować kształty .NET przy użyciu Aspose.Page?
- **Cross‑format support:** Napisz raz, wyprowadź do PS lub XPS.  
- **High fidelity:** Grafika wektorowa zachowuje jakość przy dowolnej skali.  
- **No external dependencies:** Czysty .NET, bez wymogu natywnych bibliotek.  
- **Developer‑friendly API:** Metody fluent i przejrzyste nazewnictwo ułatwiają **draw shapes .NET** aplikacjom.

## Najczęściej zadawane pytania

**Q: Czy mogę używać Aspose.Page .NET w aplikacji komercyjnej?**  
A: Tak, ważna licencja Aspose zezwala na użycie komercyjne; dostępna jest darmowa wersja próbna do oceny.

**Q: Czy muszę instalować jakiekolwiek natywne komponenty?**  
A: Nie, Aspose.Page .NET to czysta biblioteka zarządzana — wystarczy odwołać się do pakietu NuGet.

**Q: Czy można łączyć kształty z tekstem na tej samej stronie?**  
A: Oczywiście. API pozwala najpierw rysować kształty, a następnie dodawać obiekty tekstowe, kontrolując kolejność Z‑order w razie potrzeby.

**Q: Jak radzić sobie z dużymi dokumentami zawierającymi wiele kształtów?**  
A: Użyj przeciążeń `Document.Save` z buforowaniem strumieni i rozważ podział stron, aby utrzymać niskie zużycie pamięci.

**Q: Czy Aspose.Page obsługuje przezroczystość i gradienty?**  
A: Tak, zarówno API PS, jak i XPS zawierają pędzle gradientowe oraz kompozycję alfa, umożliwiając bogate efekty wizualne.
