## Görüntü ekleme – Adım adım genel bakış

Kütüphane kurulduktan sonra izleyebileceğiniz kısa bir yol haritası:

1. **Bir `Document` nesnesi oluşturun**; bu, düzenlemek istediğiniz PostScript dosyasını temsil eder.  
2. **Bir `Image` nesnesi örnekleyin**; dosya, akış veya bayt dizisinden alınabilir.  
3. **Yerleştirme dikdörtgenini tanımlayın** (X, Y, genişlik, yükseklik) – görüntünün görüneceği alan.  
4. **`document.addImage(image, rect)` metodunu çağırın**; grafiği belgeye gömün.  
5. **Güncellenen belgeyi** diske veya bir akışa kaydedin.

Bu adımlar, bağlantılı “Add Image in Java PostScript” öğreticisinde gösterildiği gibi, kod parçacıklarını doğrudan projenize kopyalayıp yapıştırmanıza olanak tanır.

## Sıkça Sorulan Sorular

**S: Aynı PostScript sayfasına birden fazla görüntü ekleyebilir miyim?**  
C: Evet. Farklı yerleştirme dikdörtgenleriyle `addImage` metodunu tekrarlı olarak çağırabilirsiniz.

**S: Aspose.Page vektör grafikleri de destekliyor mu?**  
C: Kesinlikle. Raster görüntülerin yanı sıra SVG, EPS veya ham PostScript komutlarını da gömebilirsiniz.

**S: Hangi Java sürümleri uyumludur?**  
C: Kütüphane Java 8 ve üzeri sürümlerle çalışır; Java 11, 17 ve sonraki LTS sürümlerini de destekler.

**S: Görüntüyü eklerken döndürme yapmanın bir yolu var mı?**  
C: Evet. `addImage` çağrısına önce `Matrix` dönüşüm API'si ile rotasyon ayarlayabilirsiniz.

**S: Şeffaf PNG'leri nasıl yönetirim?**  
C: Şeffaf PNG'ler otomatik olarak korunur; yalnızca hedef PostScript görüntüleyicisinin alfa kanallarını desteklediğinden emin olun.
