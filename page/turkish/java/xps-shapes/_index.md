## Sıkça Sorulan Sorular

**Q: Şekil oluşturulduktan sonra gradyan renklerini değiştirebilir miyim?**  
A: Yes, you can modify the `GradientStop` collection and re‑apply it to the ellipse’s stroke.

**Q: Dikdörtgene gölge efekti eklemek mümkün mü?**  
A: While XPS does not have a native shadow, you can simulate one by drawing a slightly offset, semi‑transparent rectangle behind the original.

**Q: Bu yöntemler PDF dönüşümüyle çalışır mı?**  
A: Absolutely. Aspose.Page lets you convert the XPS document to PDF while preserving all vector shapes.

**Q: Şekillerin DPI bağımsız olmasını nasıl sağlarsınız?**  
A: Because XPS uses vector graphics, shapes scale automatically; just avoid raster images inside the shape.

**Q: Daha karmaşık çokgenler çizmeye ihtiyacım olursa?**  
A: Use the `Path` class to define custom geometry; the same gradient and stroke settings apply.
