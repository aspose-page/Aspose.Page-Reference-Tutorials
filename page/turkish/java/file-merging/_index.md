A: API'nin kendisinde katı bir limit yoktur, ancak çok büyük belgeler için JVM heap boyutunu göz önünde bulundurmalısınız.

A: XPS belgesini yüklerken şifreyi sağlamak için `LoadOptions` sınıfını kullanın.
