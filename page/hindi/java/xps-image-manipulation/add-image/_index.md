# How to Add Image to Java XPS Documents with Aspose.Page

XPS फ़ाइलों में छवियां जोड़ना Java डेवलपर्स के लिए एक सामान्य आवश्यकता है, जो रिपोर्ट, इनवॉइस या किसी भी दृश्य दस्तावेज़ को समृद्ध करना चाहते हैं। इस ट्यूटोरियल में आप **छवि कैसे जोड़ें** को Aspose.Page for Java लाइब्रेरी की मदद से सीखेंगे। हम प्रत्येक चरण को विस्तार से देखेंगे, समझाएंगे कि प्रत्येक पंक्ति क्यों महत्वपूर्ण है, और सामान्य समस्याओं से बचने के लिए टिप्स देंगे।

## त्वरित उत्तर
- **कौन सी लाइब्रेरी आवश्यक है?** Aspose.Page for Java  
- **क्या मैं कई छवियां जोड़ सकता हूँ?** हाँ – प्रत्येक चित्र के लिए छवि‑जोड़ने के चरण दोहराएँ  
- **समर्थित छवि प्रारूप?** TIFF, JPEG, PNG, GIF (और अन्य जो .NET द्वारा समर्थित हैं)  
- **क्या मुझे लाइसेंस चाहिए?** मूल्यांकन के लिए एक मुफ्त ट्रायल काम करता है; उत्पादन के लिए व्यावसायिक लाइसेंस आवश्यक है  
- **सामान्य कार्यान्वयन समय?** बुनियादी छवि सम्मिलन के लिए लगभग 10‑15 मिनट  

## परिचय
XPS दस्तावेज़ों में छवियां जोड़ना कई Java अनुप्रयोगों में सामान्य आवश्यकता है, चाहे वह रिपोर्ट जनरेशन हो या दस्तावेज़ प्रोसेसिंग। Aspose.Page for Java इस कार्य को सरल बनाता है, प्रभावी मेथड्स प्रदान करता है जिससे आप अपनी XPS फ़ाइलों में आसानी से छवियां एकीकृत कर सकते हैं। इस ट्यूटोरियल में हम Aspose.Page for Java का उपयोग करके XPS दस्तावेज़ में छवि जोड़ने का तरीका दिखाएंगे।

## पूर्वापेक्षाएँ
ट्यूटोरियल शुरू करने से पहले सुनिश्चित करें कि आपके पास निम्नलिखित तैयार हों:
1. **Aspose.Page for Java लाइब्रेरी** – [वेबसाइट](https://releases.aspose.com/page/java/) से Aspose.Page for Java लाइब्रेरी डाउनलोड और इंस्टॉल करें।  
2. **Java विकास पर्यावरण** – सुनिश्चित करें कि आपके मशीन पर Java विकास पर्यावरण सेट अप है।

अब अगले चरणों की ओर बढ़ते हैं।

## पैकेज आयात करें
अपने Java प्रोजेक्ट में आवश्यक Aspose.Page for Java पैकेज आयात करें ताकि आप आवश्यक क्लास और मेथड्स तक पहुंच सकें।

```java
import com.aspose.xps.XpsDocument;
import com.aspose.xps.XpsPath;
import java.awt.geom.Rectangle2D;
```

## चरण 1: दस्तावेज़ निर्देशिका सेट करें
XPS दस्तावेज़ और छवि फ़ाइलों को संग्रहीत करने के लिए अपनी दस्तावेज़ निर्देशिका का पथ परिभाषित करें।

```java
String dataDir = "Your Document Directory";
```

## चरण 2: नया XPS दस्तावेज़ बनाएं
निम्न कोड स्निपेट का उपयोग करके एक नया XPS दस्तावेज़ प्रारंभ करें:

```java
XpsDocument doc = new XpsDocument();
```

## चरण 3: XPS दस्तावेज़ में छवि जोड़ें
छवि जोड़ने के लिए, XPS दस्तावेज़ में एक पथ बनाएं, और प्रदान की गई छवि फ़ाइल तथा निर्देशांक के साथ इमेज ब्रश सेट करें।

```java
XpsPath path = doc.addPath(doc.createPathGeometry("M 30,20 l 258.24,0 0,56.64 -258.24,0 Z"));
path.setRenderTransform(doc.createMatrix(0.7f, 0f, 0f, 0.7f, 0f, 20f));
path.setFill(doc.createImageBrush(dataDir + "QL_logo_color.tif", new Rectangle2D.Double(0f, 0f, 258.24f, 56.64f), new Rectangle2D.Double(50f, 20f, 193.68f, 42.48f)));
```

## चरण 4: परिणामी XPS दस्तावेज़ सहेजें
परिवर्तित XPS दस्तावेज़ को अपनी निर्दिष्ट निर्देशिका में सहेजें।

```java
doc.save(dataDir + "AddImage_out.xps");
```

इन चरणों को दोहराकर अधिक छवियां जोड़ें या अपने प्रोजेक्ट की आवश्यकताओं के अनुसार मौजूदा छवियों को अनुकूलित करें।

## अतिरिक्त सुझाव और सामान्य कठिनाइयाँ
- **पाथ जियोमेट्री सटीकता** – सुनिश्चित करें कि SVG‑शैली पाथ स्ट्रिंग आपके छवि के आयामों से मेल खाती हो; अन्यथा छवि खिंची या कट हो सकती है।  
- **इमेज ब्रश स्केलिंग** – `createImageBrush` मेथड स्रोत और गंतव्य आयत लेता है; इन मानों को समायोजित करके आप स्थिति और स्केलिंग को सटीक रूप से नियंत्रित कर सकते हैं।
