# जावा पोस्टस्क्रिप्ट में Aspose.Page के साथ आयत कैसे बनाएं

## परिचय
यदि आपको **जावा पोस्टस्क्रिप्ट फ़ाइल** के भीतर **आयत कैसे बनाएं** आकारों की आवश्यकता है, तो आप सही जगह पर आए हैं। इस ट्यूटोरियल में हम Aspose.Page for Java का उपयोग करके रंगीन आयतें जोड़ने, उनके फ़िल और स्ट्रोक को नियंत्रित करने, और परिणाम को पोस्टस्क्रिप्ट दस्तावेज़ के रूप में सहेजने की प्रक्रिया को चरण‑दर‑चरण देखेंगे। आप देखेंगे कि **पेंट कैसे सेट करें**, आयत की ज्यामिति कैसे परिभाषित करें, और यह तरीका प्रोग्रामेटिक रूप से प्रिंटेबल ग्राफ़िक्स उत्पन्न करने के लिए क्यों आदर्श है। गाइड के अंत तक आप **draw rectangle java** तकनीकों के व्यापक संदर्भ और वे **java awt rectangle drawing** वर्कफ़्लो में कैसे फिट होते हैं, समझ जाएंगे।

## त्वरित उत्तर
- **कौन सा लाइब्रेरी आवश्यक है?** Aspose.Page for Java  
- **क्या मैं आयत के रंग बदल सकता हूँ?** हाँ – किसी भी `java.awt.Color` के साथ `setPaint` का उपयोग करें  
- **क्या विकास के लिए लाइसेंस चाहिए?** परीक्षण के लिए मुफ्त ट्रायल चलती है; उत्पादन के लिए लाइसेंस आवश्यक है  
- **उदाहरण में कौन सा पेज आकार उपयोग किया गया है?** A4 (डिफ़ॉल्ट `PsSaveOptions`)  
- **क्या कोड Java 8+ के साथ संगत है?** बिल्कुल – यह मानक AWT क्लासेस का उपयोग करता है  

## पोस्टस्क्रिप्ट में “आयत कैसे बनाएं” क्या है?
पोस्टस्क्रिप्ट दस्तावेज़ में आयत बनाना मतलब एक आयताकार क्षेत्र को परिभाषित करना और उसे भरना, उसकी रूपरेखा को स्ट्रोक करना, या दोनों करना। Aspose.Page के साथ आप यह परिचित **java awt rectangle drawing** क्लासेस का उपयोग करके कर सकते हैं, जिससे कोड पढ़ने और बनाए रखने में आसानी होती है।

## आयत ग्राफ़िक्स के लिए Aspose.Page क्यों उपयोग करें?
- **क्रॉस‑प्लेटफ़ॉर्म**: मानक पोस्टस्क्रिप्ट उत्पन्न करता है जो किसी भी प्रिंटर पर काम करता है।  
- **सूक्ष्म नियंत्रण**: आप फ़िल रंग, स्ट्रोक रंग, और लाइन चौड़ाई को स्वतंत्र रूप से सेट कर सकते हैं।  
- **कोई बाहरी निर्भरताएँ नहीं**: केवल बिल्ट‑इन AWT ज्योमेट्री क्लासेस का उपयोग करता है, इसलिए अतिरिक्त ग्राफ़िक्स लाइब्रेरी की आवश्यकता नहीं।  

## पूर्वापेक्षाएँ
ट्यूटोरियल शुरू करने से पहले सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ मौजूद हों:
- जावा प्रोग्रामिंग का बुनियादी ज्ञान।  
- Aspose.Page for Java लाइब्रेरी स्थापित हो। यदि नहीं, तो इसे [Aspose.Page for Java documentation](https://reference.aspose.com/page/java/) से डाउनलोड करें।  
- आपके मशीन पर जावा विकास पर्यावरण सेट अप हो।

## पैकेज आयात करें
अपने जावा प्रोजेक्ट में आवश्यक पैकेज आयात करके शुरू करें। ये इम्पोर्ट्स आपको AWT के `Color`, `BasicStroke`, और `Rectangle2D` क्लासेस, साथ ही Aspose.Page के कोर डॉक्यूमेंट हैंडलिंग क्लासेस तक पहुँच प्रदान करेंगे।

```java
import java.awt.BasicStroke;
import java.awt.Color;
import java.awt.geom.Rectangle2D;
import java.io.FileOutputStream;
import com.aspose.eps.PsDocument;
import com.aspose.eps.device.PsSaveOptions;
```

## जावा पोस्टस्क्रिप्ट में आयत बनाने के चरण‑दर‑चरण मार्गदर्शक

### चरण 1: आयत भरने के लिए पेंट सेट करें  
**पेंट कैसे सेट करें** – हम पहली आयत के लिए नारंगी फ़िल रंग चुनते हैं। यह **set rectangle color java** क्षमता को दर्शाता है।

```java
// The path to the documents directory.
String dataDir = "Your Document Directory";
// Create output stream for PostScript document
FileOutputStream outPsStream = new FileOutputStream(dataDir + "AddRectangle_outPS.ps");
// Create save options with A4 size
PsSaveOptions options = new PsSaveOptions();
// Create new PS Document with the page opened
PsDocument document = new PsDocument(outPsStream, options, false);
// Set paint for filling rectangle
document.setPaint(Color.ORANGE);        
// Fill the first rectangle
document.fill(new Rectangle2D.Float(250, 100, 150, 100));
```

### चरण 2: आयत की रूपरेखा के लिए पेंट सेट करें  
**set rectangle color java** – अब हम पेंट को लाल बदलते हैं और स्ट्रोक की चौड़ाई निर्धारित करते हैं। इस भाग में दिखाया गया है कि कैसे **draw rectangle java** को कस्टम लाइन मोटाई के साथ किया जाता है।

```java
// Set paint for stroking rectangle
document.setPaint(Color.RED);
// Set stroke
document.setStroke(new BasicStroke(3));
// Stroke (outline) the second rectangle
document.draw(new Rectangle2D.Float(250, 300, 150, 100));
```

### चरण 3: वर्तमान पेज बंद करें और दस्तावेज़ सहेजें  
ड्रॉइंग के बाद, हम पेज को बंद करते हैं और फ़ाइल को स्थायी बनाते हैं। पेज बंद करने से सभी ड्रॉइंग कमांड पोस्टस्क्रिप्ट स्ट्रीम में फ्लश हो जाते हैं।

```java
// Close current page
document.closePage();
// Save the document
document.save();
```

## आयत ड्रॉइंग के सामान्य उपयोग केस
- **इनवॉइस या रसीद जनरेशन** – आयतें बॉर्डर या हाइलाइट सेक्शन के रूप में कार्य करती हैं।  
- **लेबल निर्माण** – उत्पाद जानकारी को अलग करने के लिए रंगीन बॉक्स बनाएं।  
- **सरल चार्ट** – बार‑ग्राफ़ तत्वों के रूप में भरी हुई आयतें उपयोग करें।  
- **कस्टम प्रिंटेबल फ़ॉर्म** – आयतों को टेक्स्ट के साथ मिलाकर फ़ॉर्म फ़ील्ड बनाएं।

## सामान्य समस्याएँ और सुझाव
- **फ़ाइल पाथ त्रुटियाँ** – सुनिश्चित करें कि `dataDir` फ़ाइल सेपरेटर (`/` या `\\`) पर समाप्त हो।  
- **लाइसेंस अपवाद** – ट्रायल संस्करण में वॉटरमार्क जुड़ता है; उत्पादन उपयोग के लिए पूर्ण लाइसेंस प्राप्त करें।  
- **रंग दृश्यता** – कुछ प्रिंटर कुछ RGB मानों को अलग तरह से व्याख्या कर सकते हैं; पहले एक साधारण काली आयत से परीक्षण करें।  
- **मेमोरी उपयोग** – बड़े दस्तावेज़ों के लिए प्रत्येक पेज के बाद स्ट्रीम फ्लश करने पर विचार करें (`document.closePage()`)।
