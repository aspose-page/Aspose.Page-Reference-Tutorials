# XPS दस्तावेज़ Java बनाएं – आयत जोड़ें

## परिचय
इस व्यापक ट्यूटोरियल में आप **XPS दस्तावेज़ Java** फ़ाइलें बनाएँगे और Aspose.Page लाइब्रेरी का उपयोग करके आयतें कैसे जोड़ें, यह सीखेंगे। चाहे आप रिपोर्ट, इनवॉइस या कस्टम ग्राफ़िक्स बना रहे हों, आयत निर्माण में महारत हासिल करने से आपको XPS लेआउट पर सटीक नियंत्रण मिलता है। हम प्रत्येक चरण को विस्तार से बताएँगे, कोड की हर पंक्ति के पीछे का कारण समझाएँगे, और पेशेवर परिणामों के लिए रंग और स्ट्रोक को कैसे कस्टमाइज़ करें, यह दिखाएँगे।

## त्वरित उत्तर
- **कौन सी लाइब्रेरी अनुशंसित है?** Aspose.Page for Java  
- **इम्प्लीमेंटेशन में कितना समय लगेगा?** बेसिक आयत के लिए लगभग 10 मिनट  
- **क्या लाइसेंस की जरूरत है?** विकास के लिए फ्री ट्रायल चलती है; प्रोडक्शन के लिए कमर्शियल लाइसेंस आवश्यक है  
- **कौन सा Java संस्करण समर्थित है?** Java 8 और उसके बाद के संस्करण  
- **क्या मैं कई शैप्स जोड़ सकता हूँ?** हाँ – प्रत्येक शैप के लिए चरण दोहराएँ

## पूर्वापेक्षाएँ
शुरू करने से पहले सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

- Java प्रोग्रामिंग भाषा का बुनियादी ज्ञान।  
- Aspose.Page लाइब्रेरी स्थापित हो। यदि नहीं, तो आप इसे [Aspose.Page Java documentation](https://reference.aspose.com/page/java/) से डाउनलोड कर सकते हैं।  
- एक कार्यशील Java विकास पर्यावरण (IDE, JDK, और यदि आप चाहें तो Maven/Gradle)।

## पैकेज आयात करें
प्रोजेक्ट में आवश्यक पैकेज आयात करने के लिए नीचे दिया गया उदाहरण देखें। सुनिश्चित करें कि Aspose.Page लाइब्रेरी आपके क्लासपाथ में सही तरीके से जोड़ी गई है।

```java
import com.aspose.xps.XpsDocument;
import com.aspose.xps.XpsPath;
```

अब, हम इस उदाहरण को कई चरणों में विभाजित करके Java XPS में आयत जोड़ेंगे।

## चरण 1: दस्तावेज़ डायरेक्टरी सेट करें
```java
// The path to the documents directory.
String dataDir = "Your Document Directory";
```

`"Your Document Directory"` को उस फ़ोल्डर के पाथ से बदलें जहाँ आप XPS फ़ाइलें सहेजना चाहते हैं।

## चरण 2: नया XPS दस्तावेज़ बनाएं
```java
// Create new XPS Document
XpsDocument doc = new XpsDocument();
```

यह पंक्ति **एक नया** XPS दस्तावेज़ बनाती है जिसे आप बाद में शैप्स, टेक्स्ट या इमेज़ से भर सकते हैं।

## चरण 3: CMYK सॉलिड कलर स्ट्रोक वाली आयत जोड़ें
```java
// CMYK (blue) solid color stroked rectangle in the lower left
XpsPath path = doc.addPath(doc.createPathGeometry("M 20,10 L 220,10 220,100 20,100 Z"));
path.setStroke(doc.createSolidColorBrush(
    doc.createColor(dataDir + "uswebuncoated.icc", 1.0f, 1.000f, 0.000f, 0.000f, 0.000f)));
path.setStrokeThickness(12f);
```

यह चरण CMYK सॉलिड कलर के साथ स्ट्रोक वाली आयत जोड़ता है। आकार या स्थिति बदलने के लिए जियोमेट्री स्ट्रिंग (`"M 20,10 L 220,10 220,100 20,100 Z"`) को संशोधित करें, और `createColor` में रंग मानों को अपनी डिज़ाइन के अनुसार बदलें।

## चरण 4: परिणामी XPS दस्तावेज़ सहेजें
```java
// Save resultant XPS document
doc.save(dataDir + "AddRectangle_out.xps");
```

अंत में, पहले निर्दिष्ट डायरेक्टरी में संशोधित XPS दस्तावेज़ को सहेजें।

## सामान्य उपयोग केस
- **रिपोर्ट हेडर** – शीर्षकों के बैकग्राउंड ब्लॉक के रूप में आयतें उपयोग करें।  
- **इनवॉइस टेबल** – सेल या सेक्शन को रंगीन बॉर्डर से हाइलाइट करें।  
- **कस्टम ग्राफ़िक्स** – जटिल शैप्स बनाने के लिए कई आयतों को संयोजित करें।

## समस्या निवारण टिप्स
- **फ़ाइल नहीं मिली त्रुटि:** सुनिश्चित करें कि `dataDir` मौजूदा फ़ोल्डर की ओर इशारा कर रहा है और ICC प्रोफ़ाइल (`uswebuncoated.icc`) मौजूद है।  
- **गलत रंग:** ICC प्रोफ़ाइल को उस कलरस्पेस से मिलाएँ जिसे आप उपयोग करना चाहते हैं (CMYK बनाम RGB)।  
- **अप्रत्याशित आयाम:** सही कोऑर्डिनेट्स के लिए जियोमेट्री स्ट्रिंग को समायोजित करें।

## निष्कर्ष
बधाई हो! आपने सफलतापूर्वक **XPS दस्तावेज़ Java** फ़ाइलें बनाना और Aspose.Page का उपयोग करके आयतें जोड़ना सीख लिया है। यह बुनियादी ज्ञान आपको अधिक समृद्ध XPS दस्तावेज़ बनाने, ग्राफ़िक्स कस्टमाइज़ करने और शैप्स को बड़े वर्कफ़्लो में एकीकृत करने में मदद करेगा।

बिल्कुल! Aspose.Page विभिन्न XPS दस्तावेज़ कार्यों के लिए एक मजबूत फीचर सेट प्रदान करता है।

### क्या मैं खरीदने से पहले Aspose.Page आज़मा सकता हूँ?

## अक्सर पूछे जाने वाले प्रश्न

**प्रश्न: क्या यह तरीका Java 11 या उससे नए संस्करणों के साथ काम करता है?**  
उत्तर: हाँ, Aspose.Page लाइब्रेरी Java 8 और उसके बाद के सभी संस्करणों, जिसमें Java 11, 17 आदि शामिल हैं, के साथ संगत है।

**प्रश्न: क्या मैं आयत क्षेत्र के अंदर इमेज़ एम्बेड कर सकता हूँ?**  
उत्तर: आप `XpsImage` एलिमेंट जोड़ सकते हैं और वही जियोमेट्री कोऑर्डिनेट्स उपयोग करके उसे आयत के ऊपर या अंदर स्थित कर सकते हैं।

**प्रश्न: स्ट्रोक के बजाय भराव रंग कैसे सेट करूँ?**  
उत्तर: `doc.createSolidColorBrush(...)` से बनाए गए सॉलिड कलर ब्रश को `path.setFill(...)` के साथ कॉल करें।

**प्रश्न: क्या आयत को घुमा सकते हैं?**  
उत्तर: `path.setTransform(...)` के साथ ट्रांसफ़ॉर्मेशन मैट्रिक्स लागू करके घुमाव या स्केलिंग प्राप्त की जा सकती है।

**प्रश्न: प्रोडक्शन उपयोग के लिए कौन सा लाइसेंस आवश्यक है?**  
उत्तर: डिप्लॉयमेंट के लिए एक कमर्शियल Aspose.Page लाइसेंस आवश्यक है; फ्री ट्रायल केवल मूल्यांकन और विकास के लिए सीमित है।
