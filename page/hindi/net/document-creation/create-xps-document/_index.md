# Aspose.Page for .NET के साथ XPS दस्तावेज़ बनाएं

## परिचय

Aspose.Page for .NET का उपयोग करके **XPS दस्तावेज़ कैसे बनाएं** पर हमारा चरण‑दर‑चरण मार्गदर्शन में आपका स्वागत है। इस ट्यूटोरियल में हम XPS फ़ाइलें बनाने की प्रक्रिया का अन्वेषण करेंगे, जो इलेक्ट्रॉनिक दस्तावेज़ों के लिए व्यापक रूप से उपयोग किया जाने वाला फ़ॉर्मेट है। चाहे आप एक अनुभवी डेवलपर हों या Aspose.Page के साथ अभी शुरुआत कर रहे हों, यह गाइड स्पष्ट उदाहरणों और विस्तृत व्याख्याओं के साथ आपको सहजता से XPS दस्तावेज़ बनाने में मदद करने के लिए तैयार किया गया है।

## नेमस्पेस इम्पोर्ट करें

Aspose.Page for .NET का उपयोग करने के लिए आपको आवश्यक नेमस्पेस अपने प्रोजेक्ट में इम्पोर्ट करने होंगे। नीचे दिए गए चरणों का पालन करें:

### चरण 1: Aspose.Page का रेफ़रेंस जोड़ें

अपने प्रोजेक्ट में Aspose.Page for .NET लाइब्रेरी का रेफ़रेंस जोड़ें। आवश्यक DLL डाउनलोड किए गए पैकेज में मिल जाएगी।

### चरण 2: नेमस्पेस इम्पोर्ट करें

अपने कोड फ़ाइल में निम्नलिखित नेमस्पेस शामिल करें:

```csharp
using Aspose.Page.XPS;
using Aspose.Page.XPS.XpsModel;
using System.Drawing;
```

अब जब हमने आवश्यकताएँ सेट कर ली हैं और आवश्यक नेमस्पेस इम्पोर्ट कर लिए हैं, चलिए XPS दस्तावेज़ बनाने की प्रक्रिया को कई चरणों में विभाजित करते हैं।

## चरण 1: दस्तावेज़ डायरेक्टरी सेट करें

```csharp
string dir = "Your Document Directory";
```

`"Your Document Directory"` को वास्तविक पाथ से बदलें जहाँ आप आउटपुट XPS फ़ाइल सहेजना चाहते हैं।

## चरण 2: XPS दस्तावेज़ बनाएं

```csharp
XpsDocument xDocs = new XpsDocument();
```

`XpsDocument` क्लास का उपयोग करके एक नया XPS दस्तावेज़ इनिशियलाइज़ करें।

## चरण 3: दस्तावेज़ में ग्लिफ़ जोड़ें

```csharp
var glyphs = xDocs.AddGlyphs("Arial", 12, FontStyle.Regular, 300f, 450f, "Hello World!");
```

`AddGlyphs` मेथड का उपयोग करके दस्तावेज़ में ग्लिफ़ (टेक्स्ट) जोड़ें। आवश्यकतानुसार फ़ॉन्ट, आकार, शैली और पोज़िशन को कस्टमाइज़ करें।

## चरण 4: ग्लिफ़ फ़िल कलर सेट करें

```csharp
glyphs.Fill = xDocs.CreateSolidColorBrush(Color.Black);
```

जोड़े गए ग्लिफ़ के फ़िल कलर को निर्दिष्ट करें। इस उदाहरण में हम काला उपयोग कर रहे हैं, लेकिन आप कोई भी रंग चुन सकते हैं।

## चरण 5: परिणाम सहेजें

```csharp
xDocs.Save(dir + "output.xps");
```

अंत में, XPS दस्तावेज़ को निर्दिष्ट डायरेक्टरी में इच्छित फ़ाइलनाम के साथ सहेजें। परिणामी XPS फ़ाइल में "Hello World!" टेक्स्ट होगा।

बधाई हो! आपने सफलतापूर्वक Aspose.Page for .NET का उपयोग करके XPS दस्तावेज़ बना लिया है।

## सामान्य टिप्स और सावधानियाँ

- **डायरेक्टरी पाथ** – विभिन्न OS पर पाथ सेपरेटर की कमी से बचने के लिए `Path.Combine(dir, "output.xps")` का उपयोग करें।  
- **फ़ॉन्ट उपलब्धता** – आप जो फ़ॉन्ट निर्दिष्ट करते हैं वह उस मशीन पर इंस्टॉल होना चाहिए जहाँ कोड चलता है; अन्यथा Aspose डिफ़ॉल्ट फ़ॉन्ट का उपयोग करेगा।  
- **एकाधिक पृष्ठ** – मल्टी‑पेज XPS फ़ाइल बनाने के लिए अतिरिक्त `XpsPage` ऑब्जेक्ट बनाएं और सहेजने से पहले प्रत्येक पृष्ठ में सामग्री जोड़ें।

इस ट्यूटोरियल में हमने Aspose.Page for .NET का उपयोग करके XPS दस्तावेज़ बनाने की प्रक्रिया को चरण‑दर‑चरण समझा। यह शक्तिशाली लाइब्रेरी इलेक्ट्रॉनिक दस्तावेज़ों को आसानी से जनरेट करने का सहज तरीका प्रदान करती है। विभिन्न फ़ॉन्ट, शैलियों और सामग्री के साथ प्रयोग करें ताकि आपके XPS फ़ाइलें आपकी विशिष्ट आवश्यकताओं के अनुरूप हों।
