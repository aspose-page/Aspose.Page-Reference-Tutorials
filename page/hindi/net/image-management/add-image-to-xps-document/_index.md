# XPS दस्तावेज़ में छवि जोड़ें Aspose.Page for .NET के साथ

इस ट्यूटोरियल में आप सीखेंगे कि कैसे **create XPS document .NET** को बनाएं और शक्तिशाली Aspose.Page लाइब्रेरी का उपयोग करके एक छवि एम्बेड करें। चाहे आप रिपोर्ट, इनवॉइस, या कस्टम ग्राफिक्स बना रहे हों, XPS फ़ाइलों में दृश्य तत्व जोड़ना .NET डेवलपर्स के लिए एक सामान्य आवश्यकता है।

## परिचय

.NET विकास की दुनिया में, XPS दस्तावेज़ों में छवियों को शामिल करना एक सामान्य आवश्यकता है। Aspose.Page for .NET इस प्रक्रिया को सरल बनाता है, एक शक्तिशाली टूलकिट प्रदान करता है जिससे आप XPS दस्तावेज़ों को आसानी से हेरफेर और सुधार सकते हैं। यह ट्यूटोरियल आपको Aspose.Page for .NET का उपयोग करके XPS दस्तावेज़ में छवि जोड़ने के चरणों के माध्यम से मार्गदर्शन करेगा।

## त्वरित उत्तर
- **इस ट्यूटोरियल में क्या कवर किया गया है?** Aspose.Page for .NET के साथ XPS दस्तावेज़ में छवि जोड़ना।  
- **कौन सा मुख्य कीवर्ड लक्षित है?** *create xps document .net*  
- **क्या मुझे लाइसेंस चाहिए?** एक मुफ्त ट्रायल उपलब्ध है; उत्पादन के लिए लाइसेंस आवश्यक है।  
- **कौन से .NET संस्करण समर्थित हैं?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7 के साथ काम करता है।

## “create XPS document .NET” क्या है?
.NET में XPS दस्तावेज़ बनाना मतलब है कि C# या VB.NET का उपयोग करके प्रोग्रामेटिक रूप से XML Paper Specification (XPS) फ़ाइल—एक फिक्स्ड‑लेआउट दस्तावेज़ फ़ॉर्मेट—जनरेट करना। Aspose.Page एक फ़्लुएंट API प्रदान करता है जो लो‑लेवल विवरणों को एब्स्ट्रैक्ट करता है, जिससे आप टेक्स्ट, शैप्स और छवियों जैसे कंटेंट पर ध्यान केंद्रित कर सकते हैं।

## छवि जोड़ने के लिए Aspose.Page क्यों उपयोग करें?
- **पूर्ण नियंत्रण** छवियों की पोजिशनिंग, स्केलिंग और क्लिपिंग पर।  
- **कोई बाहरी निर्भरताएँ नहीं** – लाइब्रेरी इमेज डिकोडिंग को आंतरिक रूप से संभालती है।  
- **क्रॉस‑प्लेटफ़ॉर्म** समर्थन Windows और Linux वातावरण के लिए।  
- **उच्च फ़िडेलिटी** रेंडरिंग जो अंतिम XPS फ़ाइल में इमेज क्वालिटी को बनाए रखती है।

## नेमस्पेस इम्पोर्ट करें

अपने .NET प्रोजेक्ट में आवश्यक नेमस्पेस इम्पोर्ट करके शुरू करें। ये नेमस्पेस Aspose.Page for .NET द्वारा प्रदान की गई सुविधाओं का उपयोग करने के लिए महत्वपूर्ण हैं।

```csharp
using Aspose.Page.XPS;
using Aspose.Page.XPS.XpsModel;
using System.Drawing;
```

## Aspose.Page के साथ XPS दस्तावेज़ में छवि जोड़ें

नीचे हम प्रत्येक चरण को देखते हैं जो **create XPS document .NET** बनाने और एक छवि एम्बेड करने के लिए आवश्यक हैं।

### चरण 1: दस्तावेज़ डायरेक्टरी सेट करें

सबसे पहले अपने दस्तावेज़ डायरेक्टरी का पाथ निर्दिष्ट करें। यह चरण सुनिश्चित करता है कि आपका प्रोजेक्ट फ़ाइलों को कहाँ खोजे और सहेजे, यह जानता है।

```csharp
// ExStart:1
string dataDir = "Your Document Directory";
// ExEnd:1
```

### चरण 2: XPS दस्तावेज़ बनाएं

Aspose.Page for .NET का उपयोग करके एक नया XPS दस्तावेज़ बनाएं।

```csharp
// ExStart:1
XpsDocument doc = new XpsDocument();
// ExEnd:1
```

### चरण 3: XPS दस्तावेज़ में छवि जोड़ें

अब, चलिए XPS दस्तावेज़ में छवि जोड़ते हैं। इस उदाहरण में, हम "QL_logo_color.tif" नाम की सैंपल इमेज का उपयोग करेंगे।

```csharp
// ExStart:1
XpsPath path = doc.AddPath(doc.CreatePathGeometry("M 30,20 l 258.24,0 0,56.64 -258.24,0 Z"));
path.RenderTransform = doc.CreateMatrix(0.7f, 0f, 0f, 0.7f, 0f, 20f);
path.Fill = doc.CreateImageBrush(dataDir + "QL_logo_color.tif", new RectangleF(0f, 0f, 258.24f, 56.64f), new RectangleF(50f, 20f, 193.68f, 42.48f));
// ExEnd:1
```

### चरण 4: परिणामी XPS दस्तावेज़ सहेजें

जोड़ी गई छवि के साथ XPS दस्तावेज़ को सहेजें।

```csharp
// ExStart:1
doc.Save(dataDir + "AddImage_outXPS.xps");
// ExEnd:1
```

अब आपने सफलतापूर्वक Aspose.Page for .NET का उपयोग करके XPS दस्तावेज़ में एक छवि जोड़ दी है!

## सामान्य समस्याएँ और समाधान

- **फ़ाइल न मिलने की त्रुटि** – सुनिश्चित करें कि `dataDir` सही फ़ोल्डर की ओर इशारा कर रहा है और इमेज फ़ाइल का नाम बिल्कुल मेल खाता है, जिसमें Linux पर केस सेंसिटिविटी शामिल है।  
- **छवि विकृत दिख रही है** – आकार और अनुपात नियंत्रित करने के लिए `CreateMatrix` स्केलिंग फैक्टर्स या `RectangleF` पैरामीटर्स को समायोजित करें।  
- **असमर्थित इमेज फ़ॉर्मेट** – Aspose.Page सामान्य रास्टर फ़ॉर्मेट (PNG, JPEG, TIFF) को सपोर्ट करता है। `CreateImageBrush` उपयोग करने से पहले असमर्थित फ़ॉर्मेट को कन्वर्ट करें।
