# Aspose.Page for .NET के साथ XPS दस्तावेज़ से पृष्ठ हटाएँ

## परिचय

यदि आपको प्रोग्रामेटिक रूप से **remove page xps** फ़ाइलें हटानी हैं, तो Aspose.Page for .NET आपको इसे करने का साफ़ और भरोसेमंद तरीका प्रदान करता है। इस ट्यूटोरियल में हम XPS दस्तावेज़ से एक विशिष्ट पृष्ठ को हटाने के लिए आवश्यक सटीक चरणों को दिखाएंगे, यह समझाएंगे कि यह ऑपरेशन क्यों महत्वपूर्ण है, और आपको दिखाएंगे कि अपडेटेड फ़ाइल को डिस्क पर कैसे सहेजा जाए।

## त्वरित उत्तर
- **remove page xps** का क्या अर्थ है? यह XPS (XML Paper Specification) दस्तावेज़ से एक पृष्ठ को हटाने को दर्शाता है।  
- **कौन सा मेथड पृष्ठ को हटाता है?** `RemovePageAt(index)` का उपयोग करें जहाँ इंडेक्स शून्य‑आधारित होता है।  
- **क्या मैं किसी भी स्थिति पर पृष्ठ हटा सकता हूँ?** हाँ – आप आवश्यकतानुसार **delete page at index** 0, 1, 2 आदि हटा सकते हैं।  
- **क्या मुझे Aspose.Page के लिए लाइसेंस चाहिए?** परीक्षण के लिए एक अस्थायी लाइसेंस आवश्यक है; उत्पादन के लिए पूर्ण लाइसेंस आवश्यक है।  
- **क्या कोड .NET 6 के साथ संगत है?** बिल्कुल – Aspose.Page .NET Framework, .NET Core, और .NET 5/6 को समर्थन देता है।

## “remove page xps” क्या है?
XPS दस्तावेज़ से पृष्ठ हटाना का अर्थ है दस्तावेज़ के एक पृष्ठ को निकाल देना जबकि बाकी सामग्री, लेआउट और मेटाडेटा को संरक्षित रखा जाता है। यह ऑपरेशन तब उपयोगी होता है जब आपको PDFs को ट्रिम करना हो, कस्टम रिपोर्ट बनानी हों, या दस्तावेज़ आकार सीमाओं का पालन करना हो।

## Aspose.Page for .NET क्यों उपयोग करें?
- **कोई बाहरी निर्भरताएँ नहीं** – शुद्ध .NET लाइब्रेरी।  
- **उच्च सटीकता** – वेक्टर ग्राफिक्स और लेआउट की शुद्धता को बनाए रखता है।  
- **क्रॉस‑प्लेटफ़ॉर्म** – Windows, Linux, और macOS पर काम करता है।  
- **सरल API** – एक ही मेथड कॉल (`RemovePageAt`) भारी काम को संभालता है।

## पूर्वापेक्षाएँ

कोड में जाने से पहले, सुनिश्चित करें कि आपके पास है:

- **Aspose.Page for .NET** – इसे [Aspose.Page for .NET documentation](https://reference.aspose.com/page/net/) से डाउनलोड करें।  
- एक **.NET विकास परिवेश** (Visual Studio, VS Code, या कोई भी पसंदीदा IDE)।  
- एक **sample XPS document** (जैसे `Sample.xps`) जिसे आप अपने प्रोजेक्ट से संदर्भित कर सकें, ऐसी फ़ोल्डर में रखें।

## नेमस्पेस आयात करें

अपनी C# फ़ाइल के शीर्ष पर आवश्यक नेमस्पेस जोड़ें ताकि कंपाइलर को XPS क्लासेज़ का पता चल सके।

```csharp
using Aspose.Page.XPS;
using Aspose.Page.XPS.XpsModel;
using System.Drawing;
```

## चरण 1: दस्तावेज़ डायरेक्टरी सेट करें

```csharp
// ExStart:3
// The path to the documents directory.
string dataDir = "Your Document Directory";
// ExEnd:3
```

> **Pro tip:** क्रॉस‑प्लेटफ़ॉर्म पाथ निर्माण के लिए `Path.Combine` का उपयोग करें।

## चरण 2: नया XPS दस्तावेज़ बनाएं

```csharp
// ExStart:4
// Create new XPS Document
XpsDocument doc = new XpsDocument(dataDir + "Sample.xps");
// ExEnd:4
```

यह पंक्ति मौजूदा XPS फ़ाइल (`Sample.xps`) को `XpsDocument` ऑब्जेक्ट में लोड करती है, जो हेरफेर के लिए तैयार है।

## चरण 3: इंडेक्स पर पृष्ठ हटाएँ

```csharp
// ExStart:5
// Remove the first page (at index 1).
doc.RemovePageAt(1);
// ExEnd:5
```

`RemovePageAt` मेथड **निर्दिष्ट इंडेक्स पर पृष्ठ को हटाता है**। याद रखें कि इंडेक्सिंग 0 से शुरू होती है, इसलिए `1` दूसरा पृष्ठ हटाता है। जिस पृष्ठ को हटाना है, उसके अनुसार इंडेक्स समायोजित करें।

## चरण 4: परिणामी XPS दस्तावेज़ सहेजें

```csharp
// ExStart:6
// Save resultant XPS document
doc.Save(dataDir + "Sample_out.xps");
// ExEnd:6
```

हटाने के बाद, दस्तावेज़ `Sample_out.xps` के रूप में सहेजा जाता है। अब आप इस फ़ाइल को खोलकर यह सत्यापित कर सकते हैं कि अनचाहा पृष्ठ हट गया है।
